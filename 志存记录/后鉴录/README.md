　后鉴录　　（明）谢蕡　撰 

　　●后鉴录上 

　　刑部等衙门题为献俘事。该钦差监督军务御马监太监陆誾，提督军务太子太保、都察院左都御史陆完，提督军务太子少保、都察院右都御史彭泽，平贼将军提督军务总兵官、咸宁侯仇钺，各将生擒反贼赵鐩、庞文宣等绑献。 
　　正德七年十月二十日，鸿胪寺官于奉天门奏，奉圣旨：三法司、锦衣卫等到午门前会多官好生打着问。钦此。钦遵。臣等会同打着问得犯人一名赵风子，即赵鐩，招系顺天府霸州文安县人。弘治年间，充本县儒学增广生员，后因患病告回侍亲。 
　　正德五年以来，有反贼首今淹死刘六、杨虎、射伤淹死刘七、已砍死齐彦名、朱千户、射伤焚死刘三即刘惠、病故邢老虎等，聚集贼众，往来山东、北直隶等处。张打旗号，僭称大王，攻破州县，烧劫库狱，杀掳人民，奸淫妇女，抢掠财畜，敌杀官军，阻绝粮运，共谋反逆，日渐势众。本年月日不等，有在官谷玉、贾能各不合投入伙内劫杀。正德六年七月十五日，有鐩同县先获董仲义，纠党五百余人，亦往河间府投入刘惠等营内。约有一万余人合伙为盗，攻劫文安县治。比因水涨沮淹，在彼住札十日。 （「在彼住札十日」，「日」原作「月」，据明朱当■〈氵眄〉国朝典故本改。） 有鐩见监妻张氏，男张龙即赵龙儿，未获男赵添、瑞、平、宁，弟赵锐、赵铠即赵钱，监故叔赵逵，并见获在官弟赵鐇、赵镐等男妇七十余口，俱躲本县地名王东村水中站立，被刘七手下贼徒吓赶上岸，要行奸掳。鐩不忍，戳伤贼人二名，当被刘七等将鐩捉住要杀。有先获已处决贼首刘三，一名刘尚质的名刘资，亦被伊掳。杨虎等伙约与鐩原上马为盗， （「原上马为盗」五字，明嘉靖抄本无有。） 鐩将家口发送回家。刘资等各添约。 
　　见获在官贾敏儿即勉儿、段朋、张秀、朱聪、王本、张实、季文金、刘梅、邢本道、陈美、郑良，并先未获见监汝宁府陈振、张玺、吴铎、张隆、李臣儿、刘江、小张勇、李海、宋忠及未获宋纪、宋虎、宋山、小宋儿、唐鉴、陈翰等，四散劫财杀人，声势愈大。各置酒会聚商说「我们这些人马，便也难成大事，不若先从本县起手，然后遍历天下。招集数万人马，以建国扶贤为名，须要下个毒手，人纔归顺。若不从的，逢州破州，逢县灭县。先要得了河北、河南，后到南京。见有空位，那时节着你们拜相封侯，全家都享富贵，却不是好」等语。贼众听允，就在本县起手，攻破大城、青县等处城池。杀伤官民，焚劫房屋，及分差哨马四散掳掠。随被官军追剿。 
　　刘六带领已死男刘仲淮与刘七、齐彦名等一枝人马，从青县前往山东去讫。本月内，刘七复回，攻破冀州，将在官小刘氏伊夫，先为强盗监候今已处决张玄劫去，改名刘玄，领贼一千五百余人各处劫掳。本年月日不等，有齐彦名伊在官妻弟庞文宣，将齐彦名男虎头儿送至贼营，就不合投入伙内，僭称大王，领贼一百余人。及有刘六在官侄刘彦深，并在官李隆、王瓒、于成强、刘惠即孙虎、阎洪，李政、王绍即邵隆、郭汉、宋禄、孙玉、于可升、吕仁、黑虎，各亦不合投入伙内。谷玉、李孜俱跟随齐彦名，贾能跟随刘彦深。李隆、王瓒俱跟随孙虎、刘六。阎洪、吕仁、黑虎俱跟随已故贼首刘彦高，邵隆、宋禄俱跟已死贼首刘廷宝。郭汉跟贾能。孙玉投入杀死贼首吕五伙内，后因吕五身死，亦跟随刘七。于可升先投已死贼首乔二， （「于可升先投已死贼首乔二」，「已」原在「投」下，据明朱当■〈氵眄〉国朝典故本改。） 后因乔二身死，投跟齐彦名。各领贼多寡不等。贾能、郭汉、宋禄、孙玉，各僭称大王。谷玉、李隆、王瓒、于成强、孙虎、阎洪、李孜、邵隆、于可升、吕仁，各号称小头儿。 
　　又有在官宋仓与伊未获朱文，亦各不合与伊已死兄朱傅僭称大王，领贼一千余人。贼众就将传顶朱千户名目与庞文宣等，各不合随同刘六、刘七、齐彦名共谋反逆。攻破枣强县，有知县段豸力屈身死，屠戮人民不计其数。又围劫景州、献县、阜城等处。回到文安，屯聚数日，复往东行攻劫青县、静海、沧州等处，烧劫粮舡。后朱傅在山东即墨县被官军射伤重， （「后朱傅在山东即墨县被官军射伤重」，原无「伤」字，据明嘉靖抄本补。） 走至宁海州身死。及有在官孙隆、孙富亦不合跟随杨虎伙内，僭称大王，四散同谋劫杀。鐩与刘惠，刘资、邢老虎、董仲义等，共推杨虎为主， （「共推杨虎为主」，「主」原作「王」，据明朱当■〈氵眄〉国朝典故本改。） 僭称大王。月日不等，攻破兴济县。又攻沧州不破。烧毁粮舡，杀掳人财不计其数。复回南皮县，起程将东光县顾知县拿住，又沿路攻劫杀人。随有马都堂、边都堂、德州桂都司，各差官前来招抚。鐩等亦回书佯许「若朝廷放赦我等，亦愿解散」等语。前到南宫、枣强等县，屠杀官民。刘三掳得在官小赵氏为妾。本年九月初一日，行至景州地名朱门村，当遇总兵官冯安官军，对敌五阵，杀死伙贼一千余人，鐩等奔遁小滩河北。本月初三日，保定田都司率领指挥赵文等官军截杀，被鐩等杀死官军七千余人，拿住赵文，次日释放。鐩等又往山东蒙山地方，当遇副总兵李瑾前来，敌杀官军四十余人，弃下神鎗盔甲一百余副，蟒衣一件。杨虎将蟒衣沿途穿着，及将在官崔氏即杨寡妇掳占为妻，往来山东济南、东兖、登、莱等府，并所属东平、高唐、济宁、沂、曹等州、青城、乐陵、荏平等县，攻劫烧掳。有见监襄阳府贼人范益、李江、李腾、刘钺、陈春、张山、韩子义、孙升、郭豹、李文举、胡友才、宋淮用、郭清，各见得鐩等行劫有钱，月日不等投入伙内，跟随各处劫杀人财，不计其数。及有见监河南贼人马五管内并未获杨杰、唐子文、朱和等，沿途抢掳人马三千余骑，亦来同伙攻劫。 
　　鐩过泰县，题诗一首：「得罪男儿愁满腹，胸中豪气虹霓吐。满怀血泪洒雕鞍，几回魂梦归乡土。贻累父母囹圄中，连坐妻子钓台苦。平欺敌将虎擒羊，纵横六合谁敢捕。惟愿君恩似海深，释我宁家万事足。」又到沂州汤头地方。经过州县月日不等，先后伙约未获贼张通，见监汝宁府汪太、张仲良、张虎、吴铎、李表，见监襄阳府王明、陈鸾、许文、张海清、王文义、赵宗、张仲、任纪、王潮、李腾、王隆，各随营。沿途劫杀人财，奸淫妇女，不计其数。又掳得见监襄阳府马经、王囊包、张表、刘全、李遂、李营、王泽、董文举、郭玉、董智，各分与马四匹，执持器仗， （「执持器仗」，「持」原作「打」，据明嘉靖抄本改。） 随营行走。及掳得已发落靳仿、徐杰、王俊、张金鸾，各打草喂马，挑担行李。本月内，庞文宣等跟随齐彦名等南行，到于地名裴子岩，被谷太监、陆都堂、督吊许游击等官军杀死伙贼有八九百人。去到大名府地方，又被官军杀死伙贼一千余人。 
　　本年十月内，鐩等又不合攻破灵山等卫，日照等县。至南直隶徐州，攻城未破。十一月初一日，至宿迁县，杀死高邮等卫官军三百余名。鐩等又不合掳得娼妇陈回儿，刘三掳得路氏，各在营奸宿。初二日过河，拿住高邮卫指挥陈朋，带至灵璧县。当有知县陈伯安领兵对敌，又遭鐩等杀死官军二百余人。当将陈伯安拿住，同陈朋带至宿州。攻城不破，烧毁西关，杀死居民无数。比有陈伯安不跪，刘惠嗔怪，是鐩劝放讫。又攻破虹县、永城、夏邑、虞城等县，放火杀人。拿住虞城知县，行至丁家道口放讫。又攻破归德州，劫掳马骡人财。当被守备万全都司官军追至亳州地方，有武平卫致仕石指挥领军一千、僧兵三百，前来截杀。鐩等迎敌，杀死僧兵七十余人。去到地名白龙王庙小黄河渡口，有颕州夏指挥、巩指挥领军对岸札营。杨虎带领伙贼黄宁等七人抢船一只过河，当被官军追翻船只，杨虎、黄宁等俱合淹死。贼众奔往河南府，被官军赶杀。贼首张富奔往孟津，过河来到临清，被获。 
　　鐩等见杨虎身死，贼众又无统束，议将刘应立为贼首。前到龙江，过河至孟城县地方龙岗，方将陈朋放回。又到六合县札营五日，掳得人财。当有副总兵白玉领军围杀，刘惠同未获朱龙等领众出小南门，敌死官军兵快一千五百余人，抢得盔甲鎗刀二千余件，神器七十余件。前到沈丘县攻城， （「前到沈丘县攻城」，原无「到」字，据明朱当■〈氵眄〉国朝典故本补。） 进入，杀死军民一千余人，拿住都司潘翀，杀死都司王保，射伤河南布政司经历任杰，抢得弓箭刀甲。起营攻入鹿邑县，拿住见获在官千户李茂，并失记名千户二员，军人二名，俱各降服跟随。内有在官陈汉即陈翰，不合自称主事，愿与刘惠为男，当将千户二员放讫。随有上蔡县致仕知府张什，同不知名生员二名，老人一名，自备马匹，假金台盏四副，银台盏十副，段二匹，前到鹿邑县馈送刘惠，哀告免攻本县，以救一县生灵。刘惠接收听允。
