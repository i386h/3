庄岳委谈　　（明）胡应麟 撰 

　　●庄岳委谈上 

　　（四库全书 子部 杂家类 杂编之属 少室山房笔丛正集卷二十五） 

　　大荒西经云西海之南流沙之滨赤水之后黑水之前有大山名曰昆仑有人戴胜虎齿豹尾穴处名曰西王母西山经云西玉山是王母所居其状如人豹尾虎齿善啸蓬髪戴胜是司天之属主疠及五残右山海经所述西王母与竹书纪年穆天子传殊不类盖因二书有王母之文而诡撰其状以实之余别辨于二书详矣集仙传则曰虎齿豹尾乃王母之使非真形也是又因山海经而讹者经云三危之山青鸟居焉广圆百里青鸟主为王母取食又云拒巫之山一曰龟其南有三青鸟为王母取食又言三足鸟主王母给使盖豹尾虎齿自为王母本形而三青鸟自为王母使者集仙既以王母为女仙而不得于豹尾虎齿之说故以三青鸟事当之皆梦中说梦也 

　　集仙传言王母生于神州姓侯氏而酉阳杂爼以王母姓杨名回字婉妗夫王母西华之气所化而侯杨皆后世姓氏本无足辩聊识此当一噱云 

　　观音大士絶不闻有妇人称近王长公取楞严普门三章合刻为大士本纪而着论以辟元僧之妄要为不学者言耳初不足辩也余母宋宜人素善病中歳虔精奉大士每困迫辄梦大士化身辄愈又余邑叶氏妇病不知人数日亦梦大士救之而愈此皆余所目击其它显化灵异往往闻之四方余尝欲因长公本纪而彚集诸经中大士言行散见者及六朝以还诸杂记小说中大士应迹较著者合为一编盖余母志云（今大士像作善财其旁亦无谓华严载善财歴叅诸善知识俱非观音变相虽观音在其中然度善财者文殊也） 

　　今塑画观音像无不作妇人者盖菩萨相端妍靓丽文殊普贤悉尔不特观世音也至冠饰以妇人之服则前此未闻考宣和画谱唐宋名手写观音像极多俱不云妇人服李廌董逌画跋所载诸观音像亦然则妇人之像当自近代始盖因大士有化身之说而闺阁多崇奉者展转流传遂致称谓皆讹若塑像势不能久前代无从证订然太平广记载一仕宦妻为神摄因作观音像其妻寻梦一僧救之得苏则唐以前塑像固不作妇人也 

　　杨用修词品记寿涯禅师咏鱼篮观音云深愿宏慈无缝罅乗时走入众生界窈窕丰姿都没赛提鱼卖堪笑马郎来纳败清泠露湿金襕坏茜裙不把珠璎盖特地掀来呈揑怪牵人爱还尽许多菩萨债据此则宋元间观音像亦有作妇人者然是变相未必如近时称谓可笑也今世女子多崇事鱼篮观音盖前代已有此像矣 

　　又宋人小说载南渡甄龙友题观世音像云巧笑倩兮美目盼兮彼美人兮西方之人兮则宋时所塑大士像或已讹为妇人而观世音之称妇人亦当起于宋世元僧謭陋无识遂以为妙庄王女可一笑也 

　　今观世音像率作妇人故人间显迹梦兆无复男子相者俗遂真以观世音为妇人不知梦生于心兆征于目心目注瞻皆非男相则恍惚示现自当女身余考法苑珠林宣验冥祥等记观世音显迹六朝至众其相或菩萨或沙门或道流絶无一作妇人者使当时崇事类今妇人像则显迹繁伙若斯讵容无一示现耶唐世亦然盖误起于宋无疑（因类识数条以袪世俗之惑） 

　　晋义熙十一年太原郭宣蜀郡文处茂先与梁州刺史杨收敬为友收敬有罪下狱宣与处茂同被桎梏念观世音十日后夜梦一菩萨慰谕之告以大命无忧俄而锁械自脱又宋张兴妻繋狱昼夜祈念观音经十日许夜梦一沙门以足蹑之曰咄咄可起妻即惊起钳锁桎梏俱解然闭户警防无由得出乃却自械又梦向沙门曰锁已开矣妻遂驰出又王球在狱至心念观音夜梦升高座见一沙门以一卷经与之题名光明又见一车轮沙门曰此五道门也既觉锁皆断脱又苻秦毕览东平人少奉法随慕容垂北征没敌单马逃窜敌骑追将及览至心诵念观世音既得免脱入山迷惑失道又专心持念中夜见一道人法服持锡示以途径安然至家又晋兴宁中沙门法义得病惟归诚观世音如此数日昼眠梦见一道人来候其病因为治之刳出肠胃洗濯毕还纳之梦觉众患豁然经云或现沙门梵志之象意者义公梦其是乎又东魏孙敬德亦梦沙门令念观音经临刑刀折为三此类甚众竟无一梦见妇人者当时像可推矣 

　　冯夷之为河伯其说逺矣好竒之士诪张眩惑纪载实繁即特立自信君子亦但斥言其妄而未尝不以河伯为水神也乃余独于竹书纪年而得其说焉纪年载帝芬十六年洛伯用与河伯冯夷鬬夫洛与河国名也伯爵也用与冯夷人名也谓河伯河神则洛伯洛神耶夫洛伯能与河伯斗矣胡用之为神寥寥后世而冯夷独盛称耶观泄十六年殷侯以河伯之师伐易则河伯为诸侯而冯夷非神鬼昭昭矣穆天子传河宗柏夭为穆王乗副车导之极于西土而后返则当时柏夭亦造父奔戎之属护从穆王者藉如后世所谓鱼身人首何以周旋天子之侧哉夫九歌屈氏之寓言而鱼水庄生之幻说本未尝谓实有且絶不道冯夷之名而茂先博物成式酉阳从而为说以实之吾不可以不辨亦幸而得之竹书也（段氏所引河伯姓名颇众并识于后以广异闻） 

　　酉阳杂俎云河伯人面乗两龙一曰冰夷一曰冯夷又曰人面鱼身又金匮言名冯循（一作修）河图言姓吕名夷穆天子传言无夷淮南子言冯迟圣贤记言服八石得水仙抱朴子曰八月上庚日溺河（按首所引山海经河图盖纬书也） 

　　今世绘八仙为图不知起自何代盖由杜陵有饮中八仙歌世俗不解何物语遂以道家者流当之要之起自元世王重阳敎盛行以锺离为正阳洞宾为纯阳何仙姑为纯阳弟子夤缘附会以成此目尝观前代书史若七贤过关四皓奕棋等图浅诞不根者甚众独无闻此可知也考其出处亦各有所本张果在诸人最先进明皇时显迹甚着叶法善以为混沌初分白蝙蝠精锺离权吕嵓俱唐中晩人锺有二絶吕有一律见唐诸选中蓝采和亦唐人有踏踏歌见沈汾续神仙传以常衣蓝袍故名韩湘文公侄孙昌黎实赠以诗贾岛亦有诗寄湘皆不言其道术独酉阳杂俎记文公吏侍日偶江淮一族子访之自云善幻文公令试其技顷刻开异花有云横秦岭一联乃录文公旧作非预兆且非湘也何仙姑见纯阳文宋人杂说以为不饮食无漏而徐神翁宣和间海陵人见三仙传颇详其余姓氏间有相同然不可深考总之不足深辩近阅元人庆寿词有锺吕二韩等八人信知起自元世人也 

　　元词有曹国舅考诸仙传曹姓无外戚而诸史曹姓外戚无得仙者据俗传为宋人检宋史惟曹佾为后弟见重于时年七十卒初不云得仙词又有跛者李孔目盖即图中跛足拄杖者尤荒唐然必合此乃得八人之数考诸传记惟神仙通鉴有刘跛子而非李姓与词语殊不相蒙未审元人何据大都委巷之谈耳（刘跛子事出冷斋夜话虽诡异然不曰仙仙鉴何以引之韩湘说尤不一并锺离亦无定论详下条） 

　　按通志有八仙图又有八仙传一卷注唐江积撰则此目唐时已似有之然徐神翁宋世甚明则唐时或他有其人如所谓五真之属且太平广记收神仙类事迹殆尽而锺吕显著若斯絶不见采并唐诸小说亦罕谈及则唐人所谓八仙传者决非锺吕之俦明矣 

　　锺吕传道记称唐施肩吾撰按肩吾晩唐诗人素不闻其有道术而宋初编太平御览搜唐以前小说殆遍独无所谓锺吕传道记者盖必宋世方士讹撰托名唐人耳且吕一律见于鼓吹锺二絶见于延之昌父所收而唐人所自选集皆无之恐俱不足凭也 

　　宣和书谱云神仙锺离先生名权不知何时人而间出接物自谓生于汉吕洞宾于先生执弟子礼有问答语及诗成集状其貎者作伟岸丈夫或峩冠绀衣或虬髯蓬鬓不冠巾而顶双髻文身跣足颀然而立睥睨物表自称天下都散汉又称散人尝草其为诗云得道高僧不易逢几时归去得相从字画飘然有凌云之气非凡笔也元佑七年七月录诗四章赠王定国多论精勤志学长生金丹之事亹亹可读终自论其书谓学龙蛇之状识者信其不诬今御府所藏草书一观此则锺离之像虬髯蓬鬓双髻跣足者与今所传正同盖宋时羽士假托锺离权以诳王定国辈其诗实唐锺离权所作而假托者不详其世以为即汉锺离昧故自称生于汉后世因以汉锺离目之盖其来非一日矣定国为其所欺可耳而蔡元长辈于唐诗亦漫不考何耶宋人淳化帖博古圗二书每供识者谈噱然宣和书画谱误处亦不在二书下杨用修尝叙宣和谱而论锺离乃以杜诗有近闻韦氏妹远在汉锺离之句世人遂赝作此名大可笑也 

　　仙传拾遗云吏部侍郎韩愈甥忘其名姓弱冠往洛下省骨肉乃慕云水不归杳絶音信元和中忽还长安吏部与语甚竒之问其伎能云染花红者可使碧或一朶具五色遂于吏部后堂前染白牡丹一丛云来春必作含棱碧色又四面合有一朶五色者无何濳去不知所之是歳上迎佛骨于凤翔御楼观之吏部上表直谏忤旨出为潮州刺史至商山泥滑雪深忽见是甥迎马首扶镫接辔意甚殷懃至翌日雪霁抵邓州乃白吏部曰某师在此不得远去将入玄扈倚帝峰矣吏部惊异其言问其师即洪崖先生也吏部为五十六字诗以别之曰一封朝奏九重天云云与诗讫挥泪而别行入林谷其速如飞明年春牡丹花开数朶花色一如其说但每一叶花中有楷书十四字曰云横秦岭家何在雪拥蓝关马不前书势精能人工所不及非神仙得道卓见先知何以至斯据此传则开顷刻花者又韩之甥然文公诗题甚明盖江淮族子之说当得其实记者或因此诗而讹为韩湘又或因江淮而讹为外甥耳（文公贬潮州歴歳始归何得次歳仍覩长安牡丹故杂爼之说近之） 

　　永州何仙姑不饮食无漏世传其神异岳州天庆观柱以震折有倒书谢仙火字仙姑云雷部夫妇二人长阔各三尺银色莫不倾信有熟于江湖间事者曰南方贾人各以火自名一火犹一部也此贾名仙刻木记已物耳是亦不可知也尝有道人自言隋唐间人谭黄巢事甚悉因曰黄大晚节至此张安道尚书云巢六兄弟而巢最小当第六由是推之则道人之言信然乎右见刘贡父诗话据吕纯阳语则何仙姑当为唐末人据此说则当是宋初人也安道以强记絶一代其博识仅见此然旧唐书巢兄弟八人揆为巢弟则巢非最小明矣道人固不足凭安道亦未为得贡父该洽冠宋又从笔之甚矣综核之难言也 

　　梦蕉诗话云广之増城有何仙姑者相传为邑民何泰之女生唐开耀间常欲絶俗去游罗浮父母怪之将婚夕忽不知其所之惟研屏间遗题云麻姑怪我恋尘嚣一隔仙凡道路遥去去沧洲弄明月倒骑黄鹤听鸾箫据此则何又当为唐人然他无可据恐好事者为之（齐东埜语亦载何及谢仙事当在庆厯间岂何显于宋世自称唐人耶） 

　　世所盛传张仙像张弓挟弹若贵逰公子以为即梓潼之神文昌之宿然梓潼自有像氅衣纱帽与张仙殊不类且道家言梓潼出处谓文昌尚近之祈嗣絶无干也偶阅陆文裕金台纪闻云张仙像是蜀王孟■〈日上永下〉挟弹图初花蘂夫人入宋宫念其故主偶携此图遂县于壁谨祀之一日太祖幸而见之诘焉花蘂跪答曰此蜀中张仙神也祀之能令人有子非实有所谓张仙也余按纪闻以此说得之蜀中一士夫或类近实盖以张弓为张仙挟弹为诞子而梓潼之神本蜀人且张姓因谬相传今又以梓潼化身传文昌耳
