　　公讳廷瑶字青书贵筑洪边里北衙人父大受乡里称善人以公贵赠如其官母宋治家有法度公以乾隆丙午举于乡四上公交车不售家有田在定番多识定广人嘉庆初兴义苗为乱定广不靖当事延公往抚全活数万人庚申马场苗又为乱公复往抚之事闻　　恩赏六品顶戴明年大挑一等奉　　旨加一等以通判用壬戌选授安徽庐州在皖十年平反全活无算宿州休宁为尤着嘉庆七年十二月宿州教匪王潮名袭据州城戕官焚掠事甫定大府檄公往权又令征本年钱粮四万公曰天庾正供何敢轻言减免惟宿州春夏遭水继以兵乱逃亡不能耕作今虽逃者渐归而已失时补种不及当奏请豁免不则缓征以纾民力大府犹豫公复力争谓与其追呼无益无宁广施恩泽大府悟立陈奏　　诏如所请是时城中居民仅七百余户公亲拊循给口粮盖棚栖止查户口请抚恤事蒇计所耗金五千余命吏无伪增数幕客以为言公恻然曰吾百姓疮痍未起即实心实政犹恐不逮更因以为利何以上告君父下对灾黎也公之定识大虑尤有过人者夜坐斋中忽讹言教匪复至城外居民仓皇奔走哭声雷动城中人亦惊惧公曰此必奸人欲动众藉以刼掠也谕其下好廵守吾卧耳天明四境寂然徽宁属邑旧多棚民佃租垦种厯年已久十三年有以棚民结连匪徒聚众称乱诉京师　　诏晓事文武大员率兵察其事巡抚檄芜湖道徽州协以兵四百会公剿捕公知其诬乞驻兵县城而自轻骑入山廉得其实分别遣散回籍念属邑棚民无虑二十万若同时驱逐将酿事生乱请于大府定为十二年递迁之法大府以公议上闻允为成例公去皖十余年皖人见公乡人犹稽首问公起居不置其嘉泽入人如此嗣以凤阳府同知升任广西平乐府因安徽旧案镌级去两广总督襄平蒋公知公能特奏请捐复调广东候补补广州府循声益着大府皆以国士待公遽引疾归病痊仍赴补广州府丁亥又引疾归庚寅八月十五日卒于里第公之在粤东也东莞匪林狗尾聚众攘夺村民汹汹赴惠州军门告急大府阮公檄公带兵发契箭假公便宜公至则驻兵县城宣榜通衢令居民合力擒贼毋令得混迹又促县令重赏购擒其渠或谓公无轻办公曰贼闻官军来已远扬如君言是必按户搜索矣夫良民以畏贼故求救有司有司复不为翦除而更累吾民是不死于贼而死于官也诸君休矣力请撤兵大府从公议不数日林狗尾等九十余人悉就擒余解散勿论曩令公贪事喜功患可胜言哉岑溪人黎维祺为仇家所诬讼连年不决公至立释之岑溪人归语其众众曰是吾平乐髯太守活尔命邪岑溪人曰然众曰在吾平乐活人多矣今复来东省邪公美髯故云公声若洪钟每遇事高谈雄辩折其座人当事无不倚如左右手喜读书始家贫力不能得及贵藏书甚富无弗阅者性孝友事二亲未尝离左右既老每与故旧言常泣下始归里从子五人皆予以恒产族中子弟诸甥悉经营佽助焉余少耳公名嘉庆甲戌丙子间随先大夫粤东凡两见公近引疾归又与公子以廉以庄善出示以某所作公传余弗谓善也读公自着宦游录缀其循绩可书者着于篇

　　●叙 

　　今之称爱民者曰好官好官何尽心焉而己何谓尽心视民之困苦冤抑如身受为之苏其困乎其冤使民各得其所而已虽然畴不愿做好官而尽心恒难者利诱之而心移威胁之而心慑于是心有所顾惜而不能尽者多矣故欲做好官必自尽心始欲尽心于民必自廉其心毅其心始廉则不疚于利毅则不怵于威而后为所欲为无所顾惜而可以尽吾爱民之心知之而能行之者惟贵筑青书先生先生以名解元出为通守擢同知晋知府初官皖厯仕粤东西余皖人也夙闻乡父老道先生德政心企仰之其官粤东西事则未之闻也洎奉　　天子命视学来黔与喆嗣心泉秀东两君交得备悉先生梗概任满受代将行秀东以先生宦游纪略乞叙于余余受而读之所载嘉言行不一端而于谳狱尤哀矜详慎屡雪奇冤之死而之生其细者亦旁推曲勘期于得其情而止他如赈饥荒捕盗贼备寇警以迄赠籍没之宦裔赎贫鬻之寒媛仁慈侧怛之心随时曲尽此无他有是爱民之心而又能守之以廉持之以毅非义之贿不得而动之长吏之势不得而屈之故能无所顾惜曲尽其心如此也然则是篇也固以窥先生政绩之本原而亦凡做好官者所当奉为圭臬者欤忆余初奉视学　　命请益于汤文端公师公曰学政与主考异主考专司考试衡文外无他事学政学之中有政事焉非徒是严关防精衡校谓可尽乃职也余谨志之不敢忘比入黔值苗教变起试事辄停輶车多暇然每念及邪说猖狂之甚干戈屠戮之惨未尝不叹变乱滋兴由于人心不正人心不正由于教术不明职司风化之谓何而可晏然安乎于是广为条教并刊辑有益人心世道之书以期正人心而挽变乱而事关民命苟有所闻不避越俎之嫌立陈之当事虽于时未必有济而此为民之心不敢不尽也呜呼学政一官自人视之何与民生之休戚而一思教养相关之故则笔舌微权未必非苍生所待命循名责实负疚滋多况于为司牧者百族之忧乐死生听治于一人之方寸而顾忍漠然视之而不殚厥心乎大学曰如保赤子心诚求之读先生书益恍然于为民父母之自有真矣心泉秀东捍卫桑梓亦己殚竭心力心泉将典郡滇中秀东亦出宰西蜀云阳吾知两君莅政必能廉毅自持尽心民瘼不愧父风他日治绩成书或有绍美兹集者则此一编也其即先生传家之治谱也乎和州鲍源深谨叙 

　　亲民之官将以布德惠安百姓苟无才以运之德惠亦不能下究所谓徒善不足为政也有其才矣而自负意气遇事卤莽灭裂剽而不留迨其蹉失益以文非遂过其害有不仅惠不下究者故又非才之难才而沈几远见仓卒无游移精神贯终始乃真难耳前史传循吏最多之代不过二三十人岂其靳哉贵筑高青书丈以名解元起皖省倅丞厯守粤东西剧郡所至结疑积件千百计会谳公局平反数十事辄夺活刀下人庶乎沈几远见精神周密无仓卒无始终者欤昔吾先子每论知旧吏才必以高丈称首且数数道其檄察怀远释误指婚丧轿头为会匪教头数百人之系谳灵壁盗而雪凤阳招解盗十三人之诬以为真神明不可及尤服其改过不吝如六安开塘盗葬诸件不肯执初断以遂非为真儒者举动友芝熟闻窃识储他日先友传事案久矣咸丰乙卯客省会获交喆嗣心泉秀东两君示宦游纪略益得见吾丈循绩之详夙闻诸端首尾曲折益了了信足为学治师法曾乞其副厕弆清端政书鹿洲公案间以时检阅己未冬秀东将官蜀令晤于都门复持本命校勘拟以抵蜀刊行因敬记简末尤愿秀东此行时时无忘此治谱也十二月几望独山莫友芝谨叙 

　　天下之治乱视郡县而已矣视守令而已矣天下至大郡县至小然其势以相积而成比县皆治郡有不治者乎比郡皆治天下有不治者乎今夫圣君哲相在上贤方镇勤施四方其于图功揆政易易也然而尤以守令为最与民亲亲则耳目易周惠泽易浃诚使守令于民皆有至诚恻怛不容已之心与之频呻相通疾痛疴痒颠连困苦相关矢以公廉将以勤慎竭其明察果断疏达开豁之才推究利弊力图兴革以富以教次第举之若风流而令行也是则治之机自守令倡之及其积而成也达之天下而治象成矣自两汉至今未有能歧是说者也柏心曩与黔中唐子方方伯游时时称其乡人高青书先生为嘉道间循吏冠尤以善折狱闻先生归道山久矣去年夏其仲君秀东司马贻书柏心缄先生服官时宦游纪略征序云得之先生手录后乃镂版以行柏心发而读之皆先生自纪所设施及交际推解诸事而载折狱尤详自皖及两粤所至以神君目之此可为后来作吏者之师至其识虑深远则尤有进权宿州时有因乱后力请豁除秋粮四万余之议在徽宁则遣散棚民数十万寝兵不用遂以大靖在粤东则东莞民告急言土匪啸聚肆行攘敓奉檄以兵往剿至则按兵不动但榜令村民擒捕悬赏购其渠魁力请罢兵已而罪人斯得余悉解散村落晏然若此者盖渊乎大臣之识虑矣非止沾沾吏能报称已也柏心窃见近二十年大盗倡乱若江河溃决不可防遏推原其故皆起于官吏阘冗侵削而蕴崇之使然向令所在守令皆如先生民且安其田里树畜家室乐生之不暇敢效篝火狐鸣也哉然后叹天下治乱视守令贤否其说诚不谬也论者谓先生才敏绝人得之于性非可学而能此殆不然夫先生本其至诚恻怛不容已之心视民如己凡公廉勤慎明察果断疏达开豁皆由是心扩充之而已人患不能勉耳今司马与其贤兄心泉廉访绍明先训皆用治行书上考亦既征其效矣世之为守令者得是书反复推求因以扩充其至诚恻怛不容已之心安见治效不能及民者然则是书非止一家治谱推而达之天下蒸为治象不难也先生出处仕宦详他序傅者不复及云监利王柏心谨叙 

　　夫胡书留碣无征则不信悝鼎章媺弗传则匪仁是以三岁不灭置诸志父之怀七叶所贻述以子山之赋旧德可食清芬必诵由来尚矣然而去日苦多浮云易变谘诹所获或人人殊朒朓屡迁难事事备求茂陵之遗橐往已云亡稽随会之治行死不可作顾安得陈编在漆记事有珠守之为手泽之成书推之即心传之治谱如青书高公宦游纪略之明且备者乎公学植不落智蕴如囊出山之云触石而雨士元非百里才惟别驾能展骥足郭公至二千石其治绩皆在人心油盖所经甘棠勿翦至其开法网理棼丝独为其难绰然有??谷化彼鸮音推赤心于疏逖夺诸虎穴起白骨于沈薶诸所措置回越丑夷若三吴若百粤不陨厥闻称到于今独是布濩羣伦濊德也隃度先几赡智也六府烛彻浚照也一字活人桀识也固皆蔚足人师良由资膺天授所惧中材尟兹上理假使舆人徒诵授简蔑□如绣鸳锦金针恡度如寻鸿迹雪泥靡留将令肤受之士耳食无稽而欲学步邯郸铸金少伯亦惟有望洋长叹景山向往已耳公以抱蜀之殚心乘退公之多暇尺木自宝寸阴不虚综厥猷为寿以文字再仕再化钢经大冶之镕一官一集辙立中馗之表惟温公处事无不可对人言故敬礼小文皆由其所自定也夫书道政事史传循良淑世之方民生繄赖彼圣读而疔行采华而忘实无论已即谓等身著作夸经义以无双秘枕异书赞清谈之第二尔能各奏岂曰不文惟元草之矜奇问苍生其奚补玉卮无当金鉴莫裨以视箎牖羣迷韟章歧路毖蓼辛之后患启桐子以前驱者相形之拙不言而喻矣是宜敬仲族昌释之后大杜家法律萃于一门石氏官禄合为万石岂非定国治狱卜车马之将兴崇公好生应鼎钟而食报也哉兹篇流播阅时寖久文孙怡□虑三写之乌焉思一扫其落叶濡墨惟勤杀青斯再潘岳之文采取程先民士衡之词赋聿追世德鸿宝堪藏象贤勿替所望读子产之法者师其捄人诵仲舒之篇者取其切世庶几痤雎得矿苇海有针一本率由固必绳其祖武百尔在位亦将感于斯文贵阳刘藻谨叙
