　皇朝本记　　（明）撰人不详 

　　大明太祖高皇帝濠梁人也姓朱氏世为农业名　　字国瑞　母太后陈氏夜梦一黄冠自西北来至舍南麦场中麦糠内取白药一丸置　太后掌中大后视渐长黄冠曰好物食之　太后应而吞之觉谓　仁祖曰口尚有香明旦　帝生生三日腹胀几殆　仁祖梦抱之寺舍欲舍之抵寺寺僧皆出复抱归家见东房檐下有一僧坐扳櫈面壁闻　仁祖至回身顾曰将来受记于是梦中受记天明病愈自后多生疾症　仁祖益欲舍之　上自始生常有神光满室每一岁间家内必数次夜惊以有火急起视之惟堂前供神之灯他无火及欲出家幼　仁祖未许至十七岁　仁祖与太后俱以疾崩　上长兄王亦逝惟仲兄王存　上自以计日窘思昔父母因疾曾许为僧于是与仲兄谋允托身　皇觉寺入寺方五十日寺主以岁饥罢僧饭食时师且有室家所用弗济乃西游庐六光固汝颖诸州如此三载复入皇觉寺始知立志勤学方 
　　四年天下兵乱过寺寺焚僧散将晓　上归祝伽蓝以珓卜凶吉曰若容吾出境避难则以阳报守旧则以一阴一阳报祝讫以珓投之于地则珓双阴如此者三复祝谓神求出不许入不许神可报我无乃欲我从雄而后昌乎则珓如前祝既投珓如前神既许之于心大惊复祝曰甚恐从雄愿神复与吉兆而往他方避难祝毕以珓投入地一阴复一卓立时神意必从雄而后巳因是固守所居未旬日友人以书自乱离中来略言从雄大意览毕即焚之又旬日有人告傍有知书来者意在觉其事　上心知之复三日斯人果至与语观其辞色未见相复礼待而归复几旬日又有来告先欲觉知事者今去不忍欲令他方人来加害乞幽察以从告　上深思之以四境逼迫讹言蜂起乃决意从诸雄时元至正 
　　十二年壬辰闰三月一日晨旦抵濠城守者不察缚而欲斩之有人觉报于首雄良久得免收入部伍几月投长久大王首雄滁阳王郭子兴是也既长久大王常召与语久之言意相孚王知　上非可久屈收为家人亲待同子弟庸以　孝慈皇后马氏妻之然滁阳王之为人志气雄暴列诸雄之上其雄有四俞曾孙潘意虽同乱及其处也异志俞鲁孙潘出于农其性麄直谋智和同独与王异在乱初防闲守御兵之进止滁阳王本合与焉而王少出外而多居内凡诸事务四雄者每待王久亦不能同谋是后四人专主王若在列与焉不在则不与三五日相会一次其会也四雄瞠目视王王自知礼亏深思不安略有赧色王比四雄志颇聪敏议事间四雄言有不当王出言似相犯四雄含忍姑容之王久乃知觉谓　上曰诸人若是柰何　上曰不过会蕳而至是耳王曰然明日出与会止勤三日后仍会蕳人事愈疏彼此防疑势将极矣 
　　遇徐州乱雄败其残雄趋濠梁合势共守其残雄势败受制不料俞孙潘郭反屈节以事之未旬月来人各受制前日防疑之事顿然释去后因彭赵僣称王号势在鲁淮赵称名而滁阳王奉鲁淮而轻赵未久俞鲁孙潘闇恃赵威于市衢擒王时　上出淮北闻王被擒急趋审由将抵其舍友人扼道而止曰尔主被擒亦欲擒尔其勿归　上曰再生之恩有难不入何丈夫之为即入见其家止存妇女而巳诸子弟皆匿　上谓妇人曰舍人安在诸妇人亦有疑心佯言不知　上曰我家人也释疑从我谋询知舍人所在诸妇人乃实告　上曰主君平日厚彭薄孙机欲脱此难彭必可求明旦以次夫人携二子往告彭君彭闻忿怒陟■〈目亥〉曰孰敢若是遂呼左右点兵搜强于是　上亦反舍去长服披坚执锐与诸人同围孙宅缘舍上掀椽揭瓦诸军杀彼祖父母于晦窟中得见滁阳王钳足系项股肤被棰打而浮虚令人负归脱去钳锁 
　　是岁冬元将贾鲁围城明年癸巳春贾鲁死夏五月元兵解去时濠城乏粮　上谒友人得盐数引乃泛舟以盐易于怀远得粮数十石以给王家六月归乡里收元兵民人七百余以献王王喜命为镇抚时彭赵二雄以力御众部下皆凌辱人　上以其非道恐七百人有所累弃而不统让他人统之唯拔大将军徐达等二十人有可行而南略定远　上中途染病而归甚危殆半月乃醒瘥方三日滁阳王扶笻过门啧啧有声　上卧闻之问傍人曰王适扶笻而过声意惧惋胡为若是傍人答曰远方有言欲降犹豫未决王知友人在其中令人往视柰何家无可行者故惋惜耳　上虽卧病方瘥未满旬日乃曰王越门而惊必将以我为弃人乎设不以我为弃方瘥何若是之惊耶予尝闻之生我父母活我者亦父母倘不善图为他雄所有功将何建生亦何安于是扶病诣王寝室王曰汝来何为荅曰闻地方有欲归者未定何人我欲扶病亲往曰汝病方瘥未可行　上知王意决行不辞王许之 
　　明日南行入定远及至复病三日而起未瘥速行又六十里抵大桥前病复作亦三日而瘥即日又南行又五十里望见他垒勒兵布阵　上所将者二骑九步步者见彼勒兵甚恐欲舍　上逃归　上谓九步曰彼众我寡况彼马我步我等　至此纵欲逃之将焉归必随我入彼营垒以验吉凶言既彼阵中遣二将来逆举手大呼来者为何　上遣人荅我来为公师首言彼归垒而告师首　上乃下马然以久病步趋艰卒前逄一渠九步中一人欲代　上越渠平凉候费聚是也　上谓聚曰诸人至此生死不得自由岂有代者耶乃同往不踰时而至首帅逆之曰何为而来荅曰彼此无食吾主兵者郭氏与尔故友知汝垒在是亦知他敌欲来相攻恐汝无救特遣吾报肯相从否则移兵避之帅首既听应声愿降请留信物特赐香囊一以此为记良久帅首以牛脯来进食毕帅首告之曰诸帅相从者归为待诸军收拾路费而诣军门　上许之　即帅九步归中留费聚于彼以候人情后三日费聚清晨而至告曰事不谐矣彼欲他往上兵三百帅诣所在谓彼曰彼为他雄所凌其冤未申雠亦未解一旦从我北何不能雪前日之耻特助三百人与雠试较胜负不亦可乎其帅首大悦然而心巳自疑进趋之间刀器不离左右巳防闲矣　上知其情状未易为也非智不得犹豫间里人过其前乃平昔里中之力勇者　上谕之曰吾欲役尔能乎曰能乃受方略佯以帅首来会彼未来时密勑三百人若师至丛而视之往则开而纵之凡此者三于丛人中缚之令壮士五十人密簇而行携离营所去将八里遣人报彼垒中尔帅首往观营地矣眷属当移营就之将焚营废垒揭营而行于是收壮士三千人 
　　七日后帅此三千东破元将老张知院营黎明袭入之老张弃军而遁汉军尽为我有精壮二万余未及旬帅而南入滁阳途中遇太师韩国公李善长诣军门而谒与语知其胸怀必能成事使掌案牍时掌牍者巳数人矣特以善长督之约曰方今羣雄并起吾见羣雄中持案牍者及谋事者多非左右善战之人不得尽其能以至于败羽翼既去未久雄亦亡矣卿智人与决大事掌行文案事无不济善长稽首再拜而谢曰谨受命遂入滁阳 
　　未踰月永义鲁淮二王遣人捉兵以驻旴泗　上知非人弗从未几二王果自相吞并善战者多死鲁淮亦亡惟存永义而巳彼时滁阳王尚受制于旴■〈日台〉几为相吞而卒幸而免焉　上遣一人往说永义从滁阳王南行及至王阅诸军独　上之兵众队伍严整旗帜鲜明甲兵洁利王乃大悦初王首倡义时兵八百人后　上亦以七百部下诸人共招诱者总不过万余　上之兵众比王至时　四方来从者共前所得巳三万有奇 
　　踰两月王为谗所惑略少疑焉掣近行掌文案者数人李郭等皆预先私相同谋皆愿从滁阳王久矣未久又欲以李善长置麾下善良弗从诉于　上涕泣弗行　上谕之曰主君之命欲若要吾首亦不自由汝安敢不行善长终不弃去久之得弗再召幸久相从是后四方征讨总兵之权王不令　上与甲午冬十月元将脱脱围六合被围者请救来使乃　上之友也中夜而至门首　上闻友人至即起诣门所隔与语其门　上所守之要道阖辟之机非王命不敢擅开谓友人曰姑少待吾告滁阳王辟门而进　上往告滁阳王尽诉求救之情王与六合雄旧有雠嫌纔闻求救喑呜奋怒不发救兵来使亦与滁阳王少旧虽尽欣其情亦不允上因与共说之尽言至日昃王怒少解仍令他将统兵以行诸将惧脱脱之威皆不敢帅师假托词以珓白神神皆不许除此之外别无可帅军者 
　　王乃召　上汝亦白神　上曰兵者凶事昔圣人不得巳则用之今六合受围雄虽异处势同一家今与元接战逼迫甚急救则生不救则死六合既亏唇失齿寒若必命我总兵神可弗白于是决出师东之六合与脱脱战微失利归彼时海内称雄者渐广互有胜负不辨贤愚死者甚众　上思之设使胜负不分互有得失如斯久之世无人矣每闻■有鬼神尝云天高听卑是非　鉴见于是发诚专意致词恳祷于　上帝曰时元至正岁在甲午天下大乱生民傍徨生离死绝数非一人战斗之际雌雄不分未见■〈女匽〉兵息民之期盛衰孰巳特竭微诚恳切谨告愿赐覆照以拯生民果元运未终乱雄蚤息或乱雄有人元当即覆然某亦值乱雄无人扰害生民亡自底定乃始致词命黄冠设坛仪俯伏于　上帝前期三月而验 
　　后三月　上兵愈昌时滁阳王名称尚微意在据滁阳而称王号与上虽不明言就中觇视可否　上知其不可槩说滁一山城也舟楫不通商贾不集非古形势非雄所居王乃默然 
　　明年至正乙未春正月戊寅　上率师取和州初兵众乏粮议谋征所时　上固数谏王为人所谮初少被责辱然　上终不以为意必欲成事不免数谏王性聪明其纳言也如流及谗俄说转若发机果受责辱因是致病当议征之际遣人占议因疾不赴召至再三终不能会复遣人至召定计以出三军　上许之谋曰曩征民寨得兵总二万其书庐州路义兵皆故衣希为改作拔其勇者三千衣青衣服皆悬号垂髻左袵佯彼兵复令万人衣绛衣继其后相去二十余里慎探骑谨队伍严号令南趋和阳其城可下王乃善谋如其筭兵将衣青者前衣绛者后青者步陟阳关和阳守备者知报庐州路义兵至耆宿以牛酒迎之其前帅青衣者异其道而饮食帅绛衣者少谋怠智循正道而抵和阳元首帅出师以逆之衣绛之士败走逐北二十里时帅衣青者将抵和阳和阳守帅获胜至暮而归遇衣青衣者至城下际昏合战一皷破之平章帖木儿兵溃遁去 
　　初衣绛者败归报滁阳王曰衣青衣者人皆陷阵滁阳王惊恐责　上失机怒问俄城南元遣使来招滁阳王惊恐益甚召问若何彼时兵出城虚待将至门兵合滁阳王至南门密令稠簇于南街然后令来者入至滁阳王所　上令来者膝行以见王待谕之及其谕也王言非智众议欲杀来者　上白王曰兵出城虚若杀来者彼必知我虚而杀其使也敌反卒至若生纵归还而告之使必逡巡弗知我之虚实王如其言纵放之明旦有人来报元兵遁去王命　上持命往收败军及总守和阳遂急奉命之和阳所帅二千人途中军败闻　上亲往喜复从征者千人南越滁阳门令兵就息喻众曰一兵夜燃十炬务在初昏令罢兵息上单骑帅骁勇者十人暮抵和阳及至知青衣者巳破城而守之是夜入城与诸将议守
