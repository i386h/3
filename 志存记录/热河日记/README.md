热河日记序 
立言设敎　通神明之故　穷事物之则者　莫尙乎易 春秋　易微而春秋显　微主谈理　流而为寓言　显主记事　变而为外传　著书家有此二涂　尝试言之　易之六十四卦所言物　龙 马 鹿 豕 牛 羊 虎 狐 鼠 雉 隼 龟 鲋　将谓有其物耶　无之矣　其在于人　笑者 泣者 咷者 歌者 眇者 跛者 臀无肤者 列其夤者 　将谓有其人耶　无之矣　然而揲蓍有卦　其象立见　吉凶悔吝　应若桴鼓者　何也　由微而之显故也　为寓言之文者因之　春秋二百四十二年之间　郊禘 搜狩 朝聘 会盟 侵伐 围入　悉有其事矣　然而左 公 谷 邹 夹之传各异　从而说者彼攻我守　至今未已者　何也　由显而入微故也　为外传之文者因之　是故曰蒙庄善著书　庄书中　帝王贤圣 当世君相 处士辩客或可补正史　匠石 轮扁必有其人　至若副墨之子 洛诵之孙　此是何人　罔两 河伯亦果能言欤　以为外传也则眞假相混　以为寓言也则微显迭变　人莫测其端倪　号为吊诡 　而其说终不可废者　善于谈理故也　可谓著书家之雄也　今夫燕岩氏之热河日记　吾未知其为何书也　涉辽野　入渝关　倘徉乎金台之墟　由密云　出古北口　纵观乎滦水之阳 白檀之北　则眞有其地矣　又交际其国之硕学韵士 　则眞有其人矣　四夷殊形诡服 呑刀呑火 黄禅短人　虽若可怪而未必罔两 河伯也　珍禽奇兽 佳花异树亦无不曲写情态　而何尝言其背千里 其寿八千岁耶　始知庄生之为外传有眞有假　燕岩氏之为外传有眞而无假　其所以兼乎寓言而归乎谈理则同　比之覇者　晋谲而齐正也　又其所谓谈理者　岂空谈恍惚而已耶 　风谣习尙　有关治乱 　城郭宫室 耕牧陶冶　一切利用厚生之道皆在其中　始不悖于立言设敎之旨矣

渡江录序 
曷为后三庚子 　记行程阴晴　将年以系月日也 　曷称后　崇祯纪元后也　曷三庚子　崇祯纪元后三周庚子也　曷不称崇祯　将渡江　故讳之也　曷讳之　江以外淸人也　天下皆奉淸正朔　故不敢称崇祯也　曷私称崇祯　＿皇明中华也 　吾初受命之上国也　崇祯十七年　＿毅宗烈皇帝殉社稷 　＿明室亡 　于今百三十余年 　曷至今称之　淸人入主中国　而先王之制度变而为胡 　环东土数千里划江而为国 　独守先王之制度　是明　＿明室犹存于鸭水以东也 　虽力不足以攘除戎狄 肃淸中原　以光复先王之旧 　然皆能尊崇祯以存中国也
