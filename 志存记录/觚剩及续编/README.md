　　《觚剩》《觚剩续编》　清 钮琇


　　●自序
    原夫天为石补，光乃丽乎三辰；地以鳌胜，力且维乎四极。莲生岳上，巨灵运掌而山开；乌走云中，后羿弯弓而日落。若其游神六合，抗想千秋，都非易测之情，实有难穷之理。然则庄生《齐物》，何得置北溟而不谈；屈子《离骚》，能无仰东皇而欲问乎？况夫鬼盈睽载，《易》留语怪之文；神降莘言，史发兴妖之论。杏坛书垂笔削，辨六之昼飞；龙门事著兴亡，志一蛇之夜哭。是知虞初小说，非尽出于荒唐，郭氏遗经，固无伤于典则也。余也生虽已晚，世不逮夫郎，思则靡涯，心常倾夫荟蕞。幼而就傅，延吴札于枫江；长且服官，谒徐陵于柏府。初垂缟带，便学长吟。继傍玉台，每聆《新语》。入燕都而怀故国，记览《梦华》；登梁苑而晤名贤，书携行秘。迨夫哀缠素к，贫典黑貂，旅食三年，不断皋鱼之泪；宦归千里，无余刘宠之钱。游迹则雁池凫渚，阅历遍赊；行装则玉格贝编，讨搜独富。既而听鸡函谷，策马芦关。叹密法于秦灰，欲辨怪哉之气；怀和声于周凤，还题吉了之名。碑洞文章，收诸绿笈；桥山陵寝，绘以黄图。今则仍绾银章，更临珠海。鹧鸪啼处，朱旗锦石之乡；蝴蝶飞时，丹灶羽衣之洞。官逢斗孛殳，得虎说于荒江；语习И隅，成鱼吟于蛮府。粲花宾至，快雄辩之当筵；话雨人归，喜华笺之在箧。于是倾觚授简，抄以小胥，因而别地稽时，汇为全帙。言其大略，盖有三焉。尔其箫断吴门，曲留小海；筑摧易水，歌起悲风。嵇散挥弦，《广陵》之音欲绝；潘阁笔，重阳之句无多。苟非阐此嘉名，谁复求之幽壤？至乃江头孝女，黄绢无传；塞外文姬，青蛾莫赎。帘窥燕子，新愁锁夜月之楼；门倚桃花，旧笑忆春风之径。维美昭于绣管，斯艳发乎得簇。更若大夫观止，雅擅多能；君子至斯，夙推博物。疏不遗乎草木，学逆葩经；注兼及乎虫鱼，功期翼雅。爰以资其考索，非止袭夫传闻。然而宇宙茫茫，人如粟渺；江河滚滚，世亦萍浮。目不越于方隅，每以常而为怪；心苟通乎大造，将何幻而非真？念兹得失之林，总归陈迹；悟彼逝来之境，庶得遐观。姑存此日琐言，岂曰珠能记事；倘附他年野史，亦云稗以备官焉尔。

　　康熙庚辰三月既望，吴江钮诱玉樵甫书于高明官署之根青阁。

　　●卷一。吴觚上
    ○岁寒集余幼从吴南村先生于家塾，受《尚书》、《左氏传》及时制义，所见著述甚富，惜在髫龀，不及抄缀成编。未几夫子下世，遗稿零落，越今三十余载；偶于故簏中，搜得数章，敬录于此。《拟道路忆山中》云：“微阳澹遥薄，羁客困尘鞅。帛策今志，烟霞厉曩想。迟迟楚粤郊，一步一回仰。灵境隔紫霄，夙昔凭诞放。坠萼荐春屐，归云丽秋氅。轩开纳曙景，襟豁眷初爽。自生性情悦，愿领丘壑长。浩浩九门津，恢恢千目网。野犊被文锦，虽贵非所赏。铜斗瞻渐碧，禹穴顾已莽。新游非故乡，登顿益慨慷。空抱嵇生琴，戚戚发哀响。”《追和孟东野审交》云：“枳棘初生时，青青似柑柚。滋养待成林，芒刺伤我手。结交非其人，不但初心负。名义如日星，斯人等埃垢。君子田中禾，小人田中莠。同亩而异根，安得秋实茂。宁为管华薄，莫效苏李厚。厥初或依违，鲜终谁执咎。”《咏梅花》云：“百卉千妍竟艳阳，有人篱角问孤芳。寒交近与苍松绝，滥籍秦官亦可伤。”《赋得城春草木深二首》云：“名区屯宿莽，春向几人家。衙晚听啼鸟，碑残卧落花。堂巢无主燕，池唱在官蛙。何必曾侯印，东门遍种瓜。”“百战余荒垒，阳和尚草莱。青帘花外肆，乌鹊雨中台。怨气流为火，腥痕绣作苔。登高如可赋，愧乏鲍生才。”《田家秋兴》云：“九月筑场后，田家事事幽。断桥霜下坂，修竹水边楼。鸡犬无心昼，菰蒲有望秋。莳蔬携插短，栽枳结篱稠。灯火邻舂共，鱼书社饮修。落桑亢人瓮，覆草秉成丘。帘曝贪朝坐，溪云动晚讴。牵萝闲补漏，策杖遍寻俦。梦以无因永，行缘得句留。所须无冻馁，城市有何求。”《寡女行》云：“明月夜照井上霜，如花寡女开中堂。当轩轧轧朝复暮，机中之锦他人裳。夜阑月转灯烬灭，独向空床抱冰铁。君不见东家鹤发娘，垂老嫁人为媵妾。”《西门行》云：“西门有高冢，乃是城中富家儿，当年不乐，为后人嗤。后人嗤，后人仍效之。日复一日，坐令青鬓丝。疏布裘，屈木卮，此物我所制，奈何坐令他人嬉！城中车马客，不向西门冢上驰。即向西门冢上驰，冢中之人不能复起相追随。请尽今日欢，明日非我知。”《东门行》云：“出东门，意欲前，欲前未及得前，远闻儿啼女泣，使我仆马为流连。君但行，勿留连。堂上有两亲，机中有丝日一缣，贱妾糟糠，终不令君亲饥且寒。道傍之水泥浊浊。长安马，多食粟。一车覆，一车续。愿君为臣忠，为吏廉，吹风到茅屋。结驷归来非我欲。”《西山爽气》云：“雨歇池馆凉，开轩睇原隰。遥峰如盥栉，苍翠净可吸。竹树延清晖，苔阶响余滴。群鸟次第起，远背晨光没。我意亦悦之，曳杖遥与即。旷然融心魂，神领兴自给。松风塞双耳，米盐讵容入。出门无可营，坦坦任所适。”《寄题周蛰庵新居二首》云：“周子幽栖处，浮名信可逃。竹才埋瓦屋，水足受鱼ザ。近渚饶菰蒋，低帘见桔槔。从知予去后，双板没新蒿。”“苎通小径，鸡犬自成村。乱水绕三面，数家同一门。塾童来问字，邻叟共携尊。纵遣桃花出，无人解觅源。”《寄故乡诸友》云：“白云红树隐溪塘，何处登高可望乡？百里畏途天外梦，两年愁鬓客中霜。刺秦惭负千金诺，归蜀空回九曲肠。倘有贫交问流落，尚悬双眼看苍苍。”夫子为王昆仑山人之甥，明末食饩归安，名驰三吴、两浙间。鼎革后改名宗汉，号南村，与兄东里、弟西山偕隐，日以文酒相娱，并手辑同志之诗，曰《岁寒集》。先生为人温厚而严正，终其身绝迹城府。吴兴庄氏子聘修《明史》，坚拒之，卒免于祸。

　　○贞白楼诗余族兄易庵先生，名，与先君同受知于黎博庵副使。鼎革后，筑楼溪滨，绕以修竹，而种菊其下，赋诗饮酒，绝意人世。自南村师殁，先君即命余从易庵游。今易庵年已八十有二，无子以继其传，手抄《贞白楼诗》寄余，若冀余之传之者。存其一二，亦可以见兴象之恬，襟怀之峻矣。《闰上巳》序云：“上巳不诗，闰而诗，纪异也。兰亭已矣，禊事犹存，虽迹类嬉游，而志存蠲祓，怀洁之士，实欣尚焉。于斯时也，澄泽流鲜，朗区披秀，洛川名阀，抚令节以舒光；洧水丽姝，激景风而吐艳。斯诚骚迈之欢辰，咏喁之良遘矣。乃者节序乖疏，宾朋散逸，黄龙梦杳，白水吟空。言念曩今，能无涕悼！当右军作记，年冠永和，乃其临流感慨，犹至叹于世殊事异，揆厥所悲，一何遐也。至元嘉祖道之篇，永明禊饮之什，金石毕陈，钧天杂奏，眇寂寥，风斯迈矣。南楼倦倚，髑绪兴哀，漫赋五言，忘其拙率。”诗云：“入春饶胜事，我复爱花辰。曲水门阑旧，熏风节序新。祓除如有待，觞咏岂辞频。寂寞寻芳侣，重来访隐沦。”《上巳日种菊》云：“未肯忘秋兴，霜花手自栽。力锄非种去，悬想故人来。宿土承新露，疏篱傍老梅。相依成密友，日夕共徘徊。”又《秋怀》十首，殊有靖节风致，今录其二：“风霜颇威厉，败叶走阶几。我庭日寥廓，一望尽披靡。人事苦衰谢，天道有倾否。古桂发幽馨，吾无隐乎尔。俯仰观幻化，斯理本如此。”又云：“西风顿凉冷，衣被应装绵。妻儿负单夹，欷当我前。问我我不应，四顾空茫然。南邻贫彻骨，北里富薰天。造物本不齐，岂复关愚贤！”

　　○西山五七言吴西山重晖，行十，南村先生之弟。少年诗名甚著，惜早卒。其诗长于写景，如《野眺》云：“花明背郭寺，柳暗入村桥。”《醉归》云：“荇乱牵摇橹，鸥惊触过船。”《村居》云：“木脱云辞径，禾登鸟集门。”《吴兴旅次》云：“老藤随树倒，危石带沙崩。”《村桥远眺》云：“天长众鸟去，林远细烟升。”《春窗对雨》云：“窗雨沈花影，檐风落鸟声。”《游硖石东山》云：“柳深莺坐稳，柏老鹤巢危。”《晓寒》云：“霜花骄古木，水骨压狂流。”《春夜步月》云：“暗香花隐月，迷路树钩衣。”七言如《题画》云：“远水日斜渔倚棹，空山云起客登楼。”《入山》云：“江寒雁落芦花国，山冷人归枫叶村。”《秋霁》云：“槛外雨过群壑响，檐前风定一蝉闻。”《春怨》云：“惊残好梦憎啼鸟，催理新妆恼侍儿。”《新迁》云：“竹孙脱箨看过屋，鸥母携雏并浴川。”《雨望》云：“废苑亭台新牧陇，孤村烟水旧渔家。”《秋夜》云：“幽砌露寒蛩近榻，荒园槿落犬穿篱。”皆可传也。

　　○布囊焚余张玄著先生起自海中，部落解散，窜身僧寺。杭守臣觇得之，与爱仆杨贯玉、幕友罗自牧同被执。就逮之日，公乌巾葛衣，不言不食者数日，惟啜水而已。临刑，二卒以竹舆舁至江口。先生从舆中出，见江上青山夹岸，始一言曰：“好山色。”因索笔研，作绝命词，付行刑者。自牧与贯玉俱从死焉。所著诗词，贮一布囊，悉为逻卒所焚。其遗在僧寺及民家者仅数篇。近体三首云：“义帜纵横二十年，岂知闰位在于阗。桐江空系严光钓，震泽难回范蠡船。生比鸿毛犹负国，死将碧血欲支天。忠贞自是孤臣事，敢望千秋青史传。”又：“国亡家破欲何之？西子湖头有我师。日月双悬于氏墓，乾坤半壁岳家祠。惭将赤手分三席，特为丹心借一枝。他日素车东浙路，怒涛岂必尽鸱夷。”又：“何事孤臣竟息机，鲁戈不复挽斜晖。到来晚节惭松柏，此去清风笑蕨薇。双鬓难容五岳往，一帆仍向十洲归。叠山迟死文山蚤，青史他年任是非。”《古风一首》云：“苏卿仗汉节，十九岁华迁。管宁客辽阳，亦阅十九年。还朝千古事，归国一身全。余生独不辰，家国两荒烟。飘零近廿载，仰止愧前贤。岂意避秦人，反作楚囚怜。蒙头来故里，城郭尚依然。仿佛丁令威，魂游华表巅。有此面目，难为父老言。智者以为辱，愚者笑我顽。或有贤达士，谓此胜锦旋。人生七尺躯，百岁岂复延。所贵一寸丹，可逾金石坚。求仁而得仁，抑又何难焉！”
