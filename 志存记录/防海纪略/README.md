防海纪略 清 芍唐居士撰





　　西南洋有五印度者，部落亚细亚洲，西汉之所谓身毒，东汉之所谓天竺也。（古音天铁因切，与身本同韵，此为译音之对音。）一名印度海，总名西南（洋）。其南印度有孟迈者，为入东南洋之咽吭，而东中两印度在其北，中印度古名温都斯坦，以兴都哥土山为界，（兴都即印度之对音也。）逾山则西北两印度地，今回部之国在焉。国朝乾隆二十年，英吉利灭东印度之孟加拉，遂以次侵并东中南三印度之地。 
　　东南两印度皆产鸦片烟土，名曰波毕，英之岁饷每年约一千数百万，其初入关口，一箱纳税银三两，沿海居民，争传其法，辗转效尤，不数十年，遂流行于各省。粤督始闻于朝。 
　　嘉庆初，奉诏申立严禁，裁其税额，自此入口之鸦片，悉暗中偷售，而其价益增。后查出叶恒澍夹带鸦片之案，奉旨重申前禁，凡洋艘至粤，先由行商出具，“所进黄埔货船，并无鸦片”甘结，方准开舱验货，其行商容隐查出者，加等治罪。鸦片趸船，遂改泊急水门、金星门等处。勾结内地奸民，往来传送包买，则有窑口。说合则有行商，私受土规则有关汛，为之奥援。包揽运载则有蟹艇，资其护送。洋商易货无多，辄载银出洋回国。内地银荒日甚，朝廷忧之，而一时九卿台谏之列，谓其有伤政体，于是内阁学士朱嶟奏请申禁鸦片例禁，（其奏未见）给事中许球奏谓：“弛鸦片之禁，既不禁其售卖，又岂能禁人吸食？若只禁官与兵，而官与兵皆从士民中出，又何以预为之地？况明知为毒人之物，而听其流行，复征其税课，堂堂天朝，无此政体。臣愚以为与其纷更法制，尽撤藩篱，曷若谨守旧章，严行整顿？自古制夷之法，详内略外，先治己而后治人，必先严定治罪条例，将贩卖之奸民，说合之行商，包买之窑口，护送之蟹艇，贿纵之兵役，严密查拿，尽法惩治，而后内地庶可肃清。若坐地夷人，先择其分住各洋行，著名奸猾者，查拿拘守，告以定例，勒令具限，使寄泊趸船尽行回国，并令寄信该国王，鸦片流毒内地，戕害民生，天朝已将内地贩卖奸民从重究治，所有坐地夷人，念系外洋，不忍加诛，如鸦片趸船不至再入，中国即行宽释，仍准照常互市，倘仍前辙，定将坐地夷人正法。一面停止互市，似此理直气壮，该夷不敢轻视，亦无所施其伎俩。” 
　　鸿胪寺卿黄爵滋奏谓：“近年各省盐务漕务之疲累，官吏之亏空，商民之交困，皆由银价昂，钱价贱，向时纹银每两兑钱千零，今则每两兑至千有六百，而银少价昂之由，由于粤东洋船鸦片烟盛行，致纹银偷漏出洋，日甚一日。此烟制自英吉利夷，严禁其国人勿食，有犯者以炮击沉海中，而专以诱他国之人，使其软弱。既以此取葛留巴，又欲以此诱安南，被安南严令诛绝，始不能入境。今则蔓延中国，横被海内，槁人形骸，蛊人心志，丧人身家，实生民以来未有之大患，其祸烈于洪水猛兽，积重难反，非雷厉风行，不足以振声发聩，请仿周官用重典，治以死罪。” 
　　沼各省将军、督抚会议速奏。时湖广总督林则徐奏最剀切，言烟不禁绝，国日贫，民日弱，十余年后，岂惟无可筹之饷，抑且无可用之兵。 
　　上谓深虑远识之言。诏林则徐来京，面受方略，以兵部尚书，颁给钦差大臣关防，驰赴粤东查办海口，节制水师。 
　　初，鸦片烟在康熙十年以前，以药材上税。乾隆三十年前，每年不过二百箱。嘉庆元年，因嗜好日众，始禁其入口。嘉庆末，每年私鬻者至三四千箱，始则囤积澳门，继则移于黄埔。道光初，严旨查禁，复移于零丁洋之趸船。零丁洋者，在老万山内，水路四达，凡中外商船之出入外洋者，皆必由焉。夷船至，皆以鸦片寄趸船，而后以货入口。又浙、闽、江苏商船，即从外洋贩运，其粤商则皆在口内议价，而从口外运入，始趸船尚不过五艘，其烟至多不过四五千箱，可筹火攻。而总督阮元，密奏请暂事羁縻，徐图驱逐，于是因循日甚。其突增至二十五艘，烟二万箱者，则在道光六年两广总督李鸿宾设巡船之后。巡船每月受规银三万六千两，放私入口，于是藩篱溃决。及道光十二年，总督卢坤，始裁巡船，奈水师积习不可挽。道光十七年，总督邓廷桢复设巡船，而水师副将韩肇庆，专以护私渔利，与夷船约，每万箱许送数百箱与水师报功，甚或以水师船代运进口，于是韩肇庆以获烟功擢总兵，赏戴孔雀翎，水师兵丁，人人充橐，而鸦片遂岁至四五万箱矣。大理寺卿许乃济，曾任广东雷琼兵备道，受大吏指，奏请将鸦片烟照药材收税，不报。 
　　十九年正月二十五日，林则徐驰驿抵粤，传洋商伍怡和，索历年贩运烟土夷商查顿及颠地二人，查顿闻风先窜，惟颠地随英吉利公司领事义律由澳门至省城夷馆。林则徐派兵役监守之，并于省河之猎得炮台，筏断来往，谕令将零丁洋所泊二十五艘之烟土，勒限呈缴，免其治罪，否即断薪水而停贸易。又采粤人舆论，奏革水师总兵韩肇庆之职。公司领事者，英国派夷官管贸易者也。他国皆夷商各自贸易，惟英吉利通国富商，合资出洋，计本而分利，故派领事官一员总管，凡与中国官吏抗衡桀骜，皆领事所为。初议三十年为一局，继议六十年为一局。道光十三年，公司局散，粤中已无领事，而总督卢坤听洋商言，反行文英吉利国，令仍派领事来粤，司贸易。初至者曰劳律，以不法驱逐。再至者即义律，在粤三载，夷商听其号令，至是遂被围在省馆，不能回澳。始于二月十二日，具印禀遵缴，并寄信各夷船，令将驶往东洋之烟，尽驶回粤，共缴鸦片烟土二万二百八十三箱，计每船大者千箱，次者数百箱。于是林则徐会同两广总督邓廷桢，亲驻虎门舟次验收。至四月初六日收毕。每箱酌赏茶叶五斤。其烟土请解京师，旋奉上渝，即在海口销毁，毋庸解京，俾沿海民夷共见共闻，咸知震惊。林则徐会同督抚于虎门监视销毁，就海滩高处，周围树栅，开池漫卤，投以石灰，顷刻汤沸，不爨自燃，夕启涵洞，随潮出海。其鸦片烟共四种，最上曰公班土，白土次之，金花土又次之。每箱四十枚，又有小公班土，每箱一百数十枚，更贵于公班。每箱百有二十斤，共计鸦片二百三十七万六千二百五十四斤，皆产于东印度之孟阿腊，南印度之孟迈及曼达拉萨。且检出印度夷埠发票，有每月发至万有二千余箱者。虽间售南洋各国，而中国居其大半，计载来中土者，每岁不下五六万箱。其烟在印度本地每箱价银二百五十元，至广东则价五六百元，为利一倍。共烧毁本银五六百万元，并利银千余万元。时有各国夷商闻风来观，作文记事，颂中国之政。 
　　林则徐下令尽逐外洋之趸船，与澳门之奸夷，不许逗遛内地，其续到商船，有鸦片者，倘自揣不敢报验，即日回国，亦免穷追，若仍沿旧，以烟卸寄趸船，则入口时丈量船旁水迹尺寸，情伪立见，必照夹带鸦片之新例，人即正法，船货入官，所有进口之船，均应照此具结。并行文与英吉利国王，词严义正。时西洋弥利坚诸国，皆遵具结。于是义律由省下澳，递票言违禁犯烟之弊，亟须设法早除，如准委员来澳，会议章程，可冀常远除绝。林则徐批示奖励，而义律复禀，请本国货船泊近澳门。林则徐以澳门向例，惟准设西洋额船二十有五艘，若英夷援此例不入黄埔，则海关虚设，而私烟夹带，更无从稽察，严驳不许。义律倡言不准泊澳，便无章程可议，因不受所赏茶叶，不肯具结，言必俟奉国王命定章程，方许货船入口。时义律已寄信，附货船回国，往返不过半年，原可少需无迫也。 
　　而五月内复有尖沙嘴夷船水手殴毙村民林维喜之事，谕义律交出夷犯抵罪，义律虽未交出，而悬赏格银千圆，购告殴犯之人，亦非故意抗违也。七月，林则徐与邓廷桢遵例禁绝薪蔬食物入澳，并以澳门寓夷，原为经理贸易，今既不进口贸易，即不应逗遛澳门，义律率其眷属及澳内英夷五十七家，同迁出澳，寄居尖沙嘴货船，于是义律始怨。暗招夷埠兵船二艘来粤，又择高大货船三艘，配以炮械，赴九龙山假索食为名，突开炮攻我水师船，我参将赖恩爵挥兵发炮，击翻双桅夷船一，杉板船二，而奸夷所雇吕宋趸船，逗遛潭仔售烟者，亦于八月初为我水师攻毁，人船并获。义律遂托澳门西夷代为转圜，愿尽遣趸船回国，其货船亦愿具结，如有夹私者，船货充公。此粤事第一转机，而林则徐必令夷书人即正法之语，且责缴凶犯，旋有其国两船遵式具结，验无夹带鸦片，于九月晦入口，而义律遣二兵船阻之，且投禀请毋攻毁尖沙嘴之船，以俟国王之信。水师提督关天培，以凶犯未缴，掷还其禀。时我师船五艘在洋弹压，夷见前禀不收，见我师船红旗，即发炮来攻。关天培令开炮应之，击断夷船头鼻，夷兵多落海死，又击溺柁楼夷兵，夷船旗落帆斜，遁回。尖沙嘴迤北，有官涌山，为我军营垒，颇得地势，轰毙夷兵无数。时十月初旬，我军连胜，夷船恐我顺潮火攻，又水泉皆下毒，无可汲饮，遂灭灯宵遁外洋。前此九龙山之战，奏奉批谕，有“不患卿等孟浪，但患过于畏葸”之语。十一月初八日，诏曰：“英国夷人，自议禁烟之后，反复无常，若仍准通商，殊属不成事体。至区区关税，何足计论。我朝抚绥外夷，恩泽极厚，英夷不知感戴，反肆鸱张，是彼曲我直，中外咸知，自外生成，尚何足惜？其即将英吉利国贸易停止。”并于原奏中“夷船遵法者保护之，桀骜者惩拒之”之语，批谕云：“同是一国之人，办理两歧，未免自相矛盾。”此因禁烟而并断英夷贸易之本末也。上又以大理寺卿曾望颜之奏，欲封关禁海，林则徐力陈不可。自封港以后，英夷货船先后至者二三十艘，皆不得入口，人人怼怨。于是义律于十一月，复遣人乞恩，（言）在粤办事多年，实欲承平。今诸事扰乱，心多忧虑，自后请遵大清律办理，而仍无违于国王之法，乞仍许英夷回居澳门，俟王谕至，彼时再开贸易。此粤事再转机，林则徐以新奉谕旨，不便骤更，复严斥，坚与之绝，其国货船先后起碇扬帆，驶出老万山者，约十余艘，并续至之艘，多观望留连，寄泊外洋不肯去，而粤洋渔船、蛋艇亡命之徒，贪薪蔬之厚值，与鸦片之交易，趋者如骛。
