自序

　　余自庚辰通籍，筮仕星垣，忝缀班行之末；敓冬徂夏，仅八月余耳。阃曹多暇，自朝参外，惟奉教海内贤人君子，时时过从。聆其谠论清议，凡朝常典故及国计民生利病安危，周咨博询；归寓细细札记，至今佩服弗忘。奉使南旋，未经复命；骤罹家戚，随駴国屯。苫块余生，鼎湖泣血，愧不能墨缞从戎。洎留都再造，晋擢西台，复改梧掖；咸以服未终阕，恋恋慈闱，竟未遑趋觐就职也。两经国变，幸免网罗。至蠡城肇兴，日偕姚江夫子赞襄戎幕；而新命宣着，以容台卿贰兼署兵垣。虽稍有补苴，四海横流难搘一木。放废以来，潜身蚕瓮，郁伊塞胸无可消遣。念及年来兆祸酿乱之由，痛愤填膺；唾壶既缺，竹如意亦击碎。酒后耳热，牢骚不平；辄吮毫伸纸，手自疾书，以当衮鈇。然其间情事，皆亲经目睹，或得之故老传闻、名贤手授。搜罗采摭，毫发靡爽；矢诸天日，鬼神鉴临，不敢以平生喜怒恩怨参错其间。纂成，名曰「荷闸丛谈」。千载而下，览者庶几知我、罪我，或亦有以谅予志也夫！

　　本朝远过前代

　　高皇帝以布衣起淮右，提三尺剑，与群雄角而胜；遂驱蒙古，还之大漠。日月涤而重辉，乾坤辟而再位。然且圣不自圣，鸡鸣而起，未辨色而视朝；六宫取充位，无燕赵二八之奉；赐租之诏，无岁不下；一起阅江楼，楼成而辄罢之，终其身不为离宫别馆。滇诏在卧榻十七年而始平，朝鲜、安南阻日本，闭关听其自服，而不开边衅；废丞相，析中书省而六之；五都督府握兵籍而不与调发，兵部得调发而不握兵；外戚食国租而不与政，南面三十年而不改元，薄海称臣妾而不加徽号；虽神武盖一世，而抑其心以下贤；尊在九重，而洞彻其神以与闾阎之志气通；此弇州山人之颂也。李本宁太史曰：『有以匹夫得天下者，未有以江左一天下；有以中华兼漠北者，未有中华胥为左衽，而能驱除之匹夫起江左。用夏变夷，德泽宰天下，而制之若制子孙，垂祀三百，伟哉高皇帝之烈也，万世一人矣』。

　　谢文肃铎云：『本朝政体，度越前代者有数事：公主寡不再择婿，中官有宠者赐袍带不与朝臣并任，王公不敢擅杀人，重臣不得自辟下僚，文庙不用塑像易以木主，岳镇海渎之神不加封号，文武官员不得挟妓』。陈眉公继儒云：『更有十事：边镇督府以文臣为将，天子自为居守，内阁六卿外廷会推，内廷不得专擅，母后不称制，勋戚不干政，皇子讲官郎宫坊僚寀，不立博望苑，不开天策府，无殉葬，生员不讦利病，不用黥鼼腐刖之刑，京师有热审，直省有恤刑，非大典不轻赦，三品以上始荫子入监，纨裤不得滥朝籍。善乎平涵朱相国之推言也。神武要于不杀，惨舒协于时中，纪钢挈于独断。声律身度，自强不息，与道合真，与天同运。夫子集群圣之大成，太祖又集古帝王之大成，至矣无以尚矣。然则定鼎卜历，宜垂裕无疆，而何甲申之春，■〈日少〉寇一入，燕京化为异域。若长江天堑，国有人焉，则东晋再造，南宋偏安，庶几复睹，而佥壬窃柄，国是诪张，金陵王气黯然消，而夏社竟屋矣。人谋固不臧，天意亦难问，将所谓参夷瓜蔓，咎在文皇者，其信然聊！抑冥冥之中，尚犹未定耶』！

　　陈太史明卿壬午殉难列传叙

　　古来死忠，代不数数。商惟墨允，汉独龚胜，唐家河北无一义士，宋季南朝止李侍郎；未有开创未逾再传，而忠义千古为盛，如昭代逊国之际者也。当靖难师起，以诛错为名，独齐、黄两人耳。洎天下大定，榜列奸臣，前后凡五十一人。盖自推戴劝进外，人人荀息豫让矣。而引绳披根，株连瓜蔓，泽量若焦，狐猿夜嗷，参夷之憯，于是极已，则皆孝孺十族之言有以激之也。愈激愈杀，愈杀愈激，至于断舌碎首，湛宗燔墓而不悔，而万乘之威亦几于殚矣。夫以逊顺天，非若萧墙之喋血也；以旦代辟，非若易姓之受命也。建文君何以得此于诸臣哉，则皆高皇帝所培养之忠臣义士也。当平元捷至，诏群臣尝仕元者不许贺。建余阙李黻祠，赠谥福寿，每岁遣官致祭，而履声橐橐之老臣，则愧之以文天祥，而终窜以死，盖激励若斯之至也，三十年亘万世矣。食报之厚，不亦宜乎？昔人有言：『平时有犯颜敢谏之士，则临难必有杖节死义之臣』。有国家者欲收忠义之报，其先务储养哉！

　　列朝年号辨误

　　宋太祖改元干德，令择前代未有者，后见宫人镜背有干德四年，窦仪知为蜀物，因叹曰：『宰相须用读书人』！高宋、孝宗初欲用炎兴、干统，以刘先主契丹尝有此号，乃复更议。若梁主张重华，南唐贼张遇贤，宋贼方腊，俱号永乐。契丹述律，元出帝，俱号天顺皇帝。西夏号正德，本朝用此三号。儒臣失考之过也。春雨堂杂抄曰：『孝宗升遐，武宗以正德改元，出于刘少师健所定。马端肃公文升在吏部考选，以宰相须用读书人命题讽之』。酌中志略云：『熹宗时，大内司钥库积有历代古钱，一日御前于旧镜内得天启大小钱数枚；色甚古。司礼李永真、向若愚说之，因考历代史并玉海等书，始知南梁萧庄，拓跋魏元法僧，并南诏俱曾有此号；亦宰相不读书之过耳。惟萧皇入继纪元，内阁初拟明良，次绍治，次嘉靖，上特用嘉靖云。乃天顺天在上、景泰天在下，万历为盈数，泰昌有两日之象，皆未为协。威庙时以崇祯纪元，而任温体仁为相，用法苛刻少恩，京师人口语「崇皇帝，温阁老」，崇与戎同音，温与瘟同音。父老至今犹为叹息』。

　　南北都大内宫殿考

　　洪武十年，大内宫殿成，阙门曰午门，翼以楼观，中三门东西为左右掖门，午门内曰奉天门，门之左右为东西角门，内正殿曰奉天殿，上御之以受朝贺，殿之左右有门，左曰中左门，右曰中右门，两庑之间，左曰文楼，右曰武楼，奉天殿之后曰华盖殿，华盖殿之后曰谨身殿，殿后则干清宫正门也。奉天门外两庑之间有门，左曰左顺门，右曰右顺门。左顺门之东曰东华门，内有殿曰文华殿，东宫视事之所也。右顺门之西为西华门，内有殿曰武英殿，上斋戒时所居也。制度皆如古而稍加增益。

　　永乐初，营建北京，凡庙社郊坛场宫殿楼阙规制，悉如南京而高敝壮丽过之。复于皇城东南建皇太孙宫，东安门外东南建十王邸，通为八千三百五十楹。自永乐十五年六月兴工，至十八年九月工竣。酌中志略云：北京大内规制，皇城南向，曰大明门，稍北向东曰长安右门，再北向东曰东安门，北向曰北安门，俗所称厚载门也。转而向西曰西安门，再南向西曰长安右门；此外围六门也。墙外周围红铺七十二处，登闻鼓院则在长安右门之外。紫禁城南向第一重，曰承天之门，二重曰端门，三重曰午门。魏阙两分：曰左掖门、右掖门，向东曰东华门，内有殿曰文华殿；向西曰西华门，内有殿曰武英殿。北向曰玄武门，此内围八门也。墙外周围红铺三十六处，护城之河绕焉。大明内曰承天之门，其内之东一门内，则太庙也；西一门内，则太社、太稷也。端门内则六科也。东曰阙左门，再东曰松林，则会推处。西曰阙右门，其居中巍然向明者为午门，钟鼓在焉。左掖门、右掖门之内居中南向者，曰皇极门，即奉天门，铜壶滴漏在焉。其左曰弘政门，即东角门也，右曰宣治门，即西角门。居西向东曰归极门，即右顺门也；居东向西曰会极门，即右顺门也。从归极门东南入曰六科廊，东西两房掌司所居，精微科及章疏在焉。会极门凡京官上本、接本俱在此处，从会极门南入曰内门，辅臣票本清禁之所，宣庙赐有文渊阁印一颗。皇极门内居中南向者曰皇极殿，即奉天殿，金砖玉瓦；左向西者曰文昭阁，即文楼；右向东者曰武成阁，即武楼；中有渗金圆顶者曰中极殿，即华盖殿也；殿之两傍，东曰中左门，召对或在此；西曰中右门，再北曰建极殿，即谨身殿也。皆嘉靖时重修改名，殿居中向后，高踞三躔白玉栏干之上者，为云台门；两傍向后者东曰后左门，西曰后右门，即云台左右门也。又名平台，凡召对阁部等官或于平台，即后左门。又东则景运门，西则隆宗门。南向者则仁德门，为干清宫，左右金狮各一，入门再升白玉石栏干，直至干清大殿，丹陛下近站台有一洞通东西行走者，左曰日精门，右曰月华门，左小门曰龙光门，右小门曰凤彩门。左有殿曰昭仁，右有殿曰弘德。再北曰交泰殿，其渗金圆顶亦犹皇极殿之制。再北坤宁宫，皇后所居也。左曰景和门，右曰隆福门。再北左曰端则门，右曰基化门，便接琼苑左右门矣。坤宁门仍有中门，向后拍闭而不开。干清大殿之左向西曰端凝殿，右向东曰懋勤殿。坤宁之后则后苑也，钦安殿在焉，供玄天上帝之所，鱼池山子，奇花异卉，景色可人。东南曰琼苑左门，西曰琼苑右门。钦安殿后曰顺贞门，其墙外则紫禁城之玄武门，每夜之更鼓在焉。此大内宫室之略。尚有慈宁、慈庆等宫，为皇太后、皇太妃所居。翼坤宫为皇贵妃所居，承干、锺粹等宫，为东宫并东宫妃所居。永和宫，为诸王所居。皇城北安门内寿皇殿之南，则万岁山，俗所谓煤山也；皆土渣堆积而成，山上树木森郁，鹤鹿成群，内有红阁，列皇帝殉国之所，鼎湖泣血，千古痛心。山之前曰万岁门，再南曰北上门，左曰北上东门，右曰北上西门，再南过北上门则与紫禁城玄武门相对，北上西门之西有大高玄殿，其前门曰始青道院，左右有牌坊曰先天白镜，曰大极僊林。又有阁二座，左曰炅明阁，右曰■〈水月〉灵阁。殿之东北曰象一宫，乃世庙玄修之所名。西内又西曰西直房，即襮房也，武庙崩于此，其东则西下马门矣。玉河桥即金海桥，有坊二：曰金鳌，曰玉蝀。再南曰五雷殿，即椒园也，亦名蕉园，凡修实录成，于此焚草。其内府各监局二十四衙门，在紫禁城内外，不概列。
