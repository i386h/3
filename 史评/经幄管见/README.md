　　提要

　　（臣）等谨案经幄管见四卷宋曹彦约撰彦约字简甫都昌人淳熙八年进士薛叔似宣抚京湖辟为主管机宜文字累官宝谟阁待制知成都府宝庆元年擢兵部侍郎迁礼部旋授兵部尚书力辞不拜以华文阁学士致仕卒谥文简事迹具宋史本传是书盖彦约侍讲筵时所辑皆取三朝宝训反复阐明以示效法盖即范祖禹帝学多陈祖宗旧事之义考仁宗天圣五年允监修王曽之请采太祖太宗真宗事迹不入正史者命李敬等别为三朝宝训三十卷宝元二年十二月诏以进读嗣是讲幄相沿遂为故事彦约是书于进读符瑞诸篇虽不免有所回护要亦当时臣子之词不得不尔其余诸篇则皆能旁证经史而归之于法诫亦可谓不失启沃之职者矣旧刻散佚久无传本惟永乐大典尚载其全文今详为校雠厘成四卷间有辨证各依文附着焉

　　经幄管见卷一

　　（宋）曹彦约 撰

　　寳庆元年九月十七日初供职同侍讲范楷候对是日读寳训谨外戚篇　景德四年上谓近臣曰每岁承天节皇诸亲为姻族求恩多过有希觊朕常念羣臣戮力尽瘁或逺在边防久厯岁时非功状显著未尝进一资一级此若尽遂所请甚紊公道自后奏封有越例者即令内省勿复降出陈尧叟曰臣等比来多见妄有陈乞如秦国公主为子求刺史诸子岁岁改转又图舟车悉免税算去年程继宗掌致逺务坐事致鞫晋国公主亟为求替如此之事陛下皆寝其奏中外之人知陛下推公御人不以亲疎为异上曰诸亲多引先朝为比朕谕以太祖开创之始太宗英睿特断朕安敢上拟今庶事动立制度朕遵守之何敢失坠若从越例之请外人必有窃议迩来渐似知非各安本分矣臣读毕口奏人主遵奉家法吝惜名器未有不身致太平者真宗皇帝有太祖太宗家法可守更于家法上倍加吝惜推此心以往事事节省谨之于外戚必将谨之于宦官谨之于宦官又将谨之于宫女上而服御下而赐予无所不谨则无所不省财用安得不裕民力安得不寛此咸平景徳间所以为本朝极盛也
　　二十二日同侍讲范楷候对是日读寳训受符瑞篇　寿州献緑毛龟【按事在太宗至道三年】上谓宰相曰朕观之介虫而毛得非天意有所警戒耶吕端曰臣等寡闻天意冥邈胡能究知然尝闻师曰圣人之意与天合符茍睿思徘徊属于一物必神灵肸蠁有所启焉今神龟所得之处地名寿春陛下顷升皇储实自寿邸由是言之则是龟也特为陛下而言将使陛下后天而老既寿且昌乎又龟在水族义主乎阴介者刚物也毛者柔物也夫夷狄之类皆禀阴气天戒若曰将有强梗不宾之虏柔伏来庭且陛下自即大位首念西北之人困于飞挽虽继迁凶羯亦降使推恩不问前罪且继迁亡命日久亦厌兵矣胁从之党亦厌乱矣悛心革面匪朝而夕北戎倔强为患滋深部族携离亦荐饥歉必恐相率怀柔愿伸欵附望陛下钦若天意弥阐皇猷虽休勿休日谨一日则灵物之出岂徒然哉臣读毕口奏用兵之害不但两军相加肝脑涂地而已飞挽之劳不减锋镝暴露之久甚于临敌以至边民避难永业渐废强壮应募耕夫渐少流离转徙使老弱疾病之人少有全者怨气所感多致旱蝗饥饿疾疫而死者又不可胜计简册所载未必详尽惟身厯而后知之此寝兵所以为帝王极功也　景德四年上谓宰相曰前诏诸路无得献祥瑞近日颇逾旧制当令礼部申禁之宰相曰福应之至以显盛猷虽睿徳谦冲务于自损若史臣不纪来世何观望止令报礼部闗送史官上勉从之臣读毕口奏祥瑞之与灾变相为有无有祥瑞则无灾变有灾变则无祥瑞古人以晏安为酖毒以谴告为仁爱盖覩祥瑞则易至自满遇灾变则易生恐惧昔魏相相汉宣帝凡四方逆贼灾变之事无不奏言之正得此意真宗皇帝禁诸路无献祥瑞圣意深逺惜乎当时大臣尚欲闗送史官也　大中祥符元年正月天书降左承天门上召近臣对崇政殿西序谕其事王旦曰陛下以至诚事天地以仁孝事祖宗恭已爱人夙夜求治以至殊方修睦犷俗请吏干戈偃戢年谷屡丰皆陛下兢兢业业日谨一日之所致臣等尝谓天道不逺必有昭报今者神授秘文实彰佑徳之应然兹事非常简策所无又未审所谕之事启封之际当屏左右上曰既有天命须当祇受适恐皇城司遽便收进已使止之朕当诣门望拜焚香跪受所云屏人以启虽云勿泄天机朕以为上天所贶当与众共之旦曰葢未测书意不欲显示于众上曰天若谪视阙政当与卿等祇畏改悔若告戒朕躬亦当责已自修岂宜隐之使人不知乎当便与卿等启读但虑文莫能辨宜访明习篆籕之人以从旦曰陛下肃奉天命非臣等所能测度臣读毕口奏天人相与之际无有间断人主与天为一得知阙政所在必将修徳用贤以补治之此周宣遇灾而惧所以王化复行也若掩覆不已如愚人之讳病其初唯恐人知其后遂至于不可措手当大中祥符时宁有阙政真宗皇帝恐惧修省如此天下安得不治　上曰朕自即位每祠祀祝词但以人民为心未尝别有祈请今内殿道场亦致宻词惟求年谷丰穰人民康乐因出示旦等旦曰自古秘祝之词皆有所请陛下以亿兆为心有以见子育之意超絶前古臣读毕口奏得人心而为天子则社稷自然巩固若福萃于人主而灾徧于天下为人主者亦何安于此也　天书扶侍使丁谓言崇政殿阅新制法物双鹤度天书辇飞舞良久翌日上顾左右曰昨所观鹤止于辇上飞度若云飞舞良久文则文矣恐不为实当易此奏也王旦曰陛下以至诚奉天以不欺临物正此数字所系尤深帝皇徽猷莫大于此望付中书载于时政记上俛然从之臣读毕口奏人主奉天临民以不欺为主若以鹤度为鹤舞欺天罔民不特天鉴昭昭民心不服滥觞不止必将有如汉臣指鹖雀为神雀者矣
　　十月初三日便殿聚讲同侍讲陈贵谊候对是日读寳训崇祀礼篇　淳化三年秘书监李至上言国初举藏氷之礼修司寒之祭常以四月行之按诗豳七月四之日献羔祭韭周以十一月为正四月即今之二月春秋传曰日在北陆而藏氷谓夏十二月日在虚危也献羔而启之谓二月春分献羔祭韭始开氷室也火出而毕赋火星昏见谓三月四月中也又月令仲春天子献羔开氷先荐寝庙详其开氷之祭当在春分乃有司之失也上曰今四月韭长可苫屋矣何谓荐新耶即命正其礼臣读毕口奏夏正建寅人道之所便周正建子天道之所始自古未有兼用之者惟豳七月之诗作于成周之时乃上述后稷先公事一篇之中既有夏正又有周正如七月流火九月授衣七月食瓜八月断壶之类凡以月言者皆夏正也一之日觱发二之日栗烈二之日凿氷冲冲三之日纳于凌阴之类凡以日言者皆周正也【按经文凡阳生以后之月言日凡阴生以后之月言月四月言月者阳极阴生当夏至进退之期也苐谓杂周正夏正虽本朱子义似未尽】李至所言引用献羔祭韭一事已合经意臣恐后人疑其诗杂周正夏正故为别白言之　至道二年礼仪使宋白上言按仪注朝享太庙皇帝先诣罍洗奠瓒郊祀未诣罍洗先奠玉币于礼未叶欲请先诣罍洗上览奏遽召白洎宰相问之曰前代祀上帝未罍洗而先奠玉币于礼可乎【按杜佑通典开元礼祀圜丘仪于进熟之先太常卿引诣罍洗在奠玉币后宋初尚沿其旧】吕端等曰王者亲执玉帛以见上帝且玉帛者接神明之贽尤宜蠲洁若未盥沃而奠献殊失恭虔之意宋白曰傥允臣所陈止一次登坛上愯然改容曰朕亲奉大祭竭诚尽物葢为苍生祈福倘变礼为允固当依卿所奏如合遵旧典朕虽百次登亦不为劳端等皆言礼官所陈得中遂从白议臣读毕口奏宋白请郊祀先盥沃而后奠献从礼之厚也以两次登坛为劳而以一次为便礼之杀也太宗皇帝竭诚尽物以事上帝岂以再登坛为劳而变其礼哉其意则是其言则非吕端等取其意而不取其言始足以释太宗之惑人臣献策意是而言非足以害事不可以不谨
　　十二日同侍讲范楷待对是日仍读崇祀礼篇　初礼官草封禅仪上览之谓王旦曰封禅久废非礼典备具岂为尽美朕遍览所定仪注有二十余事虑未合宜已手诏改定如未备者当复议之虽有司已经讲肄其如未尝躬习王旦曰大凡祀事未闻帝王自为仪升降之节止在有司上曰王者事天如子之事父臣之事君寅恭之志岂惮于劳及亲习毕又以御札记三事付有司曰行事官避朕褥位而不顾昊天上帝之位但昭报天地严配祖宗岂于朕躬过为崇奉当谕行事官无得更有回避朕以封禅之事不比常祀自十一月朔即御蔬膳【原注大中祥符元年十月二十四日辛亥有事于泰山其月以戊子为朔初四日辛卯车驾发京师恐御蔬膳当在十月朔诸本皆作十一月朔者误也】旦曰陛下躬行大礼曲尽严恭然日月尚遥道途渉寒虑非保卫之道况南郊亦祀天地不闻预屏荤茹望俟致斋或散斋方进素膳旦等累表陈前上曰朕志已定不烦固请及升泰山毎道经险峻必降辇徒歩所司议增扶卫皆却之导从者或至疲顿上词气益壮及行礼侍从导卫悉令减去拂翟止于壝门烛笼前导亦撤之及禅祭自山下歩出大次侍臣言山路峻滑靴袍而行跻升匪易请乘歩辇上曰接神在近敢不徒行五使等复固请上终不许臣读毕口奏封禅之说始见于管夷吾之书以无怀氏伏羲氏为首莫知其有无也设或有之是时礼文简畧固无伤财动众之事至三代之季齐威公偶欲行之廹于世变非昔者洪荒之比虽泰山在其近境管氏已知其不可所谓一茅三脊鱼比目而鸟比翼者皆饰辞以为难耳后世乃指为实迹更求祥瑞然泰皇汉帝因行幸而为之犹是简径至光武溺于谶纬更作宻词于是玉牒玉检之文金镂金泥之制纎悉备具为其臣梁松等所鄙至唐髙宗则又从以宫女饰以锦绣尤更可笑今真宗皇帝亲制仪注又躬自肄习御蔬膳于两旬之前却歩辇于峻滑之地凡所以事神者无不恭恪凡所以自奉者无不谦退可谓万世封禅之标准也但近世封禅之礼事大体重必须伤财必须动众不可以无怀伏羲氏为口实必如真宗之时家给人足上恬下熙而又以孝敬行之始为无玷舎是不足以格神秖足以害民也
