　　卷一商约一 
　　关章交涉为全国国际争存之要义。而或因一隅以变换全局者，多在沿边；或因全局以进趋一隅者，多在腹地。皖据长江之中，左苏右赣，首鄂尾宁，雄富虽亚于汉、沪，而游历之客轮舰之往来，上至赣、鄂以上，下至宁、苏以下，道必径于是。以是有约之国，约允之件，文传络绎，滋益以繁。先是咸、同之间，但以驻省安庐道兼理其事。逮光绪二年烟台英约成，巡抚裕禄奏请于省城特设洋务局，遴委深于交涉者候补道一员为之坐办，仍会同安庐道遵章办理。自坐办以下设翻译二员，应对二员，差遣四员：书记、收发、学习、行走称是。自局之设，三年一考其成，援例叙劳，择尤请奖。三十年来江面无洋氛。中间国际之争，荦荦大者，非皖一省之故。自皖而纪交涉，特条约之互见者尔，不则其别出者尔。今谨就约章所订，当事执行及地方官应行保护属于皖省者，分三大要，曰商约，曰教务，曰合同，而以其类属焉。 
　　皖省交通之埠三：有已设关开埠者一，芜湖是也。有初允停泊上下客货，旋允开埠而讫未开者一，安庆是也。有但允停泊上下客货者一，大通是也。溯查互市通商之约，始于道光壬寅白门之役，实始有沿海五口而未及内港。长江内港通商之约，始于咸丰戊午天津之役，实始有长江三口，而皆非皖境。皖省三口之约，始订于光绪二年英人之要求。即烟台之约，北洋大臣李鸿章与英使威妥玛议定。约内开，芜湖允开口岸，大通允作停泊处所上下客货，安庆如之。安庆之允开口岸则在二十八年商约，大臣吕海寰、盛宣怀与英使马凯在上海议定。时上海开埠已三十余年，汉、镇、九设关亦将二十年。皖之有交通商埠视各省为稍后矣。然扬子江一带，外人已习如户闼，指定之口，动中肯綮，故芜湖关一开，屹然与上九下镇鼎立为长江巨埠。关开于约允之明年二月十八日。其监督关防文曰，芜湖关监督兼管，新设税务之关防，即以该管道员兼充，逐年税入按结申报抚院，奏咨解部。是关所议十八则，初名《试办章程》，其中如与旧开之汉、镇、九三关，新开之宜昌一关轮收船钞一节，均经抚署及南洋大臣迭次函电与英驻使磋商往复，始经就绪。厥后二十八年乃有修改《长江统共章程》之约，按之芜湖专章，歧异尚鲜。兹以专章为主，其修改各章分注于各条之右，俾备参考而资遵守云。 
　　一、凡洋商船只，只准在芜湖西门外小河北岸沿江离东岸在一里路之限内，陶家沟之北、弋矶之南停泊起下货物。新章：凡船只进口，务遵理船厅指定之处照章停泊。其本口停船之界，由各口酌情自定注明。其进口时，尚未停泊以前，一切拨船、舢板等小艇均不准驶傍该船。按，此未泊以前各艇不准驶傍该船，系增修。 
　　一、凡拨货之船，必先在本关挂号注册，用汉、英文字将该船号数大书船头、船尾，以便稽查，方准拨货。按，新章与此全同。另一条，凡有江照之轮船进口时，拟将船货拨入挂号拨船或趸船栈房内，听候货主照章取货，则准照本关专章办理。 
　　一、各船在口内时，由关派人看守。夜间随时将其舱封固，不准起下货物。至起货完竣，看舱之后方准装货，不得一面起货一面下货，以致混淆。新章：凡船只在口，除领有江照之船外，其他船只均须俟进口之货起清方准下出口之货［物］。按，此并统未领江照一切船只，均须先进口货物后出口货［物］，起下先后尤为分晰。 
　　一、凡商船起下货物，均须日间，不得在日出以前日落以后。礼拜日、给假日皆不准行。其内江轮船请有准单者，起下货物以夜间十二点钟为限。倘未领有准单私行起货下货者，由关查出，将该货入官。新章：凡船只起卸货物或上下压载之石铁等件，均须有准单。如查出并无准单私行起卸上下者，即应查拿入官。其领有准单者，由日出至日落均可任便起卸上下。此外，或于夜间或礼拜日暨照章封关之期，若无专单，均不准起卸上下。按，此礼拜日、给假日、夜间封关期内有专单者准其起卸上下，并铁石压载等件有准单亦如之，系增修。 
　　一、两船欲行互相拨货者，必先请有本关准单方准互拨。如未领有准单私自拨货者，将货入官。按，此新旧无改之章。 
　　一、凡商人领照下货，因船已满载复行退回者，须将该货停泊关前，俟本关验明，发给退货单，方准起回上栈。按，新章与此无异，故不再录。 
　　一、凡有镇江护照商船及江照轮船抵芜湖者，不论有无货起下，均须将照呈关，查验放行。新章：凡有江照之轮船并不起下货物者，可由关查核该照后交还出口。按，此亦新旧无改之章，惟除有镇江护照各船外，其长江专照轮船有由各该国领事官报明本关者，有由该船主呈关请验者，须参看第九条、第十二条。 
　　一、凡各商装运茶叶至芜湖出口者，饬令该商照章完纳出口正税，取号收缴关发给准单，方准装入轮船。惟查茶叶一项，多系由芜湖运往上海转运外国之物，所有应完复进口半税，该商照章呈具保单，即可免其完纳，以省纸笔纷繁。新章：凡茶叶装于江照轮船复运进口者，无庸立在该口完纳半税，准由货主照数立具保结。若于一年限内实将该货复运出口，呈有单据，即将原结注销。倘逾期未运出口，应由该关按照保结补收复进口税。若所运茶叶限内复出此口再进彼口，此口注销原结，彼口应令该商再立保结，亦须于一年限内复运出口，再行注销。以此类推，至货主呈请销结，应于茶叶已运出口七日限内报关，呈明（明）注销。按，此专声明茶叶半税办法，系增修。
