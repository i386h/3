　　史部十三

　　明宫史

　　政书类二

　　仪制之属

　　提要

　　（臣）等谨案明宫史五卷旧本题芦城赤隠吕毖校次毖始末未详考明末有吕毖字贞九呉县人尝撰事原初畧三十四卷序题崇祯甲申姓名时代皆相合疑即其人然艹茅之士何由得详记禁庭事殆不可解毋乃奄人知文义者所私记毖依附其门为之编録故不称撰述而称校次欤其书叙述当时宫殿楼台服食宴乐及宫闱诸杂事大抵冗碎猥鄙不足据为典要至于内监职掌条内称司礼监掌印秉笔秩尊视元辅权重视总宪云云尤为悖妄葢歴代奄寺之权惟明为最重歴代奄寺之祸亦惟明为最深二百余年之中盗持魁柄浊乱朝纲卒至于宗社郊墟生灵涂炭实为汉唐宋元所未有迨其末造乆假不归视威福自専如具固有遂肆无忌惮笔之于书故迹其致亡之道虽亦多端要以宠任貂珰为病本之所在也我朝自列圣以来家法森严内竖不过供洒扫或违律令必正刑章不待于逺引周官委权冡宰而干纲独握宫掖肃清已足垂法于万世乃犹防微杜渐虑逺思深特命缮录斯编登诸册府着前代乱亡之所自以昭示无穷伏考尚书有曰殷鉴不逺在夏后之世诗大雅有曰仪鉴于殷亦越汉之髙帝使陆贾作新语亦曰着秦之所以失与我之所以得葢时代弥近资考镜者弥切也皇上于内殿丛编捡逢是帙辟其缪而仍存之圣人之所见者大矣谨恭録谕旨弁冕简端仰见衡鉴亲操折衷众论勒千古未有之鸿编皆义主劝惩言资法戒非徒以雕华浮艶为蔵弆之富也

　　总纂官 （臣）纪昀 （臣）陆锡熊 （臣）孙士毅

　　总校官 （臣）陆费墀

　　●上谕

　　乾隆四十七年四月十七日内阁奉上谕昨于养心殿存贮各书内检有明朝宫史一书其中分类叙述宫殿楼台及四时服食宴乐并内监职掌宫闱琐屑之事卷首称芦城赤隠吕毖较次其文义猥鄙本无足观葢明季寺人所为原不堪采登册府特是有明一代粃政多端总因阉寺擅权交通执政如王振刘瑾魏忠贤之流俱以司礼监秉笔生杀予夺任所欲为遂致阿柄下移干纲不振每阅明代宦官流毒事迹殊堪痛恨即如此书中所称司礼监掌印秉笔等竟有秩尊视元辅权重视总宪之语以朝廷大政付之刑余俾若軰得以妄窃国柄奔走天下卒之流冦四起社稷为墟伊谁之咎乎着将此书交该总裁等照依原本抄入四库全书以见前明之败亡实由于宫监之肆横则其书不足録而考镜得失未始不可藉此以为千百世殷鉴并将此旨録冠简端钦此

　　明宫史卷一

　　（明）吕毖 撰

　　○宫殿规制

　　皇城外层向南者曰大明门与正阳门永定门相对者也稍北过公生左门而向东者曰长安左门再东过玉河桥自十王府西夹道往北向东者曰东安门转而过天师庵草场转西向北曰北安门即俗称厚载门是也转而过太平仓迤南向西曰西安门再南过灵济宫灰厂向西曰长安右门红栅之内门之北则登闻鼓院在焉此外围之六门墙外周围红铺七十二处
　　紫禁城外向南第一重曰承天之门每年霜降后吏部等衙门朝审刑部重囚在门前中甬道之西东西甬道之南五府等衙门坐东向西吏部等衙门坐西朝东吏部主笔者第一座刑部正堂第二座都察院第三座余以次列之南二重曰端门三重曰午门魏阙西分曰左掖门曰右掖门转而向东曰东华门向西曰西华门向北曰玄武门此内围之八门也墙外周围红铺三十六处每晚有勲臣一员在门左门内直宿每更官军提铜铃巡之而护城之河遶焉
　　皇城内自北安门里街东曰黄瓦东门门之东街南曰尚衣监街北曰司设监再东曰酒醋麫局曰内织染局曰皮房纸房曰针工局巾帽局曰火药局即兵仗局之军器库也再东稍南曰内府供用库曰畨经厂曰道经厂曰司苑局曰钟鼓司再南曰新房曰都知监曰司礼监司礼监第一层门向西与新房之门一様门之内稍南有松树十余株者内书堂也先师位供安向南其楹联曰学未到孔圣门墙须努力儧行几步了不尽家庭事业且开懐丢在一边圣人位之北一间则内书堂教书词林先生讲憩之所也厨房设在后凡米肉食料俱取办于光禄寺内书堂稍北曰崇圣堂再北向南者则二层门矣入此门再东朝南者斯本监公厅之大门也门外有东西二井递封汲之西井之西一小门东井之东一小门其内皆提督监官典簿文书房掌司所居房屋也古书籍名画笔墨绫纱纸札各有库贮焉新房之北司礼监也新房之南御马监也所谓新房者东西一街南北一连二连三连等连连之十字路口各有井御马监之南向西者曰杆子房曰北膳房曰暖阁厂之东门可通河而西门最高大启闭不便遂于大门扇上复开一小门以便行走河之西岸榆栁成行花畦分列如田家也曰南膳房再南曰冥器厂曰混堂司曰内东厂曰尚膳监拨子斜对御马监向东者曰北花房亦办膳之所也曰印绶监曰中书房曰蹴圆亭曰内承运库此库掌印佥书掌司写字诸人所住之署也此路总名之曰东河边尚有房八区则司礼监秉笔随堂众住所谓河边直房是也过东上北门东中门街北曰弹子房曰学医读书处曰光禄寺街南曰篦头房再东则东安门里俗所称墙门者过桥则东安门也自东至南武庙恒临幸之门之东曰重华宫其前曰重华门曰广爱门曰咸熙门曰肃雍门曰和熙门犹干清宫之制而后有两井东西有两长街西长街则有兴善门丽京门长春门清华门宁福宫嘉福宫明徳宫永春宫永宁宫延禧宫延春宫宜春宫今为停丧之所凡妃嫔皇子皇女之丧皆于此停灵至发引时出东上南门东上北门北中门北安门诣西山等处坟园安厝东长街则有广顺门中和门景华门宣明门景明门洪庆门洪庆殿供畨佛之所也又有膳房又有门曰景和门又东则内承运库藏也自东上南门迤南街东曰永泰门门之内街北则重华宫之前门也其东有一小台台上有一小亭再东南则崇质殿俗云黑瓦殿是也【景泰间英庙自北狩回所居永泰门】再南街东则皇史宬珍藏太祖以来御笔实録要紧典籍石室金柜之书此其处也皇史宬每年六月初六日奏知晒晾司礼监第一员监官提督董其事而稽核之其看守则监工也左右小门曰■〈皇书〉【音龙】厯左门曰■〈皇书〉厯右门再东则追先阁钦天阁透珨碑在焉世庙御制钦天颂勒于碑碑石光润近似卧碑制也再南则御前作也
　　皇史宬之西过观心殿射箭处稍南有门曰苍龙其南则昭明门其西南则嘉乐馆也其北曰丹鳯门门列金狮二内有正殿曰龙徳左殿曰崇仁右殿曰广智正殿殿后则飞虹桥也桥以白石为之凿狮龙鼃鳖鱼虾海兽水波汹涌活跃如生云是三宝太监郑和自西域得之非中国石工所能制者桥之前右边一块缺损云是中国补造屡易屡泐亦古迹也桥之南北有坊二曰飞虹曰戴鳌姜立纲笔也桥之东西有天光云影二亭又北迭石为山山下有洞额曰秀岩以磴道分而上之其高高在上者干运殿也左右各耸一亭曰凌虚曰御风隔以山石藤萝花卉若墙壁焉又后为永明殿最后为圆殿引流水遶之曰环碧又再东曰玉芝宫后殿曰大徳殿外劵门曰宝庆门曰延祥门曰佳丽门其东墙外则观心殿也自皇史宬东南有门通河河上曰涌福阁旧名澄辉阁俗所谓骑马楼者是也迤东沿河转北则吕梁洪则东安桥再北有亭居桥上曰涵碧又北则回龙观在焉其殿曰崇徳者是也观多海棠每至春深盛开之时圣驾多临幸之是河也从北安门外文昌宫迤东步粮桥入经皮房内织染局巾帽局御马监之东东安门桥下至长安左门外迤东之玉河桥出焉北安门内街东曰安乐堂凡内官有疾者送此调理山陵及外厂九门官则不送也再南黄瓦西门之里则内官监也过北中门迤西则白石桥万洪殿等殿至大高玄殿则习学道经内官之所居也其北则里氷窨也北中门之南曰寿星殿左曰毓秀馆右曰育芳亭后曰万福阁其上曰臻福堂曰永禧阁其下曰聚仙室曰延宁阁曰北果园殿之西门内有树一株挂铁铸云板因年乆树长遂衔云版于树干之内止露十之三诚古迹也殿之东曰永寿殿曰观花殿植牡丹芍药甚多曰集芳亭曰会景亭曰翫春楼其下曰寿安室曰观徳殿亦射箭处也与御马监西门相对乃寿皇殿之东房万厯中年始开者殿之南则万岁山俗所谓煤山者此也【故老云土查堆筑而成崇祯己巳冬大京兆疏亦曾误指为有煤】山下树木葱郁鹤鹿成羣呦呦之鸣与在阴之和互相唤发可并闻于霄汉间上有一成磴道每重阳日圣驾在山顶升座可遥望靡涯矣山之门曰万岁门再南曰北上门左曰北上东门右曰北上西门西可望干明门东可望御马监也再南过北上门则紫禁城之玄武门也
　　此上西门之西大高玄殿也其前门曰始青道境左右有牌坊二座曰先天明镜曰太极仙林曰孔绥皇祚曰弘佑天民又有阁二座左曰炅【呼如阳】明阁右曰相【呼如阴】灵轩内曰福静门曰康生门曰高玄门曰苍精门曰黄华门殿之东北曰无上阁其下曰龙章鳯篆曰始阳斋曰象一宫所供象一帝君范金为之高尺许乃世庙玄修之御容也稍西曰石作曰圆明阁又西曰干明门门里迤南曰兵仗局曰西值房即尚衣监所属之袍房也曰旧监库属内官监地方曰尚膳外监曰甜食房曰西上北门其东向北者则西下马门矣
　　紫禁城护城河一带其东有内承运库衙门起至北花房止西岸有矮河墙罗列石作物料其东北角西北角有澡马河礓■〈石察〉石栅
