澎湖台湾纪略

　　杜臻述

　　澎湖有三十六岛，纵横三百余里（此据新图；实录云：二百余里）。闽海实录云：其地势在泉州料罗、漳州镇海之界，与旧屿、大担相对。自内地东渡，顺风一日二夜可至。闽书附晋江县，云地近琉球。昔人于此置兵防琉球，今则防倭。元岛夷志云：自泉州顺风二昼夜可至，有七澳。曹能始志胜云：自郡东台海门，舟行二日始至。郑若曾云：风顺尚有日半之程。指挥唐垣京澎湖要览云：系琉球山川，地界泉、漳、兴、福；其去内地也，埒于琉球。诸书所载不同。兹从闽志，附之晋江之后云。

　　前代事不可考。隋开皇中，遣虎贲郎将陈棱帅师至其地，虏男女数百人而还。唐施肩吾有澎湖诗云：腥臊海边多鬼市，岛夷居处无乡里；黑皮年少学采珠，手把生犀照咸水。盖亦尝有至焉者。明洪武五年，以其民叛服不常，大台兵，驱其大族徙置漳、泉间。今蚶江诸处遗民犹存。嘉靖之季，贼首曾一本、林凤据巢穴。万历元年，总兵俞大猷逐倭寇尝至其地。二十年，朝鲜告倭且入犯，议者谓不宜坐弃澎湖；于是设把总一员，以十六舟戍之，隶于南路参将及泉南游击，令与浯、铜二寨时相策应。其汛地北起北山，南尽八斗澳（地图作八罩）。北山龙门港、丁字门、西屿头曰最冲，娘宫前莳上澳曰次冲。春汛以清明前十日为期，驻三月而返；冬汛以霜降前十日为期，驻两月而返。后以孤悬海表，议罢。启、祯以后，鲸鲵荐食，益不可问矣。

　　其气候常温，多风少雨，涉海者重絮而行，既至则易暑服。其土燥刚，不宜木与禾黍，惟多杂茅。泉味涩卤。其民多泉人侨处，苫茅以居，朴野多寿。土布为衣，耕、渔、畜、牧以自给。煮海为盐，酿秫为酒，掇蠃蛤、鱼鰕以佐食，爇牛粪以爨，燃鱼骨为油。尤多畜羊。羊特肥大孳生山谷间，千万为群；其主各刻角以志，或烙毛剺耳，夜亦不收，卒不相乱也。黠者或行贾于外，致饶裕。其争讼，择年长者为主，有事则就决之；间有不能平，则诣晋江县陈白以听断焉。

　　其物产，谷则胡麻、菉豆，药则天门冬、蒺藜、山茨菇，虫兽则山猪、蜈蚣、花蛇及鼠。山猪类常猪而色赤，性矫健、善跳跃，捕之不易；有毒，食之辄发疮廱。花蛇、蜈蚣，弥山皆是。花蛇大者长丈余，小亦不下五、六尺；昼日伏藏地下，夜乃潜至几案间。揣之不啮，即偶见啮，亦不至大创也。海石间台乌鱼（一名鲻鱼），本生江南，每春日辄来就石廉打子；盖鱼子须跳掷乃台也。又产白芥菜，高五、六尺。其子中入药绝佳。茉莉千瓣，香尤酷烈。又有藤蔓，可为燂洗兵船之用。

　　其民以地近琉球，夜不举火；言夷船望见烟起，必来抄掠。居中大澳曰娘娘宫，可泊南北兵船五、六十艘，游总处之。其余诸岛，重迭相包，无大高山，望之不过如覆釜。水曰澎湖沟，分流东西。泉、漳人行贾吕宋，必经其间。自此以东为顺流，其返也过澎湖而西，复为顺流。自娘娘宫而西至西屿头，曰左哨；稍南至莳上澳，曰右哨。二哨去中营，各水程四十里；二哨自相去，水程三十里。西屿头有山稍高，度可十六丈，可泊兵船四十余。其险阨，曰内堑、外堑。稍北十里曰■〈虫咠〉仔澳，可泊南北风船十余，各有渔寮、水源（新图又有大池角、小池角、吼硫■〈石昷〉、员门司、磺礁、外篙澳）。稍北二十里曰丁字门。其澳迫窄，仅可容四、五舟。有水源，无鱼寮，为寇夷樵汲所至（新图有大吼门、小吼门）。过此十里为西北大洋。稍东二十里曰镇海港、员背屿，各有水源、渔寮（新图有赤■〈土敢〉、万丈潭、土地公〔屿〕、金屿、姑婆屿、铁砧屿）。但水浅礁多，舟不易往，第于北山筑墩探瞭而已。西屿头内十里曰果叶澳，可泊十余舟，有水源，无渔寮。又自西屿头分道至通梁仔、瓦硐港、大礵，各水程二十里；中墩三十里（新图果叶甚远、中墩甚近，小有不同。中墩一称上门）。各有渔寮、水源，亦以水浅礁多，止可远瞭。凡此，皆左哨汛游所及也。莳上澳可泊北风船四、五十（新图其旁有岑圭屿、狗沙屿、毒屿、铁线尾）。西行水程五里，有小屿曰大城，与小城相对，可泊南风船十余。自莳上东北行水程二十里曰文龙港（即龙门港之误，新图作良文港），可泊北风船十余，有水源、渔寮（新图其旁凌波屿、芝眉屿、椗齿礁），为寇夷樵汲必至之地，哨逻至此而回。稍南五里曰林投仔（新图其旁有锁管港、猪母落水），有水源，冬哨始有渔寮，春夏无之也。由此东北水程十里曰圭壁港。又北五里为青螺仔头（新图其旁有洪林罩、蚱脚屿、鼎湾仔、沙港、雁睛屿、蟳广〔澳〕），各有水源、渔寮；但礁多水浅，常于太武山远瞭及之，不能时往。自莳上而西五里曰枫柜仔山（新图其旁有大帽山沉礁、四角屿、龟龙屿、番墓下），有水源、渔寮。北行水程二十里曰安山仔，可泊南风船二十余。又北十里曰东港尾，有水源、无渔寮，可泊南北风船二十余（新图自此复返祖妈宫）。左哨汛游所及，略尽于此。西屿头之极西又有吉贝屿，孤隔多礁（新图其旁有布袋澳、沙员、目屿沉礁、北硗，其东有屈瓜、饭篮，盖自沙澳北境尽处也），与南面之八斗屿隔一潮水，虽有渔寮、水源，均非哨瞭所及也。新图与旧异。娘娘宫作祖妈宫，大城、太武山与之连壤，不云小屿也。自祖妈宫稍西，尽西埯仔尾，有新城，盖寇所筑以屯兵者（其旁有后屈潭、水烧平、牛心礁、西卫），而八斗之旁澳屿甚多（在北者曰鸳鸯窟、捕渔坑、水埯、虎井、桶盘屿、三将军石，在西者曰大花宅、小花宅、花屿沉礁、草屿沉礁、大猫屿、小猫屿，在东者曰挽毛潭、纲埯，在南者曰头巾屿、铁砧礁），稍远至西南有湖内祖（杭铁成质潭），东北有尾糍田（将军柱后代仔），极南有东屿平、西屿平、钟屿等，更南有东吉屿、西吉屿、斧头争等，皆明世所不及知也。

　　台湾旧名东番，不知所自始。其人聚处无君长，不通中国，亦未尝属于外番。泉、漳市舶私与往来，不闻官司也。万历三十年为倭所据，浯屿营将沈有容台舟师击之。有连江人陈第者以蓟镇游击家居；与偕往，共破倭。泊舟大员，其酋长大弥勒等持鹿酒以献。因备询其土俗及山海形势，述之成篇。第倜傥能文，起家诸生，后乃去而习武，为总兵俞大猷、尚书谭纶所赏识，遂以武秩进。着有意言、谬言、毛诗古音考、伏羲光天图缵，太史焦竑亟称之。自有第记，而后厘考属国者，始知有所谓东番云。其略曰：东番在澎湖外洋中，自烈屿航海一昼夜至澎湖，又一昼夜而至加老湾。其起地魍港、加老湾，历大员、尧港、打狗屿、小淡水，又有双溪口、加哩林、沙巴里、大帮坑，皆其居也。断续凡千余里，种类甚蕃，别为社。社或千人，或五、六百，推子女多者为社长，听其约束。性好勇喜斗，暇辄习走，昼夜不止。足躢皮厚至数分，履棘刺如平地，捷及奔马，度终日之力可踔数百里。或两社有隙而相攻，约日合战，战已往来如初，不复相仇。每杀人，斩其首，剔肉存骨，悬之门。门悬髑髅多者，称壮士。地暖，冬夏裸体，不知衣冠，自谓简便。妇女结草裙，略蔽下体而已。无揖让、拜跪礼，无历日、文字。视月圆为一月，计月十圆为一年。久则忘之，故率不纪岁；少壮老耄，问之不知也。交易，结绳以识。无水田，治畬种禾，山花开则耕。禾熟，拔其穗。榖粒视内地稍长，特甘香。采苦草，杂米酿之为酒，间有佳者。豪饮至一斗。燕会则置大罍团坐，各酌以竹筒，不设肴核；乐则起而跳舞，口亦呜呜若歌曲。男子剪发，留数寸披垂；女子则否。男子穿耳，女子年十五、六断去唇两傍二齿以为饰。地多竹，大数握，长十余丈。

　　伐竹构屋，蒙以茅，广长数丈。每族共屋；一区稍大曰公廨，少壮未娶者曹居之。议事必于公廨，取召集之易也。将娶，视女子可室者，遣人遗以玛瑙珠一双；女子不受则已，受则夜造其家，鼓口琴挑之。口琴者，制薄铁如搔头，尾有两岐；啮而嘘之，铮铮有声。女闻，纳宿；未明径去，不见女父母。自是，每夜必来以为常。迨产子女，妇始往婿家迎婿，如亲迎；婿始见父母，因家焉。养女父母终身，其本父母不得子也。故生女喜倍男，谓其可以继嗣云。妻死更娶；夫丧号为「鬼残」，终莫之醮。人死，击鼓环哭，置尸于地，四面炽火煏而干之，不棺；屋坏，乃立而埋之，不封、不祭。将耕，戒勿言，亦不复相仇杀。道路以目，田畔相遇，少者背立；长者径行，无所问。华人或故侮之，不之较。苟违戒，则岁辍不登，故守之惟谨。女子健作；女常劳，男常逸。有盗取人物者，严剔之，尸于市；故夜户不闭，禾积场无敢窃者。有床，无几案，席地坐。谷有大小豆、胡麻、薏苡。蔬有葱、姜、番薯、蹲鸱。果有椰、蔗、毛柿、佛手柑。畜有猫、狗、豕、鸡。兽有虎、豹、熊、鹿。鸟有鸠、雉、鸦、雀。山尤多鹿。人善用镖；镖长五尺，镞甚铦，虎鹿遇之辄毙。其捕鹿，尝以冬；伺其群台，乃集众逐而围之。掩群尽取，积如邱陵。先屠取其皮角，次腊其肉，次腊其舌与肾、与筋别藏之，盛以籍而鬻之华人。又能榨蔗为糖。二者，其本业也。得鹿子，必扰驯之。刺鹿肠，台新饲草未化者，必竞食，名百草膏。俗食豕；不食鸡、雉，惟拔取其尾以饰旗。见华人烹噉鸡、雉，辄呕。虽居岛中，不能操舟，畏见海；但捕鱼于溪涧。故老死不与他夷往来。永乐初，郑和航海谕诸夷，东番独远窜，不听约。于是，家遗以一铜铃，使悬于颈，盖狗畜之也；至今传以为宝。始皆聚居滨海。嘉靖末，遭倭寇焚掠，乃避居山。倭善鸟铳，镖不能御也。自通于中国，独见其居山耳。漳、泉之民至者既众，充龙、烈屿诸澳往往能译其语；尝以玛瑙、瓷器、布、盐、铜簪环之属，易其鹿脯筋角。间遗以故衣，辄喜；见华人则衣之，以相晋接，退则袭而藏之。得布亦藏之。盖其人习裸，不耐冠履束缚也。性颇醇朴，自通中国，始有嗜好；奸人又绐以滥恶之物，彼亦渐悟，知为巧伪矣。第之言如此。
