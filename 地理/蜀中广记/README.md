  《四库全书 蜀中广记》一百八卷　（明）曹学佺 撰
　　钦定四库全书

　　史部十一

　　蜀中广记

　　地理类八

　　杂记之属

　　提要

　　臣等谨按蜀中广记一百八卷明曺学佺撰学佺有易经通论已著录学佺尝官四川右叅政迁按察使是书盖成于其时目凡十二曰名胜曰边防曰通释曰人物曰方物曰仙曰释曰游宦曰风俗曰著作曰诗话曰画苑搜采宏富颇不愧广记之名其中如叙州府之髙州明史地理志云洪武五年由州改县正徳十三年复为州珙及筠连三县隶焉此书仍称髙州为县二县亦不为之属又成都府之资阳县明史地理志属简州此书不系简州而别于仁寿井研二县后皆未免编次偶疎王士祯古夫于亭杂录曰丹铅录载东坡赠青神杨栋词云允文事业従容了要岷峨人物后先相照见说君王曽有问似此人才多少而引小说髙宗问马骐蜀中人才如允文者有几云云按允文采石之功在南渡后东坡之殁久矣安得先有此词而曺能始蜀中十志亦载之畧无驳正又曰蜀中十志以物类相感志十八卷为东坡撰谬甚则舛讹抵牾亦时时间出盖援据既博则精粗毕括同异兼陈亦事势之所必至要之不害其大体谈蜀中掌故者终以全蜀艺文志及是书为取材之渊薮也

　　蜀中广记卷一

　　○名胜记第一

　　川西道

　　成都府一【成都华阳二县附郭】

　　华阳国志蜀之为国肇于人皇与巴同囿至黄帝为其子昌意娶蜀山氏之女生子髙阳是为帝喾封其支庶于蜀世为侯伯歴夏商周武王伐纣蜀人与焉尚书牧誓庸蜀羌髳是也有周之世限以秦巴虽奉王职不得与春秋盟会君长莫同书轨周失纲纪蜀先称王有蜀侯蚕丛其目纵始称王次王曰柏灌次王曰鱼鳬又次后有王曰杜宇始称帝号曰望帝更名蒲卑治郫邑或治瞿上后传位于其相开明开明位号曰丛帝丛帝生卢帝卢帝生保子帝九世始传开明尚焉开明尚自梦郭移乃徙治成都扬雄蜀本纪云蜀王据有巴蜀之地本治广都樊乡徙居成都者也巴与蜀雠求救于秦秦惠王二十七年遣张仪与司马错等灭蜀遂置蜀郡仪筑成都以象咸阳修整里阓市张列肆规制广备故左思蜀都赋云金城石郭兼匝中区既丽且崇实号成都矣其地东接于巴南接于越北接于秦西奄峨嶓地称天府原曰华阳汉武帝元朔二年以新启犍为牂柯越嶲三郡疆壤益广始称益州初治广汉之雒县后乃徙此李固与弟圄书曰固今年五十七鬓髪已白所谓容身而游满腹而去周观天下独未见益州耳昔严夫子常言经有五渉其四州有九游其八欲类此子矣王莽簒盗改天下郡守为率正又改蜀郡为导江迁公孙述为导江率正治临卭后据成都称帝具置百官造十层赤楼东观汉记曰公孙述造十层赤楼也于是改益州为司隶蜀郡为成都尹矣建武光复二百余载迄于孝灵王室多故刘焉求益州牧以避世难始治绵竹后徙成都先主代刘璋而有之即位于武担之南典畧曰玄德于是起馆舍筑亭障从成都至白水关四百余区也后主建兴二年以丞相亮开府领益州牧亮卒拜尚书令蒋琬为大将军领益州刺史蜀亡晋太康初乃属王国号曰成都内史永嘉之难没于夷獠成都记云府治北三里许有李特所筑赤涂城十六国春秋曰李雄定成都以严柽为蜀郡太守及即位迁柽益州刺史是矣桓温西伐李势出降刘义庆世说云宣武平蜀集叅僚置酒于李势殿巴蜀缙绅莫不来萃桓既素有雄情爽气加尔日音调英发叙古今成败由人存亡系才其状磊落一坐叹赏焉永和之际周抚为蜀郡太守四十余年境土粗安王逸少与周益州书云省足下别疏具彼土山川诸奇扬雄蜀都左太冲三都殊为不备悉彼故为多奇益令其游目意足也可得果当吿卿求迎少人足耳至时示意迟此期真以日为岁但言此心已驰于彼矣又云吾有七儿惟一小者未婚过此便得至彼得果此縁一段奇事也梁武陵王始封入蜀元帝与之书云蜀中斗絶易动难安弟可镇之吾自当灭贼又别纸云地拟孙刘各安境界情深鲁卫书信常通良所托重也萧纪反戈东下以地降于西魏再折入周至隋开皇始定之封其子秀作镇于兹营侈宫室成都记曰隋蜀王秀尝造一殿飞鸟不止其上此其凡耳后竟被征唐初开国太宗以秦王建益州台遥领之而已贞观十年分成都县之东偏置蜀县在郭下明皇西幸驻跸成都始改蜀县为华阳而成都华阳二县从此分矣
　　按今成华二县与蜀邸护卫尺籍绣错不能细为分析大较西门者属成都东门者属华阳而南北门则成都有三分之二华阳居其一焉予观扬子云答刘子骏书有少时所作成都城四隅铭则今之东西南北四门颇为近古西门者古之宣明门也南门者古之江桥门也东门者古阳城门北门者古咸阳门也李膺益州记云最东曰阳城门次西曰宣明门蜀时张仪楼即宣明门楼也重阁复道跨阳城门故太冲赋云结阳城之延阁飞观榭乎云中矣江桥者大江水所经也稍西为市桥曰市桥门以汉旧州市在此桥南桥下即石犀所濳渊亦曰石牛门故曰少城南面有三门焉咸阳门者秦惠王二十五年蜀侯恽赐死封其子绾为蜀侯初蜀人塟恽郭外绾闻恽无罪乃使使迎丧入塟之郭内丧车至城北门忽陷入地中蜀人因名北门曰咸阳门矣
　　成都县本治赤里街秦守张若徙入少城内少城者子城也惟西南北三壁东即大城之西墉故蜀都赋云亚以少城接于其西矣周地图记云初张仪筑城城屡坏不能立忽有大龟出于江周行旋走巫言依龟行处筑之城乃得立所掘处成大池龟乃伏于中龟每出则州境有贼刺史或病即此城也汉元鼎二年大城立九门少城立九门故有十八郭之称而郡县多城观矣
　　太城者今南门城也李石诗序云张仪司马错所筑太城自秦惠公乙巳岁至绍兴壬午一千四百七十八年虽颓圯所存如断壁峭立亦学舍一奇观云学舍在南则太城可知矣
　　少城者西南之间今之锦江楼也古今集记锦楼在龟城上前临大江下瞰井邑西眺雪岭东望长松白敏中尝赋诗于其上旧记谓路岩所建非也岩在敏中之后矣范成大诗注云少城张仪所筑子城也土甚坚横木皆朽有穿眼土相着不解散予按王右军法帖徃在都见诸葛颙曾见问蜀中事云成都城门屋楼观皆是秦时司马错所修令人远想慨然具示为广异闻
　　华阳国志张仪张若城成都周回十二里高七丈造作下仓上皆有屋而置楼观射圃又营广府舍置盐铁市官并长丞皆与咸阳同制更于夷里桥南立锦官锦官者犹合浦之有珠官也西又有车官城其城东西南北皆有军营垒城焉按此皆从西南隅论耳蜀汉昭烈即位于武担之南当在城之正中今蜀藩是也隋之摩诃池伪蜀之宣华苑皆在其内昔诸葛武侯以丞相开府领益州牧故晋益州刺史治太城其制因之亦唐节度使宋制置使所称建牙交戟地矣蜀太守治少城故昔之郡治因焉西门直街鼓楼即宋铜壶阁也陆游记云阁南直西川门西北距府五十步乃蒋堂知益州伐江渎庙材所创吴栻范成大相继而侈张之为成都巨观书于史又吴师孟修西楼记云西楼直府寝之北毎春月花时大帅置酒高会叅佐于其下五日复纵民游观宴嬉西园以为岁事然止得造其庑序而已合此二说则府治为西门无疑相沿蜀郡太守之故址也今乃益逊而北盖不敢与藩邸抗于是斥北城之地以居故武担山昔在城外而今在城内陆务观有游武担村落诗可证矣转运使今之藩司也钤辖使今之阃司也运司昔在城东有玉溪堂即伪蜀权臣故宅今藩司在城北阃司相沿如故若提刑漕使即今之守巡使者散在诸路是其职居位置皆有不同如此
　　南门之胜如石室石犀严真观江渎池七星桥昭烈武侯祠其最著者
　　华阳国志云始文翁立文学精舍讲堂作石室一作玉室在城南永初后堂遇火太守陈留高眹更修立又增造二石室李知几云西为文翁稍南为高眹比文翁石室差大皆有石像矣乐史寰宇记云文翁学堂一名周公礼殿方舆胜览云前代以周公为先圣至唐贞观始从房玄龄等议更祀孔子开元册谥文宣王位南向自是号文宣王殿矣而周公之祀遂废然旧像犹存帅袁说友乃为殿祀之按上南志以武侯征西南夷而梦见周公故勅所在祀周公也任豫益州记云礼殿其栾栌节制犹古建南基高六尺厦屋三间通皆图画古人之像及礼器瑞物堂西有二石室系年録云绍兴六年成都教授范仲殳奏疏言本府学殿建于东汉之初平汉人以大隶记其修筑岁月刻于东楹至今九百四十三年盖天下栋宇之古无过于此者按欧阳修集古録以周公礼殿记蔡邕所书也范镇云其柱锺会隶书刻其上李知几十咏诗说会与邓艾同入蜀在咸熙元年甲申距汉献兴平元年甲戌凡七十一年矣会盖追丈翁高君之美而书也赵明诚金石録跋云右汉周公礼殿记者今成都府学有汉时所建旧屋柱皆正方上狭下阔此记刻在柱上灵帝初平五年立距今盖千年矣而字画完好可读当时石刻在者徃徃磨灭此记托于庑楹乃与金石争寿亦异矣记有云甲午年故府梓潼文君增造吏舍二百余间按华阳国志有文参字子奇梓潼人平帝用为益州太守不从王莽公孙述光武嘉之疑此记所载即其人也盖光武建武十年岁次甲午云
　　唐卢照邻文翁讲堂诗锦里淹中馆岷山稷下亭空梁无燕雀古壁有丹青槐落犹疑市苔深不辨铭良哉二千石江汉表遗灵岑参诗文公不可见空使蜀人传讲席何时散高台岂复全丰碑文字灭冥漠不知年裴铏诗文翁石室有仪刑庠序千秋播德馨古栢尚留今日翠高岷犹蔼旧时青人心未肯抛膻蚁弟子依前学聚萤更叹沲江无限水争流祗愿到沧溟宋李石诗来为人所爱去为人所思君看文与高慈惠蜀之师至今室中像凛凛建立时知非伯有石定是桐乡祠蜀人爱二公远与千载期宋京蜀事补亡诗君不见西汉文翁为蜀守蜀学不居齐鲁后诸生竞欲保翁名石室镌磨贵难朽东汉高公又几时为作石室还如兹至今二室坚且乆文公高公名不衰喻汝砺谒文公堂诗緑荻负幽隐髙槐泛轻凉各夸一时好披风互低昂讵知五月中微阴忽催黄我行鲁侯宫独谒文翁堂若人骨已朽道在斯不亡遂令蜀文章照耀日月傍世事俱腐臭斯文真久长靡颓能几时萧条翳墟荒富贵岂不好千载同一伤三叹为何人抚已涕自滂
