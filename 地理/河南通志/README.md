　　一旧志封建列古先与春秋战国二条混杂不明以邶鄘郐蒋与夏殷诸侯并列时代不同春秋时河南有四十国未闻战国时有也东虢桧即郑地胙之后为南燕国温即为苏而并载一时殊失考核且春秋时之魏国在今山西平陆县梁今陜西郃阳县郜今山东城武县芮今山西芮城县均非河南地杜注及地里志歴歴可考而旧志列梁魏于封建列郜芮于图表因讹袭舛莫此为甚鄾鄀与唐具载左氏俱在南阳邓州境戴即甾今考城县而旧志无之今俱逐一订定列古先封国凡六春秋以前国凡九春秋时姓氏爵秩可考者凡三十三战国韩赵魏楚合春秋时宋卫凡六时代既分其更递迭居自不至前后两载详辨疑中兹不复赘
