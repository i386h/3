　　一舆图先省总次各郡暨海防庾岭地冲又次之虎门澳门险设又次之罗浮名胜又次之
　　一粤东沿革轇轕特甚匪寻源以竟委将治丝而愈棼今稽各史及舆地志稍引厥绪冀免晦杂

　　【臣】等谨案广东通志六十四巻
　　国朝总督广东兵部右侍郎兼都察院右副都御史郝玉麟等监修岭南为炎海奥区汉魏以还舆图可考然如南方草木状但志物宜岭表録异仅征杂事而山川阸塞或未之详明代有戴璟郭棐谢肇淛张云翼诸家之书大辂椎轮又不过麤具崖畧
　　国朝康熙二十二年始辑有通志视旧本渐具条理此为雍正七年玉麟等承
　　命所辑采掇补苴较为赅备开局于雍正八年六月竣事于九年五月告成视他省为独先故中间或沿袭旧文失之冗蔓或体例不一彼此抵牾皆未能悉加订正然全书三十五门内新増者四葺旧者三十有一大都首尾详明可资检阅至外畨一门为他志所罕见然粤中市舶骈集莫不瞻星戴斗防极朝宗裒而録之足见
　　圣朝声教之逺亦通典述边防而兼及海外诸国
　　之例也乾隆四十六年九月恭校上

　　御制人臣儆心录序【顺治十二年】
　　朕惟人臣立身制行本诸一心心正则为忠为直众美集焉不正则为奸为慝羣恶归焉是故心者万事之本善恶之所由出也顾事有殊涂心惟一致一于国则忘其家一于君则忘其身如此者不特名显身荣邦家亦允頼之矣若夫奸邪之流树党营私怙权乱政卒至身名俱丧为国厉阶盖縁居恒无正心之功一当势利遂昬迷瞀乱狂肆骄矜上昧王章下乖臣谊或作威而聨羽翼或比匪而效奔趋如谭泰石汉以累世旧臣久叨恩遇不思图报逞臆横行跋扈自恣目无纲纪陈名夏则一介竖儒骤蒙显拔倚任深重赐赉优隆而乃背徳植交蔑法罔上此皆自作罪孽以致陨厥身家朕歴稽徃古宵人误国代代有之观之近事复炯鉴昭然足为永戒恐后之为臣者或仍蹈覆辙负主恩而渝素志至于身罹刑宪悔悼无由故推原情状而论列之録成一编以为人臣儆心之训云
