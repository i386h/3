　　福建通志序
　　国有史太史掌之邦有志职方掌之均以纪事而相为表里孔子因鲁史作春秋编年其创也司马迁变编年而为纪传要皆成于一人之手史成不复更修他如前后汉南北宋三国五代若班范李陈欧阳令狐诸人皆专门名家卓卓自号史才即隋书长孙顔孔皆与唐书刘韦欧宋同成宋金辽史则阿鲁图脱脱互叅元史则宋濂王袆协定亦共成一家言虽纯驳不同详畧各异惟有辨其得失议其优劣从无起而更张之者故曰信史也若夫志也者记也择焉必精语焉必详非如史家纲举目张大书特书即可立言不朽其叅互考订逺引旁稽又非恃一手一足之力土地有扩辟疆域有改赋税有増减人物乗时蔚起风俗渐次转移不数十年旧志不全不备重修乌可缓哉我
　　世宗宪皇帝当久道化成之日享一统车书之盛特命纂修通志各省増辑进
　　呈典甚钜也吾闽地属扬州分来越国自无诸启疆而后命官易路歴有歳时建元移徙以还流离播迁蹙靡有定昔之七闽乆増为八台湾设而为九闽福宁陞而爲十闽其中县复分县县复改州大非昔日规模颂声教者曰东渐于海远至海止矣而闽海之外台领四县延袤数千里尽入版图则幅员之濶至我
　　朝为最大疆宇之广至吾闽为独远人将于此观极盛
　　焉独是我
　　朝重熙累洽俗美化行我
　　皇上龙飞首出万国咸宁
　　圣谟之赫濯
　　徳教之覃敷堪垂于万斯年者莫可纪极防阐幽亷
　　顽立懦
　　褒嘉之令何嵗无之邹鲁虽小考亭之流风犹在也多士挺生指不胜屈且雨若而百谷蕃大海晏而鲸蜺静耕田凿井
　　帝力俱忘航海梯山远夷攸服猗欤休哉倘搜罗不富珥笔未工奕世而下未见今日太平熙皥之象一道同风之治不曰泱泱表海一大都会也纂修者实任其咎【臣】焯来抚是邦
　　诏下已七年前人业未竣督诸绅士总裁分校各执其事悉从旧制附入新増于乾隆首歳剞劂功成披图按籍岭以东浙右山峦挺秀海以南粤中波浪沧溟郡邑碁布星罗沃野锦纷绣错佳山胜水日作卧逰往哲名贤恍如晤对即披缁化羽凿齿文身者不啻环在吾前愧政不敏不能扬
　　天子休命承宣
　　徳意行庆兆民聊存此志以备輶轩采风垂诸史册既
　　拜手飏言恭进以
　　闻爰弁数言于首是为序
　　乾隆二年歳次丁巳孟夏上浣福建巡抚兼都察院右副都御史【臣】卢焯撰














　　福建通志序
　　考之周礼惟王辨方正位体国经野以为民极于是乎太宰以八则治都鄙以九职任万民又大司徒以天下土地之图周知九州之地域广轮之数以土会之法辨五土之物生以土宜之法辨十有二土之名物故古太史有輶轩之采备陈十五国之风献诸天子俾周知列国之情形以为出治宁人之本遐哉尚矣夫于列国之间山川风物各有其宜制度更张不一其局至于户口因时而増减人文随代而盛衰莫不具有叅繁酌简之宜以垂为因革损益之制既职要而职详亦无偏而无党典诚重且钜也当
　　世宗宪皇帝临御之六年
　　特命各省纂修省志诚以
　　国家承平日乆化普徳隆休飬生息涵煦九十载之深际其盛者河海呈其清晏云物其光华芝草醴泉嘉禾穂麦靡不献瑞九州之内环海之外罔有未垦之土田无不贡赋之方域其间疆宇之开拓氓庶之殷繁山陬海澨不见萑苻异域遐方咸知礼让试览舆图而广狭不侔爰稽赋役而繁简不等观乎风化日美于前睠彼人文斯称极盛从前旧志利用増修兹其所以上烦
　　睿算欤时【臣】玉麟方督百粤凛遵
　　圣训编辑逾年刋刻呈
　　进十年夏复奉
　　命制九闽省志纂修尚未告竣兹以増设一府二州邑新分重需改建因之更歴歳时始成卷页钦惟我
　　皇上聪明天亶备神圣之全建中和之极百度惟贞一
　　本于继述之善兹编甫就恭献
　　枫宸【臣】方惴惴惟弗克襄事是惧伏思闽地虽僻处海隅素有邹鲁之旧风土人物实为东南奥区其炳炳熊熊垂诸旧志者姑不具论即如绘图之内旧志仅载郡邑城池今则山川名胜以及沿海岛澚无不歴歴摹列了然可观且首揭
　　三朝之诏谕以重
　　典谟厘订礼乐之大章以昭明备他如分析建置博纪风物普济育婴书院社学凡关典要罔不増修续増一十四卷洵足以鸣
　　国家声教四敷车书一统之盛猗欤休哉后之读是编者揽名区与胜迹益见山辉川媚皆
　　圣泽之旁敷披岛海之全图益见浪息波恬皆神威之远播况开卷即凛
　　王言有典有则礼乐归于和叙可法可地辟民聚俗美风淳虽唐虞三代之隆规不是过也是知同此舆图而焕然一新同此赋役而厘然足式同此风化同此人文而仁让蔚为名邦英声蜚于上国行将备史官之采而集成太平一统之书则斯志也虽介在方隅而上下千百年体国经野之模或者不
　　失周礼之遗意乎惟愿守斯土者小亷大法目击心存务思为乆安长治之本用佐我
　　皇上万年有道之长则拜手飏言者亦得借是以不朽斯诚【臣】玉麟之厚幸也夫谨濡笔而叙其略于简端
　　闽浙总督衔専管福建等处地方军务兼理粮饷兵部尚书兼都察院右副都御史世袭骑都尉加七级纪録四十二次功加一等【臣】郝玉麟谨序

　　福建通志凡例
　　一列国之有志乘实与史书相为表里山川风物贵在精详制度文章灿然明备近奉
　　诏防纂修典綦重也闽海天末动烦
　　宸算
　　丝纶逺被山海腾欢兹集首列
　　诏谕弁冕羣编
　　圣谟洋洋垂为世则
　　一九州应象星纪攸分闽为牛女之墟占候准之谨胪前史证以诸家辨方正位而体国经野之事可举矣
　　一国有与立建置所必详也旧志仅载名号而分析沿革未清今仿史记年表例厘为三格以朝代总部郡县缀列其下开卷按表触目了然
　　一闽地负山慿谿襟带江海议守之国也详订山川安流奠峙次则疆域井井而以形胜附焉
　　一因山为城濬川为池王公设险大易之训也闽省城池増髙浚深详厥规制金城岌岌天堑汤汤美哉国乎正封洫治桥梁王政所汲汲也连类而并书之
　　一化行俗美土物心臧此风物所由志也旧志风俗仅纪岁时贻讥弇陋兹集博援往籍益以前闻美刺昭然言非臆造物产则本莆田黄仲昭本纎悉増载谨依
　　大清一统志例次于山川城池之后
　　一任土作贡田赋以兴度地居民户役斯起闽赋一准本朝赋役全书新编续垦俱准则例撮其会萃厘如指掌盐课关税以相从百年以来生齿殷繁户口寖盛
　　盛世滋生永无加赋闽邦富庶于今为极至于普济堂育婴堂义塜之设养生送死蕃息生民所有事也附载于后以志
　　皇仁
　　一王者治定制礼功成作乐而典礼昭焉旧志未纪礼乐非制也兹考
　　大清会典厘为大典一帙乐章亦依　部颁新本礼明乐备天地为昭矣
　　一国之大事在祀与戎闽多淫祀兹集根椐正史羣神羣祀爼豆靡忒民听不惑美报昭矣至以郡县驻防营弁满汉钤辖旗丁详审増编分戎籍靡惟借以干城实可联为臂指
　　一厯代建侯树屏闽多封土间有乘时剏国息兵宁人者亦所必録考已事而镜兴衰备一方之典故焉可矣
　　本朝定鼎以还爵班五等者例入
　　一化民成俗端必由学
　　国朝敬教劝学泮林芹藻振古文明迩年
