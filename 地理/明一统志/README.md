　　钦定四库全书　　　　史部十一
　　明一统志　　　　　　地理类二【总志之属】提要
　　【臣】等谨案明一统志九十卷明吏部尚书兼翰林院学士李贤等奉勅撰案沈文圣君初政记称太祖洪武三年命儒臣魏俊等六人编类天下郡县地理形势为大明志今其书不传后成祖采天下图经图经命儒臣纂辑为一书亦未及成而中辍至英宗复辟后乃命贤等重编天顺五年四月书成奏进赐名大明一统志御制序文冠其首锓板颁行考舆记诸书出自官撰者自唐元和郡县志宋元丰九域志外惟元岳璘等所修大一统志最称繁博国史经籍志载其目共为一千卷今已散佚无传虽永乐大典各韵中颇见其文而割裂丛碎又多漏脱不能复排比成帙惟浙江汪氏所献书尚存存原刋本二卷颇可以考见其体制知明代修是书时其义例一仍元志之旧故书名亦沿用之其时纂修诸臣既不出一手舛错抵牾疎谬尤甚如以唐临泃为汉县辽无章宗而以为陵在三河金宣宗大梁而以为陵在房山以汉济王王兴居为东汉名宦以箕子所封之朝鲜为在永平境内俱乖互不合极为顾炎武日知录所讥至所摘王安石处州学记地最旷大山长谷荒之语则并句读而不通矣此本内多及嘉靖隆庆时所建置葢后人已有所续入亦不尽出天顺之旧我

　　天顺御制明一统志序
　　朕惟我太祖髙皇帝受天明命混一天下薄海内外悉入版图葢自唐虞三代下及汉唐以来一统之盛蔑以加矣顾惟覆载之内古今已然之迹精粗巨细皆所当知虽歴代地志具存可考然其间简或脱畧详或冗复甚至得此失彼舛讹殽杂往往不能无遗憾也肆我太宗文皇帝慨然有志于是遂遣使徧采天下郡邑图籍特命儒臣大加修纂必欲成书贻谋子孙以嘉惠天下后世惜乎书未就绪而龙驭上宾朕念祖宗之志有未成者谨当继述乃命文学之臣重加编辑俾繁简适宜去取惟当务臻精要用底全书庶可继成文祖之志用昭我朝一统之盛而泛求约取参极羣书三阅寒暑乃克成编名曰天下一统志着其实也朕于万几之暇试览阅之则海宇之广古今之迹了然尽在胸中矣既藏之秘府复命工锓梓以传呜呼是书之传也不独使我子孙世世相承者知祖宗开创之功广大如是思所以保守之惟谨而凡天下之士亦因得以考求古今故实增其闻见广其知识有所感发兴起出为世用以辅成雍熙泰和之治相与维持我国家一统之盛于无穷虽与天地同其久长可也于是乎序天顺五年五月十六日











　　进明一统志表
　　资政大夫吏部尚书兼翰林院学士臣李贤中宪大夫太常寺少卿兼翰林院学士臣彭时翰林院学士奉政大夫臣吕原伏防皇帝陛下命臣等充总裁官修天下一统志今编辑已成凡九十卷谨用缮冩装潢进呈臣贤等诚惶诚恐稽首顿首上言伏以日月照临诞启圣朝之运乾坤覆载咸归一统之尊民物阜康人文宣朗匪表章于今日曷垂示于将来粤昔虞代幅员备存禹贡周家疆域悉着职方惟敷土经野之详故享国歴年之久降自炎汉下逮有元志地理者数百家纪方舆者累千卷若非泛而弗切则或畧而未明于是厪圣祖之至情念舆图之重事爰命儒以纂集冀成书以铺张博采虽周全功未究时如有待事岂偶然恭惟皇帝陛下刚健钦文聪明英武重华协帝已臻盛治于前申命自天复纉洪图于后志兼隆于千古勤独断于万几凡圣贤经世之图咸留睿览于祖宗待成之志尤轸渊衷肆宵旰之靡遑益继述之是务顾惟重熙累洽千载一时自天命之有归祚皇明之抚运徳与地而兼广化与天而同流尊亲无间于华夷功烈独超乎先后必有盛制以阐丕休特简文学之臣付以纂修之任而皇猷髙逺圣学渊防涣发纶音豫为指授于凡经书所载咸在网罗至若子史所传举当收采宜加参考以正信疑是非必有据依以立义类凡例表京师为四方之极列方岳为诸郡之纲疆域必繋于九州分野悉稽乎列宿形胜风俗録奚逺迩之分物产山川载靡小大之间藩封着维城之固公署严御侮之威书学校书院以重育贤书宫室闗梁以昭资世信方来则寺观祠庙之兼録鉴已往则陵墓古迹之并存述宦迹备举夫诸司取人才不遗于一善列女彰节行之异仙释见方技之竒文虽省而意则该实足徴而名不爽欲使宇宙数万里之外不出户而可知庶几上下千百年之间一举目而毕见岂徒备一时之广览将以垂万世之宏规此诚皇上经天纬地之文足以显继志述事之孝而光前振后者也臣等章句腐儒草茅迂士叨承委任忝效编摩奈海宇广大而难穷兼简册浩繁而莫防顾兹疎浅宁免漏遗虽殚铅椠之劳尚虑豕鱼之混仰睿谟之启发幸编简之克完冀垂宥于含宏愿少供于淸燕道配天徳配地八荒咸囿于生成车同轨书同文一统永传于悠久无任瞻天仰圣激切屏营之至谨奉表随进以闻天顺五年四月十六日资政大夫吏部尚书兼翰林院学士臣李贤等谨上表

　　京师
　　古幽蓟之地左环沧海右拥太行北枕居庸南襟河济形胜甲于天下诚所谓天府之国也我太宗文皇帝龙潜于此及缵承大统遂建为北京而迁都焉于以统万邦而抚四夷真足以当形势之胜而为万世不拔之鸿基自唐虞三代以来都防之盛未有过焉者也
　　城池京城【元志至元四年建大都城本朝洪武初置北平布政司于此永乐七年为北京十九年营建宫殿成乃拓其城周回四十里立门九正南曰正阳南之左曰崇文右曰宣武北之东曰安定西曰徳胜东之北曰东直南曰朝阳西之北曰西直南曰阜成】皇城【在京城之中宫殿森严楼阙壮丽邃九重之正位迈往古之宏规允为亿万斯年之固】
　　坛庙天地坛【在正阳门之南左缭以垣墙周回十里中为大祀殿丹墀东西四坛以祀日月星辰大祀门外东西列二十坛以祀岳镇海渎山川太嵗风云雷雨歴代帝王天下神祗东坛末为具服殿西南为斋宫西南隅为神乐观牺牲所】山川坛【在天地坛之西缭以垣墙周回六里中为殿宇以祀太嵗风云雷雨岳镇海渎东西二庑以祀山川月将城隍之神左为旗纛庙西南为先农坛下皆借田】社稷坛【在皇城内南之右中为方坛四面有门坛之周垣各依方色南有前门北有行礼具服二殿】太庙【在皇城内南之左正殿两廊楹室崇深昭穆礼制法古从宜亲王及功臣配享左有神宫监】文庙【在国子监彞伦堂之东正为大成殿东西翼以两庑前有防门外有櫺星门殿前旧有元加封孔子碑本朝正统中有御制新建太学碑文立于殿前庇之以亭】
