　　金上京会宁府考 

　　金上京会宁府盛京通志谓在今宁古塔西南方昶按会宁府非今之宁古塔也 【 盛京通志实沿明一统志之误高士奇扈从禄谓在沙林东南十五里曰火茸城亦沿明一统志之误通鉴辑览注宁古塔东南有和硕城即其遗址则沿通志之误通志又谓当在色齐窝集左右则沿杨氏柳边纪略之误说详后文满洲源流考云上京宫阙在拉林河东一百七十五里正与阿勒楚喀河相近其言是也又云拉林河阿勒楚喀河之间则金之上京城在焉今城南四里尚有古城及子城宫殿遗址亦最为明确惟卷十二又云上京会宁当在今色齐窝集左右岭上有故城遗址相传为金时关门其明证也云云又谓今宁古塔西南八十里有莲花池相传为金时之曲江方昶按源流考既知会宁为阿勒楚喀河近地又复沿诸家之误谓在宁古塔则是纂修之人不出一手故致前后矛盾如此】 
　　金史地理志上京路金之旧土海陵贞元二年迁都于燕削上京之号止称会宁府东至胡里改六百三十里西至肇州五百五十里县三会宁倚与府同时置有长白山有按出虎河又书作阿朮浒混同江涞流河曲江初名镇东大定七年置十三年更今名宜春大定七年置有鸭子河肇州防御使旧出河店也天会八年以太祖胜辽肇基王绩于此遂建为州县一始兴倚与州同置有鸭子河黑龙江考肇州故城在今伯都讷城南旧名出河店今名珠赫城 【 政和四年辑览注】 以今图按之伯都讷城在宁古塔西北几及千里道里不合一也志云按出虎一作阿朮浒按今阿勒楚喀河 【 水道提纲作阿尔楚库河即褚库河方昶按今土名阿什河皆按出阿朮之音转今于阿勒楚喀城建设阿城县治】 
　　距宁古塔亦八九百之遥其不合二也宜春县云有鸭子河按鸭子河即今伊通河辽圣宗改名混同江如会宁在宁古塔则县去府千余里矣其不合三也今阿勒楚喀河即按出虎水此水之西有白城在阿勒楚喀 【 今为阿城县治】 南四里遗址犹存按出虎译言金也建国此水之旁故谓金源确无疑义核与史文四至亦无差谬矣更以太祖伐辽进军次第证之本纪云二年辽人始为备调诸军于宁江州九月太祖进军寥晦城遣婆卢火督军诸路兵皆会于来流水太祖初至辽界督士卒夷堑遇渤海兵与战敌大奔进军宁江州诸军填堑攻城十月朔克其城次来流城以俘获赐将士考之辽志宁江州混同军观察使县一曰混同县当为今伯都讷是也 【 李氏地理今释云在今吉林乌拉城东北混同江东岸不知实在吉林乌拉之西北远甚】 十一月辽将步骑十万会于鸭子河北太祖自将击之未至鸭子河夜方就枕若有扶其首者三曰神明警我也鸣鼓举燧而行黎明及河辽兵方坏陵道选壮士击走之大军继进遂登岸俄与敌遇于出河店乘风击之辽兵大溃逐至斡论泺按鸭子河即混同江即今之松花江伊通河下流即注松花江虽圣宗诏改鸭子为混同然史文仍互用之 【 出河店之役金史以为辽将步骑会于鸭子河北辽史以为辽军屯出河店女直军潜渡混同江掩击辽众】 
　　登岸者济此水也出河店在宁江州之南即今伯都讷之珠赫城金置肇州于此 【 满洲源流考谓肇州在拉林之东甚误】 是肇州在伯都讷南之铁证如会宁在宁古塔何得云去肇州五百五十里耶史又载是月??业虺等攻宾州拔之考辽志宾州怀化军节度本渤海城统和十七年迁冗惹户置剌史于鸭子混同二水之间 【 太平四年始改鸭子河曰混同江】 后升兵事隶黄龙府都部署事按宾州之设在圣宗统和十七年黄龙府之废在景宗保宁七年府废于前州置于后所废之府原为渤海扶余府后于圣宗开泰九年迁城于东北复置黄龙府则非扶余府故地而为今之农安宾州之升节度在复置黄龙府之时故曰兵事隶黄龙府都部署事 【 黄龙府沿革详见于后】 松漠纪闻云契丹徙置翁舍展图于黄龙府南曰宾州州近混同江今之农安即在伊通河之侧伊通河即鸭子河当时改名混同宾州在农安之南故曰州近混同江知黄龙府之所在即知宾州之所在皆在肇州之南为无疑也 【 松漠纪闻翁舍展图国最小不知其始所居后为契丹徙置黄龙府南百余里曰宾州州近混同江即古之粟末河部落杂处以其族之长为千户统之方昶按此即后置之黄龙府为今之农安宾州在其南一百余里以今图证之即今长春府治之东北地居两河之间即鸭子混同二水圣宗改鸭子为混同故忠宣云地近混同江即古之粟末河亦即今之松花江也忠宣路程云七十里至宾州渡混同江亦谓此也李申耆以为宾州在辽阳西南其误最甚】 【 金太祖攻黄龙府次混同江无舟乘马径涉天眷三年改黄龙府为济州利涉军大定二十九年复改隆州利涉军贞佑初升为隆安府张石州蒙古游牧记引明一统志龙安一秃河在三万卫西北金山外又引全辽志龙安城在一秃河西金山东方昶按一秃即伊通又引册说城周七里门四址尚存旁有塔亦名农安又引一统志按金史地理志天眷三年改辽黄龙府为济州大定九年更为隆州方昶按本二十九年误作九年贞佑初升为隆安府以地考之此隆 【 隆：龙】 安城即隆安之讹云云穆按一统志于奉天府下又以黄龙府为在开原县境其说于此两歧今以地望诊之非是故不取方昶按今吉林省治西北二百八十里农安城在伊通河西二里西门外半里有农安塔足证农安龙安皆沿隆安而异其字盖通志考之不精两存其说不能详其废置之故即石州亦未深考也】 
　　综观史文金之进军始则自白城而西至克宁江州后乃由北而南若会宁在宁古塔则山径之间当属军道胡无一言及之耶更以史载世宗往上京日历证之大定二十四年三月壬寅如上京 【 三月庚寅朔壬寅为十三日】 四月丙寅次东京 【 四月己未朔丙寅为四月八日】 东京辽阳州也乙酉观鱼于混同江 【 乙酉为四月二十七日】 五月己丑至上京居于光兴宫 【 己丑为五月朔日】 按观鱼之所即今松花江由此四日而至上京可知决非宁古塔矣 【 今由农安至白城约三百里而弱四日可至】 六月辛酉幸按出虎水临漪亭 【 辛酉为六月三日】 按临漪亭故地今虽无考然不曰如而曰幸则为近地可知金地志云会宁州 【 州：县】 倚与府同置有渤野淀绿野淀今考白城西南有数湖泊名三哈拉 【 今为双城县地】 盖即所谓勃野淀绿野淀矣 【 又按续通鉴宋纪徽宗重和元年十二月马政等还自金政等以闰月六日下海行经十余州至金主所居拉林河注云旧作来流河今改又宣和二年遣赵良嗣王环使金考异引马扩茅斋自序云政和八年父政过海至女直所居来流河昶按据此则会宁在拉林河畔绝无疑义】 
　　李申耆沿会宁为宁古塔之误遂谓胡里改在今宁古塔之东齐次风沿会宁为宁古塔之误遂谓宁古塔河金时曰按出虎水即金源皆不足据 【 宁古塔即胡里改宁古塔河一名忽汗河由唐忽汗州得名又名虎儿哈一名瑚尔哈一名富尔翰兴胡里改之名称皆由忽汗二字之音转今名壮 【 壮：牡】 丹江其湖曰镜泊即渤海王城所临之忽汗海】 
　　魏默深开国龙兴记谓金起于黑龙江之南在伯都讷之东二三百里独为得之而不能实指为今之阿什河西之白城杨大瓢柳边纪略云宁古塔宋曰生女真金曰鹘里改元曰胡里改万户府又云洪忠宣路程上京三十里至会宁头铺四十五里至第二铺三十五里至阿萨铺四十五里至来流河四十里至报打廑铺七十里至宾州渡混同江则是金之上京确在今宁古塔之西混同江之东其去混同江仅二百六十里以今道里计之应在色出窝集左右方昶按杨氏之言近之然谓会宁在色出窝集左右则殊不合杨氏盖不知阿勒楚喀河即按出虎水故不能确指其地误以色出窝集当之且缘此之故而于宁江州亦不知为伯都讷误以为应在厄黑木站 【 杨氏云来流城即在宁江州西引金史太祖纪十月朔克宁江州次来流城为证不知来流即今拉林水一统图有拉林城即来流城也金太祖自宁江次来流城乃胜辽之后率师东旋非进军西向辽史萧兀纳传天庆元年知黄龙府事改东北路统军使金兵来侵战于宁江州兀纳退走入城留官属守御自以三百骑渡混同江而西城遂陷此事江州在混同江东之证】 
　　考厄黑木站在色齐窝集之南吉林乌喇之东北相差甚远拉林河在阿勒楚喀河之西混同又在拉林河之西故洪忠宣路程由会宁西行先至来流河后渡混同江朗然可证杨氏不知来流即拉林故歧误如此惟来流水与拉林水在昔本属通称而于辽上京临潢府之涞流河则别为一水音虽相同字则各别金史地理志于会宁县下误作涞流不可不辨 【 上京路下作来流河者不误】 洪忠宣路程云自上京百五十里至来流河北盟会编许亢宗奉使行程录皆云至拉林河此当日两水通称之证辽史地理志上京临潢府天显十三年更名上京府曰临潢涞流河自西北南流绕京三面东入于曲江其北东流为按出河以今图证之临潢府即今巴林旗东北之波罗城 【 宋薛映记富弼行程缘所言渡潢河黑水河皆历历可证潢河即今锡喇木伦河黑水河即今哈喇木伦河金史地理志庆州西至桓州九百东至临潢一百六十庆州即今巴林西北察罕城张石州蒙古游牧记云今巴林东北一百四十里当乌尔图绰农河会和戈图绰农河之处有波罗城周二十里内有三塔久废疑即古之临潢府其说是也近人曹廷杰东三省图说斥盛京通志谓临潢府当在今呼伦布雨尔之间之谬而又以张氏之说为非别指今科尔沁右冀前旗东南五十里之西伯城一作席百城席北城者当之殊为非是盖锡伯城去潢水远矣尚得曰临潢耶】 
　　所谓自西北南流绕京三面东入于曲江者舍今之洮儿河 【 一作淘尔河】 别无他水与所述流域方位相合者若谓涞流河即拉林水则在临潢府之东南其间尚隔有混同江安有所谓自西北南流绕京三面乎 【 拉林河西北流入混同江】 洮儿河上流有南北二大源发源乌珠穆沁右翼旗东北之索岳尔济山二水分流二百余里而合其下流今名洮儿河乃挞鲁河之音转亦即所谓他鲁河是也 【 辽志上京道云又有他鲁河圣宗纪太平四年改挞鲁古河曰长春河皆一水随地异名北史勿吉国传乙力支初发其国乘船溯难河西上至太沵河即此水太沵后转为它漏唐书黑水靺鞈传它漏河即此挞鲁又即它漏之转音一名???越河旧唐书室韦传东室韦在???越河之北其河东南流与那河合那河即难河今之潄 【 漱：嫩】 江也唐书云粟末水源于太白山西北注它漏河按即松花江与淘尔河相会之处两水既会合而东流史云粟末注它漏河尚属小误】 其水源正在临潢西北南流东至伯都讷北入于嫩江以达于混同江实绕临潢三面所谓东入曲江者即洮儿河注混同江之处水自西来故曰东入两水会合之处
