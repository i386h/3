　　钦定四库全书　　　　　史部六
　　谭襄敏奏议　　　　　　诏令奏议二【奏议之属】提要
　　【臣】等谨案谭襄敏奏议十巻明谭纶撰纶字子理宜黄人嘉靖甲辰进士官至兵部尚书襄敏其諡也事迹具明史本此编乃其厯官疏草分为三集曰闽稿嘉靖四十二年再起为右佥都御史巡抚福建时所上也曰蜀稿嘉靖四十四年起为陜西巡抚防大足民作乱陷七城调任四川以后所上也曰蓟辽稿隆庆元年至四年由兵部右侍郎进左侍郎兼右佥都御史总督蓟辽保定军务时所上也史称纶沈毅知兵为台州知府时即与戚继光立束伍法练兵破倭冦擒斩殆尽官浙江海道副使时又连破之再起为浙江右叅政时破饶平贼林朝曦调福建叅政时郡县多为倭所陷力战恢复闽地以平官四川巡抚时灭云南叛酋鳯继祖于防理总督两广时岑冈贼江月照等望风而降朝廷倚以办贼遇警輙调居官无淹嵗在蓟辽与戚继光协力邉办三卫诸部迄不敢南牧终始兵事垂三十年积首功二万一千五百计其功名不在王守仁下而儒者顾艳称守仁则以守仁聚徒讲学羽翼者众也今特録是集以见其谋画之大畧庶不没其实焉乾隆四十六年十二月恭校上

　　恭谢天恩疏【嘉靖四十二年二月望日题】
　　臣前任浙江布政使司右叅政兼按察司副使奉勑巡海嘉靖四十年三月十九日以父忧离任本年九月间因广冦流突江西伏蒙皇上以金革复臣原职事宁伏防皇上许臣终制四十二年正月间闽倭猖獗兵部又题奉钦依复臣原职时臣读礼甫及大祥伏惟大君制命时事孔臣子之私其何敢防臣随于三月初一日割情就道催取淛兵至本月二十八日进至建阳县防圣恩陞臣都察院右佥都御史提督军务兼巡抚福建臣领咨望阙叩头谢恩讫臣念闽海之冦盘据已乆徴兵方至易将临敌诚恐交代之间人心弛懈因致误事臣即破除常格凡事关战守未代之先一面施行寻于四月十一日入省准前任巡抚游震得将闗防符验旗牌并文巻与臣交代领事外臣惟兵戎为国之大事巡抚乃今之重臣必遴选之极精始付托之有效窃念臣以一介防贱滥厠班行才本不逮于中人学实未通乎军旅始由郎署而出守徒负恩私继秉宪节以临戎深惭尸素忽冐知兵之誉祗縁任事之愚两被命而墨缞已非所据荐承恩而秉孰谓其宜且八闽之受倭今防十载而易巡抚者四姓曽未有终彼皆抱竒杰之才尚无救地方之患如臣菲劣抑又何胜闻命自天措躬无地欲感恩而圗奋实有志而未能惟无改其实心或粗得乎胜筭更假之以时日誓竭愚忠不与贼共戴于以开八闽之太平纾九重之南顾臣之愿也亦臣之分也臣下情无任感激屏营之至
　　乞恩终制疏
　　淛江等处承宣布政使司右叅政兼按察司副使谭纶奏为江西地方流贼已靖仰遵成命恳乞天恩放回终制以广圣孝事窃念臣以一介草茅叨逢盛世仕防廿年官阶三品揣分弗堪圗报靡及不幸丧父苫居甫及两月即遭流冦之来毁形扶衬窜伏林莽艰难备厯有死无生伏防皇上悯念元元横被锋镝屡下哀痛之诏亟兴问罪之师江西臣庶皆焚香吁天祝愿皇上万夀无疆顾臣不自意驽下得以墨缞备数金革闻命兢惕措躬无所防时事戒严天威霆震臣乌鸟蹢躅之私未敢上请而犬马奔走之力因效前驱今幸皇上威广运庙万全渠冦授首三省底宁老终少长咸无失所之嗟孝子慈孙尽遂人伦之愿独臣三年有其懐之爱既与人同而一日愈于已之心未获自尽诚宇宙内之罪人为圣明朝之弃物恭防明防准臣以暂起复及奉钦依许臣以事宁守制天语昭如日星君恩侔于天地臣虽粉骨碎身无以为报伏望皇上念臣哀苦恻怛之情恤臣无所解于其心之私勅下该部仰遵成命放臣回籍守制待制终之日遵例听用则臣自此有生之年皆是效忠之日臣下情无任殒越恳切之至
　　水陆官兵勦杀新旧倭冦捷音疏【三月十九日题】
　　嘉靖四十二年三月十八日倭贼进至建阳巡按福建监察御史李邦珍趣臣督战臣启行间防兵备副使汪道昆呈称奉文防同副总兵官戚继光选募淛兵于本年三月十七日入境行至浦城据报新倭四百余徒自长乐县地方登岸劄营小址窥伺省城本道先发前部六营把总胡守仁褚应宿率兵兼程赴省贼闻兵至随奔福清县岱后地方总兵官刘显俞大猷各发官兵分伏要路截击四月初十日提督军门右佥都御史游震得亲督胡守仁等三营劲兵径趋福清县追勦贼闻大兵压境夤夜潜由江口逃遯本月十五日总兵官刘显防同俞大猷各发把总郭成与领江西援兵把总乐埙及领兵名色把总包堂郭文刘添庆刘添荣刘招桂龚仁朱相指挥魏宗瀚等率兵分伏要路本日夘时官兵齐出奋击俘斩首级一百四十五颗生擒一十六名夺回被掳男妇数百又准总兵官俞大猷报称本镇一面督发中军指挥魏宗瀚统兵合勦一面督令名色把总朱相带兵前往可通平海要路开沟防桥以截去路仍令各兵埋伏俟贼渡水邀击至日夘时前贼果突大营刘兵放铳为号伏兵四起刘兵从左冲击我兵从右夹攻贼众大败我兵乗胜冲杀追至开沟处所朱相伏兵齐起贼无去路擒斩过半余党势穷奔投入海不能取级我兵全胜各收回营本镇部下名色把总俞义重俞义安等官兵共斩获首级四十一颗四月十九日准总兵刘显开称本月十八日把总许朝光称平海倭贼知新贼杀尽即惊慌下船欲走因光防范严谨不敢开行至十六日夜风大作光将哨船驾至南向避风大船摘留平海系泊贼欺船少乗风揺驾大小船共三十二只一齐开洋光即弃艇急追连犂贼船四只乱放发熕火器铳沉二只焚烧三只贼见势蹙将财帛妇女抛弃入水投奔老岸比因昏夜不能尽杀止斩获真倭四十九颗夺回被掳男女一百三十名口兵夫阵亡五人被伤一十一人余倭复回旧巢见在督兵围勦除首级人口解发纪验外具报到臣为照福建地方被倭荼毒年甚一年民既凋残兵粮俱困毎遇贼至无所措手以致延日深荡平无计兹防皇上悯念元元俯从诸臣所请遣将徴兵荐出内帑济给粮饷用是军声稍振贼势消沮在陆则新来之冦悉已授首在水则舟遯之贼寻报成擒是皆仰仗皇上威逺镇庙万全将士用命所致也
　　飞报异常捷音疏【四月二十二日题】
　　嘉靖四十二年四月二十一日臣督兵驻总兵俞大猷营内据整饬福州兵备按察司副使汪道昆报称四月初八等日督募淛兵入省监督总兵刘显俞大猷副总兵戚继光三大营官兵尅期进剿原袭破兴化府城大势倭冦本月十九日兵至兴化地方探得前贼移据平海卫总兵官刘显俞大猷分屯渚林养威蓄鋭以圗大举令义士许朝光刘文敬各督舟师邀诸海上贼坐是困穷探知戚副总兵同本道统督大兵入境遂拔营前至酒林迤南据险结巢犄角迎敌二十日防本院督同本道与兴泉兵备叅议万民英布政司戴罪杀贼左叅政翁时器亲至渚林防集三大营官兵面议机宜分定哨道以副总兵戚继光统督把总胡守仁等部下官兵为中哨冲锋以总兵都督刘显统督把总郭成等部下兵为左哨而以江西巡抚都御史胡松所遣领援兵把总乐埙与前福建军门都御史游震得所发标兵把总陈仓等兵佐之以总兵都督俞大猷统督南贑军门都御史陆穏调发指挥魏宗瀚名色把总朱相等部下兵为右哨而以淛江巡抚都御史赵炳然所遣领援兵把总杨文与本院发与叅政翁时器监督领标兵把总陈其可蒋伯清傅应嘉部下等兵佐之仍悬赏冲锋银二万两以鼓作士气定以二十一日各照分定哨道刻期进攻军门各道随军监督本日丑时中哨官兵仍分三路鱼贯而进直薄贼巢贼亦分道拥出拒敌贼首跃马当先我兵勇气益倍三哨官兵一时并进贼势不支奔□回巢各兵乗胜攻打益急贼复并入大巢死守各兵四面围合因风纵火血战破巢贼众败匿房中或焚死或刃伤自相枕籍积尸充霤难以数计间以遁避山海我兵随处奋击堕崖蹈海死者数亦难计止据在阵敌杀生擒真倭首功二千二百余名颗器械称是夺回被掳男妇三千余人卫所印信一十五颗即今福州以南郡县并无一贼臣惟蠢尔倭奴突如内犯袭我郡卫辱我冠裳防刘我人民占据我城池流毒已经数月被害奚啻一方今乃仰仗皇上威广运庙谟审固文武同心将士用命遂使莫敢谁何之冦不崇朝而授首既防被据之城以一鼓而克复全闽按堵万姓欢腾捷书露布皆稽首吁天祝愿皇上万夀无疆臣待罪督抚仰念圣明万里南顾臣寝食靡宁今幸有此非常之捷除有功効劳人员另行查叙外恭先驰报上慰圣懐
