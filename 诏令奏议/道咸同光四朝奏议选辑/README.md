　　查办台湾清庄事宜疏（道光五年）赵慎畛、孙尔准

　　闽浙总督臣赵慎畛、福建巡抚臣孙尔准跪奏，为查办台湾清庄缉私各事宜事。

　　窃照台湾孤悬海外，民无土著，俗本轻浮，素有内地游民，偷越私渡，此等游手好闲之人，既无艺业可守，又无田地可耕，且懒惰性成，即佣工力作，亦复不耐辛劳，非匿迹于赌场，即潜踪于鼠窃，异乡飘泊，无可稽查，恐渐聚渐多，成群结党，当地方无事之时，肆行抢劫，为害闾阎，而台郡率皆漳泉民人，素尚忿争，一遇雀角微嫌，该游民等即乘机附和，助势逞凶，小则激成械斗，大则谋为不轨。上年南路匪徒许尚、杨良斌等，纠众滋事，即恐有游民从中煽诱。幸该镇道府等拏办妥速，当将首伙各犯，歼除殆尽。但内地游民，尚未绝迹，亟须查逐清厘，以为久安之计。然欲查逐游民，莫善于清庄一法，其事较编查保甲，尤为简而易行。前年臣等因延平、建宁等郡捕务紧要，曾饬各属于保甲之外，小为变通，办理连甲，颇有成效。清庄亦与联甲相仿。上年臣孙尔准赴台时，亦饬台属力行清庄之法，毋许容留匪类，均于稽察棚民覆查保甲案内，先后奏蒙圣鉴在案。缘台湾向系漳、泉、粤三籍之人，分庄居住，各庄设有头人，庄民咸听头人管束。臣等见复分饬台湾镇道府，督饬厅县，责成该头人，率同客长乡耆，并庄正、保正人等，实力查察。如有面生可疑，或在古庙凉亭及饭歇处所飘流无定之人，查无亲属相依者，既属外来游民。该头人等立即禀报地方官，讯明内地籍贯，照例逐令过水，面刺过水二字，递回原籍安插，毋许复行偷渡赴台；即在夫行投充小夫者，亦令夫头查明，果系良民诚实安分者，出具保结，仍准在行充夫，如来历不明，及好勇斗很，动辄生事之徒，亦俱报明本管官，一律驱逐回籍。该头人等，各清各庄，家喻户晓，既无胥役纷查之扰，又无门牌册籍之繁。各庄民自卫自家，亦乐于从事。台地各厅县，一据头人禀报，立时讯明，先将该游民严加看管，或数人或数十人，分起押解回内，海洋险阻，必须妥为管解，亦令议立章程，以免中途滋事疏脱，俟解至内地入口，即由地方官逐程递解，并通饬泉州、漳州各府厅县，如遇台湾递解游民到境，按照籍贯，饬命该处乡耆族房取保，严行管束，毋许私自外出，一面分檄内地，台湾该管各文武，严饬守口员弁，凡遇船只出入口者，务须实力盘诘。如有奸民越渡台洋，即将人船拏解究办。似此稽查严密，庶游民不能溷迹赴台，地方获就安谧。至台湾府盐务，向由贩户承办，自行选雇哨捕巡缉，见闻该处沿海奸民，私设埕坎，擅自晒盐贩卖，惟嘉义县境内为尤甚。此等晒私匪类，即系游民，若不及早查拏，日久啸聚，亦足为土地之害，业经飞饬台湾道府，督饬厅县，选派干役，协同贩户，严紧查缉，将各处私晒匪徒，拏获解究，俾私贩敛戢，官引畅销，不特课赋民食，无短绌匮乏之虞，且令匪棍流民，无托足谋生之所，则内地奸徒，闻风畏缩，不敢再行前往，而台地得以一律肃清，实于海外要区，大有裨益。惟台湾重洋间隔，非如内地之耳目易周，恐该厅县以距省窵远，意存泄玩，臣等见在密加访察，并严札通饬该厅县等，如将境内游民人等，认真细查，押逐净尽，使民间不受扰累，即系实心任事之员，定当量予奖励。如奉行不力，或由于办事懈怠，或限于才质迂疏，虽经查逐，未能悉数驱除，即不加紧搜查，有初鲜终，亦难胜海疆之任，应即撤回内地，或改补简缺，或降补佐杂，分别轻重，奏明办理。若竟阳奉阴违，或畏难苟安，膜置不办，则自甘暴弃，立即严参示儆。如此酌定劝惩，使各厅县，共知奋兴警惧，方可冀收实效，仍令该镇道府，就近确查，随时据实禀报，毋得稍事徇护，并干重咎。臣等为海疆地方紧要起见，谨恭折具奏，伏乞皇上圣鉴训示。

　　会议台湾善后事宜疏（道光十四年）曹振镛等

　　大学士臣曹振镛等跪奏，为遵旨会议具奏事。

　　道光十三年十二月二十三日，程祖洛奏，台湾善后事宜一折，奉朱批军机大臣会同该部议奏钦此。臣等谨会同各该部悉心核议，为我皇上陈之。

　　一、原奏内称禁偷渡以杜盗源等语。臣等查内地自福宁以讫漳州，无处不可偷渡。泉州府属之崇武等处，渔船皆可渡台，偷渡尤易，并有潜身海滨，俟正口商船载兵哨船验放出口，即用小船载登附搭到台后，沿海之浅水浮埔，无处不可登岸，逃逸凶盗，违禁货物，胥由此脱漏，台湾遂成藏垢纳污之所。该处人情本极浮动，而又有此等桀黠好事者趋之若骛，无怪乎屡有不靖之案也。此等偷渡匪徒，无客头船户为之引领揽载，重洋岂能飞越。客头船户，若不勾通澳甲汛兵，断难作奸。其口员徇情私给照票，更所不免，应如所请，嗣后渡台民人，照例由地方官给照票，至正口验放配渡，赴台投缴原照，仍责成给照之州县，验放验收之台内管口厅员，按月造册通报。如查无照票，或人照不符及所执系口员私票，即将本人照例治罪，仍严究客头船户、澳甲汛兵姓名，指拏究办。如不将客头人等据实拱指，即将本人坐以客头等应得之罪。商船哨船夹带，坐船户及管驾弁目以客头之罪。至各处偏僻港澳及泉州府属之崇武等处，责令文武印官，督率分汛官弁，逐段清查。凡有渔船之可放大洋者；一概押归正口，以便稽查，不得停泊私澳。其梁头短窄不能远涉者，仍听其便，并令台湾水陆汛弁，在于浅水浮埔支河■〈氵义〉港，互相稽查，见有船只，亦即押赴正口，如遇涉水登岸之人，将人船一并拏交地方官讯办，仍严饬分汛官弁，不得藉此需索扰累，别滋事端。其内地口员疏于防范，照例参处，私给照票者，照知情卖放例革职治罪。台湾口员，例无失察内地民人偷渡处分，惟既有偷渡之人，难保无脱漏米粮情事，应如所请，比照内地口员例一体参处。

　　一、原奏内称行清庄以除盗薮等语。臣等伏查台湾清庄章程，与保甲略同，而更寓团练之意。惟形势袤长，村居星散，言语咮■〈口离〉，地方官赴乡查办，不能不藉书役为引路通事。该书役等惮于繁远，或竟得规包庇，不免指东画西。查台湾各营，千总以下等官之分防汛地者，有一百十五员，较文质汛官多至十倍，籍隶闽省，土音是操，所辖地面，亦甚有限，不难亲历清查。慎选总董，编联保甲，应如所奏。嗣后每年秋收后，由台湾镇道，遴委干员编查一次，如遇原册无名之人，或去来人数与所报不符者，即拏交地方官严讯究办，委员汛弁编查不力，由台湾镇道查明详参，倘实心经理，计功奖赏。

　　一、原奏内称严连坐以杜包庇等语。臣等查台湾总董庄耆牌头甲长人等，未必尽贤而有力，前年逆匪粤棍，俱系各庄无赖，并非一乡一里之人，各庄总董人等，果能知一家有犯，十家连坐，自必各顾身家，何至纵令滋蔓。应如所奏，清庄之后，如有来历不明，为匪犯法之人，该总董庄耆保甲邻右，并不随时捆解，发觉到官，一并连坐，文武汛弁，亦不得无故恐吓平民，致滋事端。

　　一、原奏内称禁搬徙以免窥伺等语。臣等伏查台湾人情浮动，而又屡遭不靖，悉属惊弓之鸟，奸徒凭空造谣，或称闽粤分类，或称匪徒竖旗，闻者纷纷迁避，闽粤既各怀疑惧，棍徒即乘机构煽，此实致乱之由也。前年逆匪窥伺笨港镇，该处总理林西泰等，联约各庄，不准妄动，三战三捷，嘉义境内惟该处最为完善。应如所奏，嗣后如有闻谣迁徙之事，先将该总理枷号示众，再严拏造谣之人，照例治罪，庶几人无衅焉，妖不自作矣。

　　一、原奏内称实力化导以挽颓风等语。臣等伏查台湾之俗，嗜利尚气，而鲜仁让之风。百数十年来，教养涵濡，读书明理者，固不乏人，而乡曲愚民，不知择师，故讼师奸棍，即藉训蒙以为构煽。此次逆案内之伪军师等，大半皆蒙师也，应如所奏，嗣后责令台湾镇道府厅县及驻防乡镇之县丞巡检等官，每逢朔望，传集衿耆，敬谨宣讲圣谕广训，讲明父子、君臣、长幼之道，身心性命之理，使知孝弟忠信，即可以造于圣贤，并责成各学教官，于农隙时周历稽查，训蒙之人是否皆堪为后生矜式，明示奖罚，如有奸匪溷迹及荒谬怪癖之徒，即送县讯办，道府厅县教官，每遇听讼课士，或因公下乡与民接见，随时训导，务令正学明而人心定，俾愚夫愚妇，皆知为善之乐，则风俗自化矣。
