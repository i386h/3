　　一、巡台御史觉罗柏修等奏折

　　巡视台湾陕西道监察御史臣觉罗柏修、巡视台湾兼理学政兵科掌印给事中臣高山谨奏为奏闻事：窃台湾四县，每年额征供粟一十六万六千余石。其贮粟仓廒，系每石派盖仓银六分，修仓银二分，约有一万三千二百余两。除支放兵米并拨运内地，所存仅有数万，何需每年摊派如许之多？再加胥役斗级各项需索，小民殊属为累。但系陋弊相沿，难以穷究。臣等思积贮仓廪，贵于充裕，而存留谷数，宜有常经。查台湾现在贮仓之粟，计有七十余万，似不为少。应将每年新收供粟存仓，将年久仓粟除散给兵米，其余运贮内地。即此七十余万以为存台额数，则仓廒自有定规，每年只用小修，官吏无从苛索，小民实受无穷之福矣。如臣言可采，伏乞皇上谕旨施行。为此谨奏。

　　雍正十一年三月初三日，五月初四日奉朱批：该部议奏。

　　——录自明清史料戊编第一本二七页。

　　二、福建巡抚卢揭帖

　　巡抚福建等处地方提督军务都察院右副都御史纪录四次驻札福州府卢谨揭为钦奉上谕事：据福建布政司布政使王士任呈详：奉臣案验，乾隆元年九月初六日，准户部咨：福建司案呈，乾隆元年八月初八日，内阁奉上谕：朕爱养元元，凡内地百姓与海外番民，皆一视同仁，轻徭薄赋，使之各得其所。闻福建台湾丁银一项，每丁征银四钱七分，再加火耗则至五钱有零矣。查内地每丁征银一钱至二钱、三钱不等，而台湾则加倍有余，民间未免竭蹶。着将台湾四县丁银，悉照内地之例酌中减则，每丁征银二钱，以纾民力。从乾隆元年为始，永着为例。该督抚可速行晓谕，实力奉行。若因地隔海洋，官吏等有多索滥征等弊，着该督抚不时访察，严参治罪。钦此。钦遵抄出到部，相应行文福建督抚钦遵可也等因，咨院行司。奉此，又奉总督闽浙郝部院案验，准咨同前因行司。奉此，遵即移行台湾道、府钦遵遍示晓谕在案。今据台湾府申：据淡、澎二厅、台、凤、诸、彰四县士民祝万年、施士安、王世恩、黄恩溥、陈国辉、张方高、蓝日宠、潘俊、陈应魁、吴振纶等佥呈称：为皇恩无远弗届，台民受恩倍深，公吁详题，以达舆情事：钦惟我皇上帝恩广被，已遍及于九州岛；圣德覃敷，屡沛泽于海甸。蠲租赈恤，千百万生灵咸沾雨露；发帑赏锡，南北中民番均沐恩膏。万年等海氓顶祝高厚，固已感激无涯。兹蒙睿虑周详，又垂念台地丁粮每丁征银四钱七分，较之内地稍重，特沛恩纶，酌中减则每丁止征银二钱，以纾民力。闻命自天，欢呼遍地。男妇老幼，群思赴阙谢恩。祗缘九重听远，下情难以上达。用是各执瓣香，相率北望泥首，叩谢皇恩。伏乞赐文详请题达，俾展草野蚁诚。万年等曷胜沾感之至等情到府。据此，该台湾府知府徐治民看得：台地丁粮，每丁额征银四钱七分六厘，较之内地稍重。兹蒙皇上仁恩浩荡，垂念海疆，特颁上谕，酌中减则，每丁止征银二钱。恩纶下贲，万姓欢呼。士农工贾，共颁帝德同天；男妇老幼，齐祝皇仁匝地。踊跃忻忭，感激陈词，叩谢皇恩，情形恳切。卑府职司民牧，不敢壅于上闻。合就据情详请恩赐详题，以达民情，实为恩便等由到司。据此，该本司布政使王士任钦惟我皇上至德如天，深仁协帝，全蠲积赋，既已惠普寰区，酌减丁徭，尤独虑周遐甸。天纶特沛，舒海外民力而群臻乐利之休；旷典频颁，合台郡生灵而悉归清宁之域。百千岛屿，家家共沐恩膏；亿万边黎，世世永沾雨露。宜民番之欢呼遍地，感激陈词。兹据台湾府知府徐治民详，据淡、澎二厅，台、凤、诸、彰四县各士民祝万年，施士安、王世恩、黄恩溥、陈国辉、张方高等各户丁佥呈，各执瓣香，相率北望泥首，叩谢皇恩，转详请题前来，合就转请，伏候察照代题，以伸谢悃等因到臣。

　　据此，钦惟我皇上恩周万类，德洽群生。免赋蠲租，久庆盈宁于海表；轻徭减则，特施雨露于台阳。合民番而均戴皇仁，欢呼雷动；普亿兆而悉沾圣泽，忭舞嵩齐。从斯丁口繁滋，咸颂一人之有庆；自此烝黎广育，共祝万寿以无疆。臣接准部咨，立即钦遵转行，出示晓谕。兹据布政使王士任详，据台湾阖郡丁户祝万年等佥吁详题，恭谢天恩等情前来。臣不敢壅于上闻，谨恭疏代题，伏乞皇上睿鉴施行。再照：总督印务系臣署理，毋庸会疏，合并陈明。为此除具题外，理合具揭，须至揭帖者。右具揭帖。乾隆元年十二月十二日。

　　——录自明清史料戊编第一本三九页

　　三、闽浙总督郝玉麟题本

　　闽浙总督衔专管福建等处地方军务兼理粮饷兵部尚书兼都察院右副都御史世袭骑都尉加七级纪录四十二次功加一等驻札福州府臣郝玉麟谨题为钦奉上谕事：乾隆二年五月三十日，准兵部咨开职方清吏司案呈，乾隆二年四月二十五日，内阁奉上谕：朕查闽省澎湖地方，系海中孤岛，并无田地可耕。附岛居民，咸置小艇捕鱼，以餬其口。昔年提臣施琅倚势霸占，立为独行，每年得规礼一千二百两。及许良彬到任后，遂将此项奏请归公，以为提督衙门公事之用，每年交纳，率以为常。行家任意苛求，渔人多受剥削，颇为沿海穷民之扰累。着总督郝玉麟宣朕谕旨，永行禁革。其见在捕鱼船只，饬令该地方官照例编号稽查办理。此项陋规既经裁除，若水师提督衙门有公用必不可少之处，着郝玉麟将他项银两酌拨数百金补之。钦此。钦遵抄出到部，相应行文该督可也。合咨钦遵施行等因。又准户部咨同前因各到臣。准此，随经钦遵出示宣布晓谕禁革，并移行遵照外，兹准福建水师提督臣王郡咨称：为照本提督衙门有应援通省之责，所遣官兵，不有赏罚，无以劝惩。如每年两班哨巡，春秋二季大操，以及按期操考、获匪缉私、远近差遣，动需赏给，此乃按岁必不可少之公用。向来俱就澎湖渔例内支应。今此项既经奉旨禁革，所有公用之费，见荷圣恩体恤，交与本部院将他项银两酌拨补之，自应听候酌裁而行。相应据实咨明，请烦察照施行等因到臣。

　　准此，该臣看得：接准部咨，钦奉上谕：朕查闽省澎湖地方，系海中孤岛，并无田地可耕。附岛居民，咸置小艇捕鱼，以餬其口。昔年提臣施琅倚势霸占，立为独行，每年得规礼一千二百两。及许良彬到任后，遂将此项奏请归公，以为提督衙门公事之用，每年交纳，率以为常。行家任意苛求，渔人多受剥削，颇为沿海穷民之扰累。着总督郝玉麟宣朕谕旨，永行禁革。其见在捕鱼船只，饬令该地方官照例编号稽查办理。此项陋规既经裁除，若水师提督衙门有公用必不可少之处，着郝玉麟将他项银两酌拨数百金补之。钦此。臣钦遵谕旨，随即出示宣布皇仁，晓谕禁革，并移行遵照去后。兹准福建水师提督臣王郡咨称：水师提督衙门有应援通省之责，所遣官兵，不有赏罚，无以劝惩。如每年两班哨巡、春秋二季大操以及按期操考、获匪缉私、远近差遣，动需赏给。此乃按岁必不可少之公用，向来俱就澎湖渔例内支应。今此项既经奉旨禁革，所有公用之费，见荷圣恩体恤，交臣将他项银两酌拨补之，应听臣酌裁等因。臣查陆路提督衙门，亦系应援全省，从前曾于减报倒马案内，经臣奏请于下剩朋银内拨给银五百两以为赏号之需，荷蒙世宗宪皇帝俞允在案。今水师提督衙门事同一例，相应仰恳皇上天恩，一视同仁，俯准每年酌拨银五百两以为赏号等项之用。其银亦请于减报倒马下剩朋扣银内支销。理合遵旨具疏题明，伏乞皇上睿鉴施行。为此具本谨具奏闻。乾隆二年七月初二日，闽浙总督衔专管福建等处地方军务兼理粮饷兵部尚书兼都察院右副都御史世袭骑都尉加七级纪录四十二次功加一等驻札福州府郝玉麟。
