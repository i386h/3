序

光绪二十四年正月

朝廷俞贵州学政严修之请援康乾间两开鸿博例

诏开经济特科抚髀求才甚盛心也第经济者经义以治事特实见施行之谓岂与夫鸿博可程其尺利于豪楮间哉其能者不肯应其应者策马以赴不求闻达之科有识唏焉非但试者寡也抑主试者亦实难茅苇收而不知辨亦弊而已矣乌睹所谓经济者乎愿海内学子久病思啑一闻其名灵滃起砥淬铎锷争向新的然则政教学术亦视

明旨所以风示之者何如天下固无不可变之法耳其后未举而辍二十七年三月复有举行特科之

谕老友润甫髦不废学孤怀闳识衿缨不逮所辑经世文统编适成昔者邵阳魏氏为善化贺氏增损吴江陆氏切问斋文钞为经世文编一时纸贵几乎家荆璧而人隋珠矣续编自以阳湖盛氏为善本东乡饶续逊其富上海葛续逊其翔实然缘体立迾要可观览丁戊之交弋名之士射利之估三编外编新编信手捃摭为急就章等之郐下今润叟共炉而冶依经济八科以为之目其无可专属者以通论着二目系之蕝分瓜别绥绥有章都凡

国朝宝臣魁士有用之文略具于是或东西人士所譔有稗政学亦着焉卷帙伙颐虽所割隶容有出入过而存之间亦不免乌虖可谓勤矣窃尝论之天下为三大界洪水为一界郡县为一界互市为一界天地所开谁能锢之神州陆沈匹夫有责辑成一书予以镜求古今之得失斠审中外之异同亦有志者之事也是编之成虽茵席诸编而剪薙野言确有别择学者得而治诵之经济要则或不尽此其亦有以副大科而济时变矣倘所谓愈愚之砭针馈贫之资粮无作为巽岩李氏所编之制科题目观可也浙东汤寿潜叙星江俞浚●书
