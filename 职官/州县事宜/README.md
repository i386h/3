　　钦颁州县事宜世宗宪皇帝谕旨 

　　奉上谕牧令为亲民之官一人之贤否关系万姓之休戚故自古以来慎重其选而朕之广揽旁求训勉告诫冀其奏循良之绩以惠我蒸黎者亦备极苦心矣惟是地方事务皆发端于州县头绪纷繁情伪百出而膺斯任者类皆初登仕籍之人未练习于平时而欲措施于一旦无怪乎彷徨瞻顾心志茫然即采访咨询而告之者未必其尽言无隐此古人所以有学制美锦之叹也向以大学士朱轼左都御史沈近思外任多年周知地方利弊雍正二年曾令二臣商酌规则以为州县之南车乃书未就绪而沈近思物故迩年以来朱轼复时多病此事遂至迟延去年始降旨委诸总督田文镜李卫今二臣各抒所见缮录诸条以进朕亲加披览见其条理详明言词剀切民情吏习罔不兼该大纲细目莫不备举诚新进之津梁庶官之模范也在二臣各就其所阅历者而言繁简同异之间不必一致而慎守官方勤恤民隐兴利除害易俗移风其大旨则一而巳矣爰就本文付之剞劂颁赐州县官各一帙俾置之几案问朝夕观览省察提撕治效未臻必思所以勉之弊端未革必思所以去之本之以实心行之以实力毋始勤而终怠毋静言而庸违如此则不但国家得司牧之贤草野有父母之颂而尔等身膺显擢叨被荣光福贻子孙名标史册岂不美欤特谕

　　○到任 

　　州县初膺民社。通籍仕版。上为圣天子抚字符元。加惠地方。致身事君。始基之矣。正宜崇简持约。饬官方。励操守。以图远大。建白当时。无如俗吏以到任为荣。而奸胥喜以铺垫为事。衙门则必重为修饰。轿伞则必另制新鲜。甚至毡彩围屏。务求华丽。桌几灯烛。不厌碎烦。在长官释褐方新。扬扬入署。视为衙门旧例。快意当前。而不知派累行户。苛敛里民。追呼悉索。已费中人之产矣。其实备办供官者。不过十之二三。而侵蚀分肥者。已踰十之七八。是民为鱼肉。官为唇齿。恣意大嚼。乃供羣蠹之饱。俾其扪腹匿笑。岂不愚哉。况先声狼籍。民口难防。一遇铁□之上司。则朝垂黄绶。而夕登白简。亦又可惜也。故于未到任之先。必须严切发谕。禁止铺垫。务要棱棱铮铮说。得啮钉嚼铁。则伊等捧檄而读。便已心凛此官具有风力。不敢擅自派敛。然非徒托诸空言也。到任之后。青毡原有故物。环堵且自萧然。地方之一糹?系一粟。一物一器。不但不取。并亦不借。临时需用。发价平买。吏既不能为奸。民亦无从滋扰。新政精明。人歌来暮矣。但胥役巧猾。小民愚懦。保无有阳奉阴违。藉名暗诈者乎。此又在留心密访。遇有发觉者倍法以惩。慎勿令若辈当面卖却也。 

　　○交盘 

　　州县到任。交盘乃第一要务。盖今日之受于人者。即异日之交于人者也。苟不慎之于始。鲜不贻累于后。虽不可刻薄居心。过于苛求。尤不可狥情滥接。代人受过。亏足务得其实。巨细务得其详。清查之法。不可不知也。如地丁钱粮则应查其全书额编实征银数。按前官到任离任月日。任内征收过若干。解支过各若干。应存库银若干。未完民欠若干。分年分款。先清其纲领。将应存库银即按款照数兑交。其征收数目。又必验其串根流水。核其日报总簿。而串根流水报簿。尚有假捏改造。征多报少之弊。其解支款项。又必照验其批回。稽查其领状。而领状批回。尚有洗补年月。改换银数之弊。均当细察。然有批领未交。而前官称系解过某项。支过某项者。则当详明所解衙门。查明所领员役。果非虚捏。方可凭信。至于民欠。清厘尤难。官侵役蚀。多暗藏于民欠之中。新任之员。第知责成于经管钱粮之总吏库吏经承柜书。取其并无捏欠。愿甘赔补之结状。不知此辈或受前官贿嘱。或系已身侵蚀。通同狥隐。彼此欺瞒。结状实为具文。是在责令经承柜书。开造花名细册。先查某年民欠总数。次查某里某甲民欠细数。然后将花户姓名。欠银数目。大张告示。晓谕通知。有以完作欠。以少作多。令本户告理。更须出其不意。摘拘完纳。查询虚实。一有弊端。即穷究到底。官侵则据实揭报。役蚀则立即监追。务使户户无虚。厘厘着实。不但已身无赔累之虞。即小民亦免重纳之累。此清查民欠。尤为交盘中第一要着也。若夫仓储米谷。颗粒为重。从前州县多用量仓之法。用杆丈量。约畧算交。其中有无亏缺霉烂。不能确知。今仓粮交盘。已按米谷之多寡。定限期之迟速。自当彻底盘查。如有亏空短少。霉烂糠粃。即据实揭报。其有前官私粜未补。折价留交者。断不可违例接收。至奉文平粜。虽非前官私动。然须察其实在粜卖之数。催交价银贮库。惟民借一项。其弊多端。有本系亏空。乘奉文出借。以少报多。捏混掩饰者。有远年民欠。改换领状。以旧作新者。有奸胥蠹役。侵收冒领。混人民欠者。尤不可草率误征。必须详细查察。如查钱粮民欠之法。开造花名细册。另取欠户领状。以杜诸弊。他如水旱驿递船只。则查其是否齐全整备。马匹则查其有无疲瘦缺少。农坛藉田。忠节祠宇。仓廒城垣。则查有无坍损。此外或有前官经修工程。经手钱粮。又当随事察核。不可使有纤悉遗漏。然此皆就关系钱谷者而言。至于钦部事件。命盗重情。承审迟延。失察违匿。接任官均有处分。亦何可以刑名不入交盘。而姑置勿论也。总之前官之事。一交卸于后官。即为后官之肩承。欲免后累而尽职守。则授受之际。不可不慎也。 

　　○关防 

　　百官分职。各有攸司。虽体制不无大小之分。而关防则有内外之辨。所以肃政治。别嫌疑。防奸伪也。况州县职司民社。堂隅最近。一不加谨。则内外交弛。情伪百出矣。然而关防者。非徒严锁钥。扃门户之谓也。盖州县一官。刑名钱谷。难自为理。事权所托。易启钻营。则关防首在幕地。一人出仕。举室皆荣。子弟相依。渐为耳目。则关防又在亲戚。管宅门。管仓驿。管签押用印等项。处处需人。事事有弊。则关防多在家奴。用火夫。用水夫。用厨役泥水等匠。出入不觉。传递最便。则关防下在厮养。更有星流相士。墨客山人。捕影捉风。诈称交好。拉蓬扯纤。遇事招摇。此尤不在衙门之中。而关防更宜周密者也。故州县到任。延请名贤。分勤夙夜。而商酌办公之地。不得闲杂窥探。走漏机事。二三子弟。代治家政。而数米量盐之外。不许干预公事。滋其威柄。长随家人。择其诚朴堪信者。管宅门。管签押。而驿递仓粮。自有经承可问。何须僮仆。夫役匠作。酌其当紧。自宜需用。而出入作息。则须稽察维严。防有指使。至于乡村寺观。不时查访。流寓藉名。即行拏究。如此则内外不既肃清。而关防之道。其尽矣乎。然皆事之在人者也。倘利欲有以熏心。苞苴不无染指。目无定见。耳有偏听。器量不公。或喜怒而任性。面皮不冷。或瞻顾以狥情。则是在已一身。全无检束。众人匿笑。奚有遵循。夫身任地方。而欲澄清吏治者。其于众人之关防。固不可不严。而于已身之关防。尤不可不切也。故曰关防者。非徒严锁钥。扃门户之词也。 

　　○宜讲圣谕律条 

　　伏读圣祖仁皇帝圣谕十六条。大哉王言。化民成俗。六十载熙皞从风。千百国梯航向义。良有由也。恭逢我皇上承天立极。法祖施仁。乃于万几余暇。阐发精微。昭垂广训。其所以正人心。端世教者。剀切详明。巨细毕举。此诚天下率由之旧章。万世遵循之宝鉴也。又悯小民无知犯法。特命刊示律条。使之家喻户晓。庶几革面洗心。仰见皇上保赤之怀。提撕警觉。仁至义尽。有加无已。凡为州县者。父母斯民。首先教导。每遇朔望两期。务须率同教官佐贰杂职各员。亲至公所。齐集兵民。敬将圣谕广训。逐条讲解。浅譬曲喻。使之通晓。并将所刊律条。亦为明白宣宗。俾知凛惕。至于四外乡村。不能分身兼到者。则遵照定例。在于大乡大村。设立讲约所。选举诚实堪信。素无过犯之绅士。充为约正。值月分讲。印官仍不时亲往查督。以重其事。总之宣讲时。必要州县官先有一段实心。认定此事不是虚文。谆谆然如父之教子。如师之诲弟。恳切往复。津津不倦。然后精诚浃洽。百姓环听之下。亦自触动天良。悦于耳。得于心。民日迁善。诚有不知其为之者。若但视为习套。朔望之辰。鸣锣张葢。前诣城隍庙中。公服端坐。不出一语。视同木偶。而礼生绅士。请诵圣谕一遍。讲不晰其义。听不得其详。官民杂沓。哄然各散。上无以扬圣天子启牖斯民之化。下无以振众百姓观感自新之风。此俗吏之苟且因循。所谓尸位而素餐者也。贤有司励志功名。将为朝廷宣德意。加惠元元。可不于此知所重哉。
