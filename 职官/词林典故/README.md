　　御制词林典故序
　　词林典故书成大学士张廷玉等以序请朕惟六经之士岂易言哉而况跻玉堂而列芸署者尽六经之彦也木天故事厯代沿革具见于斯则又何言惟是国家重馆阁之选极优遇之隆讵止蜚其英声将以华国而已哉如濓溪所谓文以载道者舎是无他求夫布衣韦带之士由立言以期不朽足矣若夫国家右文重道将以淑世熙绩继徃圣之絶学开万世之太平胥是赖焉名之盛者实难副任之大者责弥重誉之来者毁亦随其尚思春华秋实之喻而凛虚车麟楦之讥哉
