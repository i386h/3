　　○详河道宪兴化水利禀 

　　窃卑职自到任以来查考兴化情形士习民风已非昔比然或者勉竭驽骀妄希转移于万一惟查下河水利有谓治其源者则复归仁堤以遏黄流使淮水势杀庶运河堤可保无虞也有谓导其流者则浚白驹小海草堰诸闸使由牛湾河入海庶不至白涂等河壅塞也有谓复十四塘者纳泗州贝于贻天长诸水使不至泛滥于高宝诸湖致伤运河堤也或议开子婴河闸浚射阳湖神台出天妃口入海则高邮以下水势减泄庶不至决坝破堤也或议开瓜洲仪征二闸挖郡城东之沙坝及芒稻河坝则河湖两水皆可稍减而高宝运堤即可以不惧也更查芒稻河中钉品字桩祗令通水不令通船故私盐不得偷渡若开沙坝即照芒稻河钉桩再开瓜闸亦照芒稻河钉桩视湖水之增减为启闭则湖水南流愈多减闸之水自杀是亦权宜之计也然皆言易行难工费浩大惟查芒稻河奉高宗纯皇帝上谕永远开放泐石现存而盐务以为有妨挽运于芒稻河下流另筑拦江坝以遏之是以下河七州县迭有灾年夫果来源太旺江潮顶上虽拦江坝开犹恐宣泄不及至以盐运省费之故而使亿万生灵尽受淹没不惟民命至重即钱漕之岁入其视盐运之牵费相去几何耶傥芒稻河可以永远开放则沙坝瓜口皆不复再开矣卑职甫经到任祗悉大概情形而就中可行不可行之处尚未能了然于心除更详细审查另行申禀外现虑夏秋之际水势涨发即为亿万人生命所关身职司牧能无惴惴仰维大人宣力有年下河之灾早廑慈虑伏乞不弃菲材详明指示俾得有所依据以为救败之图则作德作福心祝瓣香矣 

　　○请随时开放拦江坝以全下河七州县禀 

　　伏查自黄河徙夺淮流河强淮弱不能并流归海全淮之水汇注洪泽湖者不得不开高堰河坝归高宝等湖导之入江一线漕堤遏障东注以保下河州县危乎危矣入江之口岸自金湾新旧闸而南则东西湾凤凰桥壁虎桥湾头闸人字河芒稻河等处前河督宪张以上游之来源宽于入江之去委量源度委奏起芒稻七闸矶心开瓦窑铺新河使去委与来源相当疏通宣畅必来源甚旺去委全开犹属宣泄不及万不得巳乃开高邮以南之五里坝新坝草坝车逻昭关等坝东注下河归海但各坝开放水性盈科而进必先灌满下河民田积阻于民灶分界之范堤乃由兴化东台两邑之丁溪草堰白驹刘庄四闸受水东注归鬬龙港入海其在盐城者则东门之石□闸北门之天妃闸皆归新洋港上冈镇之闸入野潮洋其在阜宁者则南门外之大关归通洋港野潮洋浅窄不足以资宣泄其三港受水计闸祇十有八座金门仅六七十丈而入海之程皆约三百余里邮南五坝迤逦二百四十丈每开一坝皆悬流下注高八九尺不等以二百余丈悬流下注之水归六七十丈曲折萦回之港又三百余里始得入海焉能宣泄得及故每开五坝下河州县必被淹漫即泰州分司所辖北五场灶亦皆被淹厯年有民赈即有灶赈有案可稽又兴邑地当水冲形同釜底受西南北三面之水惟东路可通于海而地形偏高范堤内束又成侧釜之势故兴之受患尢深且久承前河宪张大辟归江之路故湖水消泄较速非入江各口全开之后不轻开下坝其入江之口惟人字河宣泄最灵向为蓄水运盐故于人字河口芒稻河之上游筑坝即所谓拦江坝者特以蓄资挽运此盐务持议不肯轻开也其不欲开者恐坝开溜急牵挽稍艰后经河道宪议以邵伯镇运河三沟闸之志水至一丈二尺于盐运无碍即先开拦江坝上年水已过志前府刘勘明即据应开情形通禀将坝土尽行启除水无壅遏是以下坝未开下河州县皆获全收今年水亦过志府县禀请开放未奉回批坝自溃决下河又获全收上年九月并据刘府将开坝之后与盐运毫无窒碍等情禀请署督宪林照三沟闸水一丈二尺之定志即由府会扬粮厅启放不必禀请多延时日致有壅遏如禀批准饬司道议覆督宪陶回任批详俟扬州运河长生庵志水至五尺以外府厅飞速禀请办理不得径开等因在案夫既经禀请往返必十数日乃奉批回此十数日闲来源既旺去委未通已成迫不及待之势如是则必开放下坝下坝一开不但七州县被淹蠲免赈恤即五场灶亦被淹待赈于民大害于灶亦殊有损至盐务所虑坝开溜急难以牵挽者不过十四里牵挽之费所用几何以较下河七邑赋课田庐亿百万生灵之命其轻重利害当有彰明较著者也职分居司牧不忍膜视谨据众情上禀并将五坝一开下河各邑被水淹没情形绘图呈览仰恳大人俯察全局轸念民依嗣后请如前署督林议高堰坝开邵伯运河水至三沟闸一丈二尺之志即由府会厅将人字河拦江坝启放不必禀请辗转多延时日致有满涨淹没之虞似于国赋民命两有裨益如蒙允准即饬遵泐石以垂永远愚昧之见合肃禀闻图附后

　　○戒招摇撞骗示 

　　窃维士以廉隅为贵官以清慎为先未有有志砺节而行同市狯者也本县家道奇穷而此志不敢少易诚谓读古人书所学何事傥被不韪之名致抱惭于衾影上无以对父母下更无以式子孙自问何人能无恐惧况暮夜之金尢为儒者所耻向曾有句云黑夜有人持白镪苍生何处见青天本志如是是以十年辉县两袖清风不特可以自信亦有可以共信者矣春仲来宰兴阳秉性硁硁断不改弦易辙所以听讼必于大堂者俾庶民共见以示清白之心也犹恐与民未信或容有不尽知我者此中招摇撞骗正未可必将使我受贪墨之名其害犹浅而使乡民受身家之累其害实深为此徧谕绅民人等知悉嗣后雀角鼠牙可已则已不得已而兴讼是非曲直本县自定权衡旁人不得稍参末议虽不敢谓事事确当人心而胸中独断决无所谓先入之言尔等幸勿误听人欺私相属托致堕招摇撞骗之谋庶几我得其名而尔辈亦不至于受累不然者力求争胜辄至倾家自干与受之科徒饱奸宄之腹身罹重典悔不可追本县执法綦严断不从宽姑贷也凛之慎之毋违特示 

　　○劝民十约 

　　为剀切晓谕事本县莅任两月查勘士习备询民风虽未周知已悉大概力学敦行之士原不乏人其或迫于贫穷激于意气遂致不检于制行者有之耕田凿井之民本乐无事其或轻于听信狃于乡愚遂致不安于本分者有之尔民各具心知岂其终无悔悟本县任父母斯人之责喜与更始更乐与图终此后或读或耕羣相劝勉日趋于善勿罹于刑是所切属其有急不能变之顽民与其刑之使惧不若导之使明因儗为十约徧谕乡城有则改之无则加勉本县有厚望焉爰列其目于左 
　　一男子不可无执业窃见街坊镇市游手者多此辈或习技艺或力佣工年华方盛何事不可为乃甘心浪荡好懒偷闲琐尾流离势不至为盗贼不止今与尔辈约急须改换心肠力图操作切勿自甘颓惰即于匪彝本县不忍不教而诛然教而不改亦不能为尔等宽也凛之 
　　一女子之职以纺织为功本县莅辉十载劝桑课织至今聿有成效乃观此地妇女半属宽闲或倚门观望徒耗日时或甘学清音竟忘羞耻此等恶习风化攸关今与尔辈约与其偷安而受困何如急起而有功务望每一家各置纺车一架五家共置机床一架买棉纺线费少功多不惟可以束身心亦可以资衣食尚何至于朝夕不给甚至流为娼妓而不悔也此事责成乡保挨门劝谕如谓本邑向不习此本县现已捐廉设局置办车床延聘女师为之教习除另示晓谕外尔等万勿观望再蹈前习为勖 
　　一狱讼之兴原为必不得已之事男子为之尚滋拖累乃更使妇女出头相习成风恬不知耻其意恃妇逞刁希图泼赖甚至乡里中偶有口角辄率妇女涂闹以为莫之敢撄此种恶风断不可长今与尔辈约嗣后再有妇人具控而丈夫作抱告者先责丈夫有老妇具控而其子作抱告者先责其子更有聚党成羣口冈嗓 【 兴俗谓吵闹为口冈嗓】 构祸者即按律重惩之本县言出法随决不姑贷着乡保传谕各村镇知之毋忽 
　　一纠众滋事例载三人以上者即从重治罪法最严也乃此地动辄聚众或私行抄抢或恃众把持此风一长何所不至即如乡镇中恶棍殃民尽可令乡保出首送官惩治或身受者自行具控本县以除暴安良为务果有劣迹断不姑容亦断不因人众而迁就也今与尔辈约嗣后如再有聚众抄毁及联名呈控挟制官长者定当按律究办不贷 
　　一此地乡民皆聚族而居多至数百家少亦数十家不等如果协力同心守望相助何至盗贼之来如入无人之境也今与尔辈约既居同里必敦亲睦每十家为一联联有长择其精悍者为之五十家为一团团有总择其持重老成者主之出具耰锄入资捍卫偶有贼至羣起而捉之将见水环四面人集四围贼何逃之有否则祗知为己虽害及邻家而不顾安知其不遂及于己也至被贼而后鸣官迟之又久贼已远扬纵或获案而失物已不能瓦全矣此事责成乡保各就庄村公议务须实力奉行切莫空言了事
