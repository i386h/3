　　◆六部 

　　督臣卢兴祖疏曰法令烦多则内外衙门在大官则惟例是遵每多驳查之烦在有司惟功令是畏有不遑救过之虑岂能计及民生之休戚而施抚字之仁且因丛迫之际以致左右作奸则民生愈受其困夫滋弊缘于多法则救弊惟在减法有不辨而自明者矣今将六部见行事例令其逐件讲求如书云刑故无小宥过无大止将贪暴奸私明知故犯者按例处治其余因功获过之事概免琐议减之又减至于必不可减而始存其大且要者是则法令简明大官易于综核有司得以展舒始能共识抚字之术以施膏泽于民 
　　台臣田六善疏曰近日吏部用人行政或有偶出意见或有实系过误得幸奉旨率皆循之为例例既无所不有弊故无所不为可重可轻可上可下皇上不得而诘之吏部曰有例在也科道不得而紏之吏部曰有例在也抑知其意欲如此则有此例意欲如彼则有彼例哉且头绪纷纭册挡充栋司官有时而忽堂官何由而知日见处分究竟弊不可除请将从前诸例何件系请旨定者何件经部院科道条陈准行者汇集一册其余吏部日行之事不得槩引为例堂官一览可悉则弊端不禁自清

　　△考课 

　　考课　考课之法原系六年京察三年大计五年军政康熙元年七月因京察大计多有营求徇庇被处之官纵有屈抑不准申辨无罪被诬者甚多奉上谕停止内外大小官员除议处罚俸实厯俸三年已满者考满分别去留以昭劝惩应列五等看语办事一等称职者加一级办事二等称职者纪录一次办事平常者仍留原任办事不及者降一级调用不称职者革衙门之职遇缺升转照考语次序一等者先用二年四月会覆科臣薛奋生条议以部院各官未及三年升转者多不与考满又增定一年考核三年四月部覆台臣张冲翼疏限定一二等员数不许额外冒滥奉旨以一二等者甚多遂令部院督抚保奏事发一并治罪十一月又奉上谕于四年内考起到任不及三月者不考以后三年一次汇考一年考核停止在内部院官员仍照前例每人一本一齐具题在外一省文武官员一等者一疏二等者一疏平常者一疏办事不及者一疏不称职者一疏每省各汇齐具题其办过事件功过亦造册与题疏一同移送部院仍照例覆考具题四年正月台臣季振宜条陈以部院大臣上疏自陈不过铺张履厯表里羊酒旋已盈门是以一篇之虚文而博朝廷之实惠至于司属各官全凭堂上官斟酌等第堂官属官朝夕同事时时接见一旦尽破情面皆出至公难乎易乎况一等二等即可躐等升转人心既不淳朴仕进复虑艰难欲其息奔竞安义命岂可得耶今年正月二十日始迄四月终止此六七十日皆自陈考满之日也一人一本当以数千计臣思朝廷一日万几六部覆奏殷烦倘亟亟皇皇日事考满推敲等第断难草率满汉章奏何等纷纭势必诸务停阁臣恐丛脞之虞反从此生而有用之精神徒耗于纸笔矣又加县官考满一由厅二由府三由分守巡道四由按察五由布政六由督抚是一县官考满所厯要紧衙门凡有六处此六处者欲尽皆洁已奉公不受贿赂不听情面一凭公道清议品骘优劣能乎不能乎既不能洁己奉公层层剥削极于县官而止县官岂皆温饱素封之家纵温饱素封又何肯捐己家赀奉媚上官势必横加征派侵欺钱粮从知县之考满递而数之凡府厅守巡道布按之考满求其出于公道清议也难矣揆厥由来总因考一等二等者即得越俸升转故其奔竞钻刺如夜蛾之投火若悃愊无华稍以百姓为心稍以廉耻为事则曲注下考仰屋长嗟又绿旗官兵如单丁一人每月所恃者三斗之米一二两之饷即一家数口每月所恃者亦三斗之米一二两之饷全藉领兵官爱养调护按数给发以救其嗷嗷待哺之急今副将参游以及守备等官类皆考满使总督提督一尘不染夤缘打点无从而入上下优劣等第尽凭公道斯亦可矣若总督提督苟怀不肖借端需索副将以下各官复图一等二等以规功名快捷方式武职无钱粮及可侵欺百姓之可鱼肉势必扣克此穷兵三斗之米一二两之饷倘起意外之盗贼有需征剿而精神膂力消于饥寒盔甲刀枪罄于典鬻贻悞封疆正不能无杞人之忧矣伏乞立停考满之法申饬羣工循名责实安心办事厯俸深者自有应得之升转厯俸浅者可息躁进之妄想省事清心莫大于此矣随经会议停止考满复行京察奉旨自康熙四年算起从今以后六年将内外官员俱行考察五年正月以考期太远恐为积弛之患将考察年分改于康熙元年起仍为六年于康熙七年考察六年二月又奉上谕京察大计及军政俱应于今岁举行议照旧例满汉官员有公差丁忧养病侍亲给假裁缺候补降调未补者俱应见任衙门注考其升迁降调已补官员在内离任半年者于新任衙门考察离任不及半年者原任衙门考察在外离任一年已上者不必原任注考应于新任注考不及一年者仍应旧任注考俱照八法处分年老有疾者致仕贪酷者革职提问罢软无为及素行不谨者俱革职浮躁浅露才力不及者降级调外用外官卓异优迁赏服各衙门注考如有徇情庇护者部院科道据实题参处分之官有妄行造言奉告者重处武官龙钟衰迈者照老疾例勒令休致轻穉率妄者照浮躁不及例降调庸懦无能者照罢软例革职赃私狼籍者照贪例革职追拟酷而有款者革职提问恣睢虐下者照酷例罢职不叙至败伦伤化行止有亏者照不谨例革职自此考核年分算起六年一次考察京官三年一次大计外官五年一次考选军政则又变考满之法而各复旧例矣

　　钱粮总归户部　康熙二年四月户部会覆科臣吴国龙条议将一应钱粮俱归户部总理责令各抚将府州县卫所丁地各若干应征本折钱粮各若干及不在丁地之内杂项租税各若干分别起存将赋役全书订后续增添丁地钱粮数目一并算成总数每省各造一本简明赋役之书明白刊送户部较对前订赋役全书以作查核之据诸凡内外催提征解钱粮俱称为丁地钱粮名色至于易知由单内止开丁地银米各若干颁发民间其余别项名色一槩停止外省兵饷户部每年正月全拨余剩全解户部各部寺应用钱粮年前将银两物料等数目算明具题户部照钱粮解到陆续给发仍令该部寺年终将收放细数查核明确题报
