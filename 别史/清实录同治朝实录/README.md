穆宗继天开运受中居正保大定功圣智诚孝信敏恭宽毅皇帝讳载淳文宗协天翊运执中垂谟懋德振武圣孝渊恭端仁宽敏庄俭显皇帝长子也。母慈禧端佑康颐昭豫庄诚皇太后那拉氏。原任安徽徽甯池太广道追封承恩公惠徵之女。初封懿嫔。寻晋封懿妃。懿贵妃。上登极。尊为皇太后。禀粹通门。宣勤中掖。事文宗显皇帝。柔嘉维则。淑慎其仪。毓圣延厘。以咸丰六年丙辰三月二十三日未时。诞上于储秀宫。母后皇后今慈安端裕康庆昭和庄敬皇太后。因主器之占已协。祈禖之愿适符。顾复恩勤。同一致焉。上之生也。文宗以天恩祖佑。喜溢丹毫。爰有庶慰在天六年望。更欣率土万斯人之咏。由是常雩成礼。则有云鸿恩幼子千春庆。湛静升香。则又云喜见麟孙慰母思。而且喜音楣额。羲画焜煌。周□山戊不□庆辰。尧章炳焕。固不待诞膺洪祚。早知世德之克承已。上聪明天亶。孝敬性成。恭读文宗御制诗有云。绕膝堂前助笑颜。则知愉婉之容。达于斖斖。岐嶷之德。本乎安安。此帝业之权舆。洵天家之乐事焉。

○癸卯。文宗显皇帝宾天。时文宗巡幸热河。驻跸避暑山庄。上侍母后皇后圣母从焉。文宗圣躬不豫。上侍疾烟波致爽殿。吁天虔祷。问视维谨。壬寅。文宗疾大渐。召御前大臣载垣、端华、景寿、肃顺、军机大臣穆荫、匡源、杜翰、焦祐瀛、承写朱谕。立皇长子载淳为皇太子。越翼日寅刻。文宗升遐。上稽颡大恸。擗踊无算。扈从诸臣。遵奉遗旨。请上即正尊位。上号恸仆地。良久方起。辰刻。上恭视小殓毕。奉大行皇帝灵驾。奉安于烟波致爽殿东闲。上翦发成服。亲王以下文武大臣官员俱成服。各按位次。齐集举哀。上哀恸深至。自朝至昃。哭不停声。竟日水浆不入口。群臣伏地环跪。恳上节哀。上悲恸不能自已。左右皆弗忍仰视。　　

○谕内阁、朕受皇考大行皇帝鞠育顾复深恩。昊天罔极。圣寿甫逾三旬。朕宫庭侍奉。正幸爱日方长。期颐可卜。上年夏闲。偶患痰嗽。旋即调摄就痊。秋闲巡幸滦阳。圣体康强犹昔。乃因各省寇氛未靖。宵旰焦劳。至本年春闲。风寒感发旧疾。六月闲复患暑泻。以致元气渐亏。本月十六日子刻。力疾召见载垣、端华、景寿、肃顺、穆荫、匡源、杜翰、售祐瀛、特命承写朱谕。立朕为皇太子。朕痛哭受命。哀迫战兢。方冀慈躬转危为安。常承恩诲。讵意亲奉顾命后。病势增剧。遂至大渐。十七日寅刻。龙驭上宾。抢地呼天。攀号莫及。敬思皇考御宇十有一年。惕厉忧勤。万几鲜暇。无日不以敬天法祖勤政爱民为急务。蠲缓赋税。＜??闲＞拔人材。国计民生。时廑圣虑。凡有血气者。其悲哀感恋。罔不出于至诚。朕之泣血椎心。尚忍言乎。惟念付托至重。责在藐躬。尚赖内外文武大小臣工。共矢公忠。弼予郅治。其带兵大员。尤须严申军律迅殄贼氛。各直省督抚。亦应抚辑斯民。以仰慰我皇考在天之灵。朕实有厚望焉。至丧服之制。钦奉皇考遗诏。令依旧制二十七日而除。朕心实所不忍。仍当恪遵古制。敬行三年之丧。庶几稍尽孺慕之诚。至于郊庙祭祀大典。自不应因大丧而稍略其礼。其应如何遣官恭代、及亲诣行礼之处。著各该衙门查照向例。集议以闻。其天下臣民应持服制。仍照定例行。将此通谕中外知之。

○甲辰。上恭视大行皇帝升吉轝。于烟波致爽殿阶下跪送。即趋诣澹泊敬诚殿后檐下。恭迎大行皇帝吉轝。巳刻。敬视大殓毕。奉安梓宫于澹泊敬诚正殿。行殓奠礼。上以烟波致爽殿西所为倚庐。席地寝苫。每日诣几筵前奠献三次。自是日至启奠以前。有常礼。哭必尽哀。　　

○申刻。诣烟波致爽殿东所问母后皇后圣母安。　　

○谕内阁、朕缵承大统。母后皇后应尊为皇太后。圣母应尊为皇太后。所有应行典礼。该衙门敬谨查例具奏。　　

○又谕、道光三十年。皇祖宣宗成皇帝大事。皇考曾遵嘉庆二十五年成例。停止各省将军督抚提镇藩臬及盐关织造等来京叩谒梓宫。今皇考大行皇帝在山庄升遐。恭奉梓宫回京。除热河都统本在热河。直隶总督已谕令前来热河叩谒梓宫外。其余直隶各大员俱不必前来。各路统兵大臣、各省将军督抚副都统提镇城守尉、并盛京侍郎、奉天府尹、西北两路将军大臣、及学政织造关差等、均不必奏请前来。致旷职守。各该大臣将军督抚提镇等受皇考深恩。惟当竭诚尽职。以期无负委任。不在仪节虚文也。将此通谕知之。　　

○又谕、皇考大行皇帝龙驭上宾。朕应恭奉梓宫回京。进宫后暂安观德殿。所有一切典礼。著在京恭理丧仪王大臣先期敬谨务备。其沿途经过地方。所有芦殿地盘及桥深道路。古北口以内。著直隶总督恭办。古北口以外。著直隶总督会同热河都统恭办。著即督饬各该地方官妥速办理。务须一律宽平稳回。不得草率迟误。其承办各员。不准藉名科派。扰累闾阎。以副皇考惠爱群黎至意。　　

○又谕、道光二十六年三月。皇祖特降谕旨。以二名不偏讳。将来继体承绪者。上一字仍旧毋庸改避。亦毋庸缺笔。其下一字应如何缺笔之处。临时酌定。以是著为令典等因。钦此。今朕敬遵成宪。将御名上一字仍旧书写。毋庸改避。下一字毋庸缺笔。凡臣工奏章内遇有此字。著用淳字改避。其奉旨以前所刻书籍。俱毋庸议。　　

○乙巳。上诣四知书屋问母后皇太后圣母皇太后安。　　

○谕内阁、自古神圣之君。继天立极。必有隆称显号。昭示来兹。洵万世不易之典也。钦惟我皇考大行皇帝御宇十有一年。寅畏敬恭。朝乾夕惕。自承皇祖付畀以来。无日不以敬天法祖勤政爱民为念。郊坛大祀。至诚昭格。展敬陵庙。尽制尽伦。即位之初。广开言路。褒答直臣。手谕训励臣工。痛戒因循。杜贡献以昭节俭。阐性理以崇圣学。勤求郅治。整肃纪纲。孜孜惟恐不及。洎乎边隅不靖。小丑跳梁。我皇考命将出师。任贤勿贰。丰县连镇之役。叠次翦除巨憝。捷书入告。弥切战兢。未尝稍自矜伐。每当军书旁午。日昃不遑。被览章奏。召对臣工。亲裁庶政。不敢自暇自逸。盖十一年如一日。偶值水旱偏灾。蠲除赈贷。惟恐后时。河工军务。不惜屡发帑金。开海运以实仓储。广中额以振文教。敦睦宗支。钦恤庶狱。赏功罚罪。莫不一秉至公。至于聪明天亶。圣学渊深。炳焕大文。超越今古。举行团练而武备脩明。整饬戎行而军威克振。愚民悔罪投诚。即宽其既往。予以自新。好生之德。洽于民心。凡此皆以天地之量为量。以尧舜之心为心。中外臣民。所共闻共见者也。予冲人仰承遗命。寅绍丕基。虽至德谦冲。圣不自圣。而永怀佑启之恩。宜备尊亲之典。详考彝章。恭上尊谥庙号。著大学士九卿等详察典礼。敬谨具奏。　　

○又谕、惠亲王系朕之叔祖。其寻常召对、及内廷筵宴等礼节。业经皇考大行皇帝加恩免其叩拜。以示尊崇。惇亲王、恭亲王、醇郡王、钟郡王、孚郡王、均系朕之叔。若于内廷召对。动辄叩拜。与诸臣略无区别。朕心实有不安。嗣后除朝会大典仍遵定例行礼外。其寻常召对、以及三年后内廷宴赉。均无庸叩拜。至宫中祭神。赐食祭肉。应向上叩拜。惟不必于朕前随众行礼。遇有赏赐物件。著具摺谢恩。无庸当面叩首。以示朕敬长亲亲之至意。
