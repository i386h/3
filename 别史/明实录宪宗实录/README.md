　大明宪宗纯皇帝实录卷之一
　　宪宗继天凝道诚明仁敬崇文肃武宏德圣孝纯皇帝讳见深  英宗睿皇帝之长子  母今  圣慈仁寿太皇太皇太后于丁卯十一月二日生  上于宫闱生时红光满室是岁天下大稔海宇清晏至十四年己巳虏寇也先大举犯边  英宗睿皇帝率师亲征抵大同而还至土木为虏所遮  车驾北狩京师戒严中外汹汹朝不保暮时  上甫三岁  郕王监国公侯驸马伯皇亲及文武群臣请命于  圣烈慈寿皇太后册上为皇太子诏告天下人心始安既而闻虏复至在廷文武群臣请立监国为皇帝以纾国难  圣烈慈寿皇太后复勉从所请立焉是为  景泰帝遂尊  英宗睿皇帝为太上皇帝逾年  太上皇帝回銮  南宫  上  圣烈慈寿皇太后居清宁宫年虽幼已岐嶷如成人视瞻非常不轻言笑稍长知读书天资頴悟  圣烈慈寿皇太后恒今人为直说大义輙通解不忘景泰三年贼臣广西都指挥黄＜王厷＞犯罪当死自广西狱中遣其子上书劝  景泰帝立其子见济为皇太子遂降封  上为沂王制下之日人心皆痛愤不平未几见济薨朝臣有请复  上为皇太子者輙被棰挞几死天顺丁丑  英宗睿皇帝为众所拥戴复辟废  景泰帝仍为郕王复立  上为皇太子  上初名见深至是更名见濡诏书失写其故颁行天下人皆惊相问曰此非向所立太子乎何名之不同也盖  上为天下人心归向久唉戊寅出阁读书命内阁臣李贤彭时吕原及翰林诸儒臣侍讲读于文华殿群臣见  上相表奇异玉色和粹无不惊服  上广额豊硕方面大耳目睛如漆黑光彩射左右侍者皆莫敢
　　仰视  英宗睿皇帝欲  上进学赐敕讲读官二帝三王之心法与凡修齐治平之要道务令以见诸行事于他日  上读书音向洪亮不数遍即作字运笔有法尤便习骑射至于侍  膳问安孝敬备至宗睿皇帝特钟爱焉尝谓  慈懿皇太后及今  圣慈仁寿皇太后曰此太平天子也福德非吾所及主是年十□宗睿皇帝不豫召  上至榻前属以后事甚悉受命已而  英宗宾天  上哀恸之甚令自成典礼而行中外臣民益忻忻仰戴焉天顺八年正月甲寅朔

　　○文武百官军民耆老人等奉笺于皇太子劝进其词曰自古圣帝明王之有天下也继体垂统必传之子以为天下臣民主斯乃国家之常典古今之通谊诚千万世之所共由而不可易者也钦惟  大行皇帝聪明睿知文武圣神大烈耿光有二帝三□之盛治至仁厚泽得四方万国之惟心正东宫以隆国本发宝册以建  储君诚以  皇太子殿下位居冢嗣性本仁明禀天锡之英资广日新之圣学足以继神灵之统承  祖宗之业天下国家万万年入平之本实所系焉伏自  殿下正储位以来施仁赞政行允协于  圣心视膳问安心愈隆于孝德代祀而  宗庙  社稷之来享临民而华夏蛮貊之归心兹者  大行皇帝宾天遗命  殿下早正大位中外臣民诚切瞻仰伏望  殿下仰遵  先志少抑哀情思远大之弘规顾缵承之大计早登宝位永固皇图臣等俯伏阙庭合词劝进仰六龙之临御如日方升况万□之延长与天同久臣等无任瞻仰激切屏营之至  皇太子令谕答曰卿等所言足见忧国之诚予抱痛方殷而继统之事奚忍闻之所请不允
　　○壬申文武百官军民耆老人等再奉笺劝进其词曰治世安民大统必归于主器承祧继体万方允属于元良今昔所同邦家攸系洪惟我朝  列圣创业垂统致治保邦德化同于尧舜功烈并于禹汤逮我  大行皇帝法  祖宪  天覃恩育物极礼乐文章之制作遂飞潜动植之生成敬惟皇太子殿下文武英明宽仁孝友明睿察于庶□温恭协于重华位久止于东宫礼式明于大分以贤以长宜民宜人乃承遗诏之颁俾正南面之位臣等合词劝进未荷俞允切以天下不可以无主神器不可以暂虗虽哀思之切未忍于遽承而圣孝之大必先于继述伏望体  先皇付托之重徇臣下瞻仰之情即赐德音早登宸极以为  天地神明之主以慰臣民亿兆之心居正体元嗣徽猷于  列圣弘仁茂德隆盛治于万年迓龙驭于九重效嵩呼于三祝臣等无任虔恳俟命之至  皇太子令谕答曰卿等虑  宗社之重言益谆切披览之余俞增哀恸安忍遽即大位所请不允
　　○癸酉文武百官军民耆老人等三奉笺劝进其词曰重明丽正惟统绪之常经景命维新用缵承于大位天人协应远迩具瞻恭惟  皇太子殿下孝友英明刚健中正久居储副主鬯春宫既大分之已明致臣民之归向至仁茂德简在  帝心嗣位承祧特存遗命  殿下位居储贰义合继承而深切哀情未登宝位臣等劝进表已再上皆未赐俞允群臣惶惶莫知所措夫为天下国家者以善继述为重以广德教为先以安  宗社为急以固基业为悠久重计伏望  殿下少回睿意早正  天位三灵百神有所亲赖四方万国有所归依此国家大计臣民之至愿也臣等下情无任恳切俟命之至  皇太子令谕答曰卿等惓惓陈情至再至三为国家弘远之虑至矣大宝之位诚难久虚况予已承遗命无所逊避勉从所请  皇太子令谕礼部曰  大行皇帝上宾遗命眇躬俾承大统顾在哀疚奚忍遽承而宗亲文武群臣军民耆老人等奉表劝进至于再三永惟  宗社生民之托至重不敢终拒尔礼部择日祗令大保会昌侯孙继宗广宁侯刘安怀宁侯孙镗告  天地  宗庙  社稷予即皇帝位其具仪以闻
