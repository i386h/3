　　提要
　　【臣】等谨案尚史一百七卷
　　国朝李锴撰锴字铁君镶白旗汉军卷首自署曰襄平考襄平为汉辽东郡治今为
　　盛京辽阳州地盖其祖籍也康熙中邹平马骕作绎史采摭百家杂説上起鸿荒下迄秦代仿袁枢纪事本末之体各立标题以类编次凢所徴引悉录原文虽若不相属而实有端绪锴是编以骕书为藁本而离析其文为之翦裁连络改为纪传之体作世系图一卷本纪六卷世家十五卷列传五十八卷系六卷表六巻志十四卷末附序传一卷仍于毎叚下各注所出书名其遗文琐事不入正文者则以类附注于句下盖体例凖诸史记而排纂之法则仿路史而小变之自序谓始事于雍正庚戌卒业于乾隆乙丑阅十六载而后就其用力颇勤考古来渔猎百家勒为一史实始于司马迁今观史记诸篇其出迁自撰者率经纬分明疎密得当操纵变化惟意所如而其杂采诸书以成文者非唯事迹异同时相抵牾亦往往防窜补缀不能隠斧凿之痕知镕铸众説之难也此书一用旧文翦裁排比使事迹聫属语意贯通体如诗家之集句于厯代史家时为创格较镕铸众説为尤难虽运掉或不自如组织或不尽密亦可云有条不紊矣至于晋逸民传中列杜蒉狼瞫鉏麑提弥明灵輙逆臣传中列赵穿而不列赵盾乱臣传中列郤芮吕饴甥嬖臣传中列头湏鲁列女传中列施氏妇予夺多所未允又诸国公子皆别立传而鲁宋蔡曹莒邾六国则杂列诸臣中叛臣传中如巫狐庸叛楚入吴吴楚两见公山不狃叛鲁入吴吴鲁两见已为重出而屈巫见于楚不见于晋苗贲皇见于晋不见于楚又复自乱其例如斯之类不一而足亦未能一一精核固不必为之曲讳焉乾隆五十四年正月恭校上

　　世纪曰少皞帝名挚字青阳国语曰青阳黄帝之子是少皞为黄帝之子矣史记曰黄帝生昌意昌意生髙阳是颛顼为黄帝之孙矣又曰帝喾父曰蟜极蟜极父曰嚣嚣父曰黄帝是帝喾为黄帝之曽孙矣世本曰帝喾元妃姜嫄生后稷次妃简狄生契次妃庆都生尧次妃常仪生挚是尧与稷契皆为帝喾之子黄帝之孙矣史记曰禹之父曰鲧鲧之父曰颛顼颛顼之父曰昌意昌意之父曰黄帝是禹亦黄帝之孙尧之同髙祖兄弟矣又曰重华父曰瞽叟瞽叟父曰桥牛桥牛父曰句望句望父曰敬康敬康父曰穷蝉穷蝉父曰颛顼颛顼父曰昌意以至舜七世是舜为黄帝之八世孙矣何舜之世次独促于稷契及禹也史记又曰契传十二世而成汤立稷传十五世而文王立夫契至汤十二世汤又传二十九王而后亡而稷裁十五世遂至文王何契之世次又独促于稷也大抵世纪诸书漏阙殊多而史记缪戾未深考也吕梁碑以舜祖虞幕而不祖黄帝或曰颛顼亦不当祖黄帝予窃以青阳嚣昌意为三宗而五帝三王之世由是生焉其原之同出黄帝与不则俟博闻之君子　列后图如左













　　汉书曰少皞帝考徳曰清清者黄帝之子清阳也是其子孙名挚立号曰金天氏是少皞出自清阳而非黄帝之子也国语曰少皞之衰九黎乱徳颛顼受之所谓少皞者指其后世而言少皞生而有圣徳岂及其身而遽衰也传曰少皞有裔子曰昧生允格台骀允格台骀能业其官帝颛顼嘉之封诸汾川是颛顼出自昌意而非昌意之子黄帝之孙也世纪曰帝喾年十五而佐颛顼颛顼且不得父昌意而祖黄帝帝喾亦不得祖嚣而为黄帝之曽孙也春秋命厯序曰颛顼传十世是帝喾且不得与颛顼亲承接也传曰髙阳氏有才子八人世谓之八恺髙辛氏有才子八人世谓之八元此十六族者世济其美不陨其名至于尧尧未能举所谓世济者累世之谓也夫八元既出自髙辛而尧为帝喾之子是帝尧兄弟八元又何世美之不陨而举且未能也以髙辛之圣舍稷之适与尧之徳而立挚先儒亦疑之矣传又曰八元者髙辛氏之裔也其文明甚是尧出自嚣而非帝喾之子也吕梁碑以舜祖虞幕左传史赵之言曰自幕至于瞽叟无违命是己国语又曰幕能帅颛顼者也传又曰陈颛顼之族也是舜出自颛顼而幕又非即颛顼之子也汉书曰颛顼五代而生鲧是禹亦不得祖颛顼而为黄帝之孙而五代之説犹有未得也谯周曰契生尧代舜始举之必非喾子以其父微故不著名又曰弃帝喾之胄其父亦不着是谯知稷契非帝喾之子而不知帝尧亦非帝喾之子也国语曰自后稷之始基靖民十五王而文始平之又曰基徳十五而始平所谓十五王者论其徳非论其世而史乃以十五世足之谬矣吕梁碑以后稷生漦蠒漦蠒生叔均叔均又数世而生不窋世本公非以下复有辟方侯牟云都诸盭是不窋非后稷之子而下距武王且十七世也春秋命厯序曰少皞传八世颛顼传十世帝喾传十世防纬之言殊不足信要其世数虽不可知然自帝尧以上大抵如三代之迭兴非唐虞之授受也

　　尚史卷二　　　　　　　　本纪一
　　镶白旗汉军李锴撰
　　轩辕本纪
　　世传三皇多以伏羲神农其一或曰燧人曰祝融曰女娲汉孔氏书序曰伏羲神农黄帝之书谓之三坟少皥颛顼髙辛唐虞之书谓之五典故黄帝以上为三皇少皥以下为五帝虽不知其所本然其序五帝与家语宰我问及春秋内外传大戴礼畧合至宋五峯胡氏直断以易大传以伏羲神农黄帝尧舜为五帝然亦非系辞本防不足以为定论今从书序纪五帝又上溯黄帝以纪其所自出而畧羲农惧邃逺无征云
　　黄帝
　　黄帝者少典之子
　　史记索隠少典诸侯国号
　　姬姓也
　　史记姓公孙
　　母曰附寳感大电绕北斗枢星照郊野而孕二十四月生黄帝于寿邱长于姬水受国于有熊居轩辕之邱因以为名又以为号【帝王世纪】
　　左传号帝鸿氏白虎通黄帝有天下号曰自然拾遗记轩辕出自有熊之国母曰昊枢以戊巳之日生故以土徳王时有黄星之祥五行书以戊子日生
　　生而神灵弱而能言幼而徇齐长而敦敏成而聪眀白虎通黄帝龙顔得天匡阳上法中宿取象文昌鬻子黄帝十嵗知神农之非而改其政　今本无
　　轩辕之时神农氏世衰诸侯相侵伐暴虐百姓而神农氏弗能征于是黄帝乃习用干戈以征不享诸侯咸来宾从而蚩尤最为暴莫能伐炎帝欲侵陵诸侯诸侯咸归黄帝黄帝乃修徳振兵治五气艺五种抚万民度四方教熊罴貔貅防虎以与炎帝战于阪泉之野
　　帝王世记黄帝乃扰驯猛兽与神农氏战于阪泉之野史记索隠猛兽可以教战周礼有服不氏掌教扰猛兽即古服牛乗马之类正义言教士卒习战以猛兽之名名之用威敌也　此説近是
　　三战然后得志
　　国语少典取有峤氏女而生黄帝炎帝成而异徳用师以相济也韦昭注言生者谓二帝本所生出也黄帝灭炎帝灭其子孙耳明非神农也贾谊新书炎帝者黄帝同母异父兄弟各有天下之半黄帝行道而炎帝不听故战于涿鹿之野血流漂杵归藏黄帝与炎帝争鬬涿鹿之野将战筮于巫咸曰果哉而有咎
　　蚩尤作乱不用帝命于是黄帝乃征师诸侯与蚩尤战于涿鹿之野遂禽杀蚩尤
　　崔豹古今注黄帝与蚩尤战于涿鹿之野蚩尤作大雾兵士皆迷于是作指南车以示四方遂擒蚩尤故后常建焉又涿鹿之战常有五色云气金枝玉叶止于帝上有花葩之象故作华葢盐铁论轩辕战涿鹿杀两曎蚩尤而为帝
　　而诸侯咸尊黄帝为天子代神农氏有土徳之瑞故号黄帝
　　吕氏春秋黄帝之时天先见大螾大蝼黄帝曰土气胜土气胜故其色尚黄其事则土螾土精大五六围长十余丈
　　天下有不顺者黄帝从而征之平者去之
　　万机论黄帝之初养性爱民不好战伐而四帝各以方色称号交共谋之邉城日警介胄不释黄帝叹曰君危于上民不安于下主失于国其臣再嫁厥疾之由非养冦邪今取民萌之上而四道亢衡逓震予师于是遂师营垒以灭四帝向令黄帝若不龙骧虎变而与俗同道则其民臣亦嫁于四帝矣
　　披山通道未尝宁居东至于海登九山及岱宗西至于空峒登鸡头南至于江登熊湘北逐荤粥合符釜山而邑于涿鹿之阿迁徙往来无常处以师兵为营卫官名皆以云命为云师
　　杜预左传注黄帝受命有云瑞故百官师长皆以云名为号缙云氏其一官也
　　置左右大监于万国万国和获寳鼎
　　汉书黄帝作寳鼎三象天地人
　　迎日推防
　　按日月朔望未来而逆推之故曰迎日
　　举风后力牧常先大鸿以治民
　　尸子子贡问于孔子曰古者黄帝四面信乎孔子曰黄帝取合已者四人使治四方不谋而亲不约而成功此之谓四面也
　　顺天地之纪幽明之占死生之説存亡之难时播百谷草木淳化鸟兽虫蛾旁罗日月星辰水波土石金玉劳勤心力耳目节用水火材物【史记】
　　黄帝内传黄帝升为天子地献草木述耕种之利因之以广耕种拾遗记轩辕考定厯纪吹玉律正璇衡谯周古史考黄帝作釜甑始蒸谷为饭烹谷为粥
　　天下人民野居穴处未有室屋则与鸟兽同域于是黄帝乃伐木构材筑作宫室上栋下宇以避风雨【贾谊新书】始作合宫祀上帝布政教【国语】
　　白虎通黄帝作宫室以避暑寒此宫室之始也黄帝内传帝既斩蚩尤因立台榭
　　始作舟车以济不通旁行天下方制万里画埜分州得百里之国万区【汉书】
　　古史考黄帝作车引重致逺少昊时畧加牛禹时奚仲加马帝王世纪自神农以上有大九州柱州迎州神州之等黄帝以来徳不及逺惟于神州内分为九州黄帝受命风后受图割地布九州置十二国拾遗记置四史以主图籍使九行之士以统万国九行者孝慈文信言忠恭勇义以观天地以祠万灵
　　始造书契服冕垂衣【拾遗记】
　　铸十二钟以和五音以施英韶以仲春之月日在奎始奏之命之曰咸池【吕氏春秋】
　　黄帝崩塟桥山【史记】
　　帝王世纪黄帝在位百年而崩年百一十嵗或传以为仙或言寿三百嵗皇览黄帝冡在上郡桥山地理志在上郡阳周县桥山南括地志在宁州罗川县东八十里子午山
