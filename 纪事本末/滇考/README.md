●滇考序

学而不眀于治乱之故无贵乎其学也仕而不察于治乱之由无贵乎其仕也頋学在端其本源而仕必镜乎往事知往所以治即知今所以致治矣知往所以乱即知今所以弥乱矣滇固昔所称易乱而难治者也我国家劳百万师以取之特留藩旗设行台大臣以理之内而三使司外而分四道十镇一十八郡以交制之其不鄙弃逺人欲与同享太平之盛之意甚厚官此土者頋勿能考人情土俗所宜登斯民于袵席揆诸学古入官之义不有恧乎予小子承乏理金齿三稔矣幸得免覆餗平反之暇思进观前代之事迹而载籍缺然年来往还滇洱间残编断简以次购集大抵乗苦于繁而寡要稗失于怪而不经因不揣固陋举其事之最著者分题彚辑为三十有七篇名之曰滇考自战国迄明其间治乱之数厘然备陈于是知滇非难治也予之以郑纯张翕赛典赤诸人欲无治不可得也非易乱也予之以张虔陀刘深陈典诸人欲无乱不可得也非易乱而难治也予之以陈立杨竦韦仁寿诸人欲无转乱以为治不可得也传曰学者效也又曰仕而优则学此予滇考一书之所为作乎若夫山川之融结人物之生聚仙真佛子之出没则向者乗与稗备之非予所能尽收也予所考者为仕而学者考也又曰在端其本源何也操之以亷洁行之以仁恕无往而不得治焉不必考亦可也旹康熈四年岁次乙巳仲春中浣永昌军民府推官天台冯苏书

楚庄蹻王滇

滇在三代不见于经传按其地之可考者帝颛顼生于若水水经注云若水南经云南郡之遂久县即今之金沙江也禹道黑水至于三危入于南海梁州水入南海者惟澜沧江元史张立道使交址并黑水跨云南以至其国今云龙州有三崇山一名三危澜沧经其麓其地有黑水祠焉逸周书献令曰西南产里百濮以象齿短狗为献产里即今车里西南有百国目为百濮武王东征髳人濮人与焉楚语曰叔熊逃难于濮今姚安楚雄间有濮落蛮定逺名髳州顺寜称古濮地是滇之通中国久矣然山川名号与世移易滇国之称自楚庄蹻始庄蹻者楚庄王之苖裔也当顷襄王时为将从沅水伐夜郎至且兰■〈木豕〉船于岸而步战既灭夜郎以且兰有■〈木豕〉船牂牁处乃改其名为牂牁牂牁地多雨潦俗好巫鬼禁忌寡畜生又无蚕桑故其民最贫蹻引兵至滇池池周回三百里源深广而末更浅狭有似倒流故谓之滇其旁地平敞多出鹦鹉孔雀有盐池田渔之饶金银畜产之富可数千里蹻以兵威略定滇地使部将小卜分兵收滇西诸蛮欲归报会秦击夺楚巴黔中郡道塞不通因还滇筑苴兰城居之以声敎诱服诸夷夷人皆悦共推蹻为君长蹻变服从其俗以王之及秦并天下使常■〈晏欠〉略通五尺道西南诸国颇置吏庄氏仍世王滇至汉武帝时滇王常羌蹻之后也蹻墓在南广县晋时为五苓蛮所发刺史王逊欲诛蛮以此为罪而讨灭之然则蹻之有功于滇而为人所尸祝非一日矣故予纪滇事断自蹻始云

汉通西南夷置郡县

西南夷在蜀郡徼外夜郎国东接交址西有滇国北有卭都国各立君长其人皆椎结耕田有邑聚外有巂昆眀西极同师东北至楪榆地方数千里皆编髪随畜迁徙毋常处毋君长自巂东北又有莋都冉■〈马龙〉白马三国亦有君长汉初皆弃不通巴蜀民或窃出故徼商贾取其莋马僰僮髦牛以此巴蜀殷富武帝建元六年大行王恢击东越因兵威使番阳令唐蒙风指晓南越南越食蒙蜀蒟酱蒙问所从来曰道西北牂牁蒙归至长安问蜀贾人贾人曰蜀出蒟酱多持窃出市夜郎夜郎者临牂牁江江广百余步足以行船南越以财物役属夜郎西至同师然亦不能任使也蒙乃上书曰南越地东西万余里名为外臣寔一州主也今以长沙豫章往水道多絶难行窃闻夜郎所有精兵可得十余万浮船牂牁江出其不意此制越一竒也诚以汉之疆巴蜀之饶通夜郎道为置吏易甚帝许之乃拜蒙为中郎将将三千人食重万余人从巴筰闗入遂见夜郎侯多同蒙厚赐喻以威徳约为置吏使其子为令夜郎旁小邑皆贪汉缯帛以为道险终不能有乃且听蒙约还报以为犍为郡发巴蜀卒数万人治道自僰道诣牂牁江卒多物故有逃亡者用军兴法诛之巴蜀民大惊恐帝因使司马相如责蒙等因谕告巴蜀民以非上意及还而卭莋之君长闻南夷与汉通得赏赐多多欲求内附比南夷武帝问相如相如因亦言西彝卭莋近蜀可置郡帝乃拜相如为中郎将建节及副使王然于等乗传因巴蜀吏币物以赂西彝卭莋冉駹斯榆之君皆请为臣妾除边闗闗益斥西至沫若水南至牂牁为徼通灵闗道桥孙水因置一都尉十余县属蜀当是时蜀四郡通西南夷道戍转相饟数歳士罢饥离湿死者甚众西南夷又数反帝患之使公孙宏往视还对盛毁西南夷无所用帝不听及筑朔方以据河始听宏言罢西夷独置南夷夜郎两县一都尉稍令犍为自保就专力城朔方元狩元年博望侯张骞使大夏归言居大夏时见蜀布卭竹杖使问所从来曰从东南身毒国身毒国有蜀物此去蜀不逺今使大夏从羌中险诚通蜀身毒国道便近无害帝悦令因蜀犍为发间使王然于栢始昌吕越人等十余軰间出西南夷指求身毒国始至滇滇王常羌乃留为求道四歳余皆闲巂昆眀莫能达滇王与汉使者言曰汉孰与我大及夜郎侯亦然以道不通故各自为一州主不知汉广大使者还因盛言滇大国足事亲附武帝注意焉以昆明数阻汉使谋伐之闻其地有滇池于长安西南作池象之以习水战因名昆眀池云元鼎五年南越相吕嘉反汉使驰义侯因巴蜀罪人发南夷兵下牂牁江会楼船伏波二将军击南越且兰君恐逺行旁国掳其老弱乃与众反杀使者及犍为太守眀年南越平驰义侯所发巴蜀罪人八校尉俱未下牂牁即引还遣中郎将郭昌卫广将击且兰诛之及其旁国卭君莋侯遂平南夷为牂牁郡夜郎侯惧遂入朝夜郎侯即前多同也范史称初夜郎有女子浣于遯水有三节大竹流入足间闻其中有号声剖竹得一男归而养之及长有才武自立为夜郎侯以竹为姓既降汉武帝赐以王印后竟杀之夷僚咸以竹王非血气所生甚重之求为立后牂牁太守呉霸以闻汉乃封其三子为侯死配食其父南中志云今夜郎县有竹王三郎祠是也汉既定夜郎冉■〈马龙〉白马皆震恐请臣置吏乃以卭都为越嶲郡莋都为沈黎郡冉■〈马龙〉为汶山郡白马为武都郡使王然于以破越及诛南夷兵威风谕滇王入朝滇王有众数万人其旁东北有劳浸靡莫皆同姓相扶未肯听元封元年郎中司马迁奉使西征巴蜀以南略卭莋昆明还报命二年发巴蜀兵击灭劳浸靡莫以兵临滇滇始首善以故弗诛举国降请置吏入朝于是以为益州郡割牂牁越嶲各数县配之赐滇王王印复长其民其后歳遣使者十余軰出求通大夏仍闭嶲昆明为所杀夺币物六年赦京师亡命遣拔胡将军郭昌将击之无功罢后复遣使竟不得通此汉武帝经理西南夷之大略也通黔为伐南越而八校尉终未下牂牁通滇为指求身毒国而卒闭嶲昆眀得无劳而罔功哉今以地形考之黔水惟盘江最大或言即古牂牁其江由滇阿迷罗雄过广南泗城田州三土司出粤自广南以上皆﨑岖不通舟楫乃欲于此浮船制越蒙之术疎矣嶲即越嶲郡今为建昌卫地丽江有通安州汉初置县曰定莋属越嶲郡至唐称昆眀意即所谓闭嶲昆眀者乎滇会城首邑今亦名昆眀则以汉象滇池凿昆明池于长安故名之非史所谓昆眀也身毒国唐书以为即天竺国去京师九千六百里居葱岭南幅员三万里分东西南北中五天竺东南两天竺俱濒海东与扶南林邑接北距雪山圜抱如璧释元奘常至其地贞观十五年遣使上书太宗使梁懐璥持节往其王尸罗逸多惊问国人曰自古亦有摩诃震旦使者至吾国乎皆曰无有然则其言有蜀布卭竹杖亦谬矣地既不与滇嶲接徒糜烂六郡之民命以从事于无益为可惜也虽然汉武虽好大喜功疲罢中国卒能使荒服边夷千载而下长被声敎其徳及生民亦逺矣其所置六郡沈黎汶山后皆并于蜀郡武都北属凉部惟越嶲之青蛉遂久今属姚安定莋今属丽江为滇北境牂牁之宛温平彝今属曲靖■〈田句〉町今属临安为滇东境余皆黔蜀之地而兹并详之者楚之定滇由牂牁汉之入滇由越嶲非二郡道且中絶故于其事必记焉若益州郡领县二十有四则皆滇地也其可考者滇池今首邑俞元今澄江连然今安寜味县今曲靖弄栋今姚安楪榆今大理云南即大理之云南县武帝元封时以彩云见南中因以为名蜀汉即其地置云南郡至元始移云南行省于中庆路眀因之称云南府则古益州郡治非汉所谓云南也不韦今永昌汉武帝定南越徙吕嘉子孙居之以嘉为吕不韦后故名不韦以彰其先人之恶至后汉哀牢夷内附更置永昌郡割益州郡之六县以属之

金马碧鸡之神

汉宣帝五鳯三年有方士盛言益州金马碧鸡之神可祷祀而致乃遣谏议大夫王褒持节往求之金马碧鸡二山名在滇池左右颜师古云金形似马碧形似鸡元人张道宗纪古滇说云周宣王时天竺摩耶提国阿育王生三子长曰福邦次曰宏徳次曰至徳三子俱健勇因父有神骥争欲得之王莫能决乃命左右曰将我神骥纵驰而去有能追获者主之纵骥东奔季子先至滇之东山得之因以金马名其山长子次至西山有碧鳯集山上滇人呼鳯为鸡因名山为碧鸡次子后至北野各留屯不回阿育王忧思遣舅氏神明以兵迎之为哀牢夷所阻遂居滇各主其山死而为神王褒既受命是时蛮叛道不通乃就蜀巴郡醮祭移文颂之其词曰持节使王褒谨拜南崖敬移金精神马缥碧之鸡处南之荒深溪回谷非土之乡归徕归徕汉德无疆广乎唐虞泽配三皇黄龙见兮白虎仁归徕归徕可以为伦归兮翔兮何事南荒也褒还未至道病卒上惜之今成都亦有金马碧鸡山相传即褒醮祭处后汉地里志又载青蛉县禺同山有金马碧鸡光景时时出见人多见之其地葢在越嶲郡未知孰是
