　　钦定四库全书　　　　　史部三
　　绥寇纪略　　　　　　　纪事本末类
　　提要
　　【臣】等谨案绥寇纪略十二卷
　　国朝呉伟业撰伟业字骏公号梅村太仓人明崇祯辛未以第三人及第入
　　国朝官至国子监祭酒是编专纪崇祯时流寇迄于明亡分为十二篇曰渑池渡曰车箱困曰真宁恨曰朱阳溃曰黒水擒曰谷城变曰开县败曰汴渠垫曰通城击曰盐亭诛曰九江哀曰虞渊沈每篇后加以论断其虞渊沈一篇皆记明末灾异与篇名不相应考朱彞尊曝书亭集有此书跋云梅村以顺治壬辰舍馆嘉兴之万夀宫辑绥寇纪畧久之其乡人发雕是编仅十二卷而止虞渊沈中下二卷未付枣木传刻明史开局求天下野史尽上史馆于是先生足本出予抄入百六丛书归田之后为友人借失云云意者明末降闯劝进诸臣子孙尚存故当时讳而不出欤此本为康熈甲寅邹式金所刻在未开史局之前故亦无虞渊沈中下二卷而彞尊所辑百六丛书为人借失者虽称后十八年从呉兴书贾购得今亦不可复见此二卷遂佚之矣彞尊又称其书以三字标题仿苏鹗杜阳襍编何光逺鉴戒録之例考文章全以三字标题始于缪袭魏铙歌词鹗光逺遂沿以著书伟业叙述时事乃用此例颇不免小说纤仄之体其回防杨嗣昌左良玉亦涉恩怨之私未为公论然记事尚颇详悉尊所谓闻之于朝虽不及见者之确切而终胜草野传闻可资国史之采辑亦公论也乾隆四十三年五月恭校上

　　钦定四库全书
　　绥冦纪略卷一
　　国子监祭酒吴伟业撰
　　渑池渡
　　崇祯元年戊辰三月二十日昧爽前全秦天赤如血五六七月西安有孽火入人家色青光荧荧然广轮盈尺者数十旋于地若鬬不滥炎民磔鸡犬禳之乃去自四月至七月不雨八月恒雨霜杀稼冬大雨雪木冰嵗大饥秦以延绥宁夏甘肃为三边延绥据河为塞斥衍千二百里筑三十六堡分险列戍沙碛不生五谷富民计口仰食县官承平忨愒算崇祯改元上距万厯四十七年延饷缺额者至一百三十八万天啓中秦抚乔应甲延抚朱童蒙皆奄党也应甲贪婪抂易法纪溃弛四封之内攻亭长杀长吏而莫禁【尝有道臣袁养和被刼请捕应甲反胁取养和千金中书张贺昌被刼执巡捕典史王允徳索赂二百金以是被盗之家莫敢控告】童蒙朘削军廪助三殿大工兵民侧目始祸实自两人秦地所出新饷均输间架三者专以备他急数日增吏因缘为奸羡余辄倍西安镇兵数万率买闲占役以空名隷尺籍【兵科刘懋疏秦兵属边操者卖于边官嵗纳班价属叅都者卖于叅都月纳班钱操兵三千占于各衙门司道宗室乡绅以供役使】民不堪复无所惮以此纠合边人为贼　十一月白水男子王二反【王二刼宜君县狱拒捕逃入苜蓿沟贼巢穴十一月内通白水衙役杨发蒲城王髙等为乱】掠蒲城之孝童韩城之淄川镇　王左挂【原名王之爵有苗美飞山虎大红狼等同起】起于宜川【地名龙耳嘴】　十二月三边总督武之望奏固原兵变【元年十二月二十四日兵刼固原州库】总兵钱中选无军纪得免论巡按御史吴焕纠巡抚胡廷宴瞆眊不视事与延抚岳和声以边兵饥民互相推匿所奏报俱不实致成大患兵部尚书王洽请覈其事以闻　刑给事薛国观奏言淳化绥徳等处流贼千百为羣由乔应甲纵盗酿祸得防报闻　二年己巳正月郧阳抚治梁应泽奏陜西贼王大梁【一名大梁王】三千人入略阳偪汉中【十月二十六日汉南贼大梁王从成县两当逼近略阳汉中官兵追至宁羌州逸去后复趋略阳官兵战败郧抚告急】　商洛道刘应遇讨白水贼王二斩之【余党窜入黄龙山苜蓿沟】　固原逃兵掠泾阳富平执游击李英　二月总督武之望病死胡廷宴闲住以陜西左布政刘广生为秦抚右副都御史杨鹤总督三边　岳和声罢【吴焕疏盗自戊辰七月王二作难己巳盗首韩世盘韩世友胡之槐吴养臣皆边兵故和声亦罢】以河南左布政张梦鲸为延抚吴自勉为延绥总兵　四月阶州叛兵周大旺等作乱副将贺虎臣捕得伏诛督粮道叅议洪承畴率乡勇围左挂于云阳防成擒暮大雷雨作遁去【三月廿一日贼自真宁逼三水逰击髙从龙却之贼入马兰山四月九日复犯耀州承畴令官兵乡勇万余人分十二营围贼于云阳防获之乗夜雷雨溃围走淳化入神道岭追斩二百余级】　刘应遇追汉南贼于大石川所将卒斩王大梁于阵围其余党于伏羌之五丁峡将尽歼之宁羌知州陈元瓒失防贼遁去【刘应遇提毛兵入汉合川将吴国辅兵击贼于略阳十战皆捷贼屯垣口有守备王元极陷阵马蹶被杀犹手握贼级不置贼退汉隂应遇令都司解文英同蜀将吴国辅李标竒追至大石川斩五百级把总李旺得贼首王大梁余解散】　十一月京师戒严延绥总兵吴自勉入援侵行粮军中道逃归为乱延抚张梦鲸愤恚死【自勉沿途逗遛贿放精兵变卖营马梦鲸愤郁以死濒死移书杨鹤纠叅鹤不以上闻】山西巡抚耿如杞亦率兵入衞叛逃为贼【耿所綂皆沿边劲卒巳至都下矣兵部调守通明日又调守昌又明日又调守良乡功令兵到日不准开粮次日列营汛地乃给之西兵连调三日三日不得粮既馁且怒遂沿山东一带刼掠耿以不戢军士逮问耿既逮五千人哄然尽散溃归山西而晋中流贼起矣】　以洪承畴为延绥巡抚　兵部尚书王洽有罪下狱左侍郎申用懋晋尚书　三年庚午正月申用懋自劾免以蓟督梁廷栋为兵部尚书　吴自勉罢以临洮总兵王承恩为延绥总兵【其改用王承恩以兵将不安剥军士未上闻也后李应期阅边据所闻叅劾帝怒甚提问自勉于四年九月病故】蓟督张凤翼畱承恩春防甘肃总兵杨嘉谟入援践更率其兵驻东协时钱中选亦入援行秦督抚以三边四镇止宁夏总兵贺虎臣一人非所以办冦请用故总兵杜文焕摄承恩镇西将事兼督延绥固原兵三千便宜勦抚报可　王子顺苗美勾逃兵掠绥徳围韩城复犯清涧总督杨鹤巡抚刘广生击却之斩级三千降其众三百人总兵杜文焕又败之于安定先是万厯时朝廷念西军劳苦预给三月粮以为常崇祯己巳秦大旱粟踊贵军饷告匮延抚杨鹤甘抚梅之焕分道勤王是年复以稽饷而哗其溃卒畏捕诛亡命山谷间倡饥民为乱时东事急朝议核兵饷各边镇咸厘汰裁额至数十万乗障兵咸噪而下而兵科给事刘懋疏请裁定驿站嵗可节金钱数十万帝喜着为令有滥予者罪勿赦懋意谓苏民力也而河北游民借食驿糈至是遂无所得食益无赖溃兵乗之而全陜无宁宇矣【先是懋有补足两年缺饷与赈金十万之请事既不行再疏以陜西衞兵冒滥虚弱求按册清汰操练并秦有司重火耗多赎锾宜拏问其尤以谢关中之人乱将自定其论最剀切惟举全秦公费存留巡方賍罚驿站节裁银两充勦抚之用颇有近于捜括然因号呼帑金弗应而疆事日益亟岂得已哉而京师向有刘给事裁省驿递驱民为盗之谣不知何故愚谓必出于士大夫輶轩之口所以流传至今夫鄜延絶塞无与流传关陇饥民宁关舆隷若早从其请不啻曲突徙薪是不可以不辨然考之御史姜思睿疏云各递贫民千百为羣任辇舆以续命者饥饿待死散而为盗闻懋易箦时亦深悔为此言后死京邸棺至山东莫有为辇负者各处饥民甚至呼其名而诅咒之图其形而丛射之是又非无见云然姑两存之】　二月杜文焕檄谕米脂清涧诸贼王左挂以其党二百人降　荧惑入东井退舍复嬴居数月又入鬼宿犯积尸气　延安知府张辇都司艾穆蹙贼于延川贼王子顺张述圣姬三儿等俱乞降王嘉引掠延安庆阳城堡多陷杨鹤主抚不以闻【鹤与刘广生出示崇祯三年三四等月有贼首黄虎小红狼一丈青龙得水混江龙掠地虎孟良等俱给免死牌安置延绥河曲其淫掠如故民罹荼毒有司莫敢御冦患成于此矣】　四月贼从神木渡河入晋　五月杨鹤移镇耀州贼攻破金锁关杀都司王廉调榆林道张福臻提兵讨之　六月御史李应期巡按陜西疏请招抚许之　职方郎李继贞兵科给事刘懋先后请以十万金赈延绥不报贼王嘉引破黄甫川清水木瓜三堡孤山副将李钊被杀寻破府谷据其城又破孤山【先是钊欲图贼而左右皆贼党钊出巡至清水营遂遇害贼在府谷不过数百官兵大集可以尽歼而轻用招安不渝旬日复号召成旅东陷山西河曲】刘广生称病以陜西左布政王顺行为陜抚　以曹文诏为孤山副将　七月李应期奏庆阳守备李极败绩于鄜州之雷公岭【巡检陈其佐被杀守备白邦政重创失战马四十三匹】　王顺行请三年裁扣驿站银三万两充养兵费下戸部尚书毕自严酌议以闻【杨鹤疏云帝允旧抚臣王顺行之请准发裁扣站银三万金而布政吕逊又以抵补借动司库之数是于顺行去后从其请而终于不行也】　八月府谷贼王嘉引以其众奔黄甫川　王顺行解任以太仆寺少卿练国事为秦抚　九月洪承畴杜文焕攻黄甫川贼破走之　降贼王左挂谋叛洪承畴李应期定计诛之　钱中选罢以杨麒为固原总兵率其兵驻西协　兵部尚书梁廷栋请勅秦督抚逮治二三贪残有司尽补三边两年缺饷而用杜文焕为大将秦晋诸将兵皆属焉【两省选精锐万人属之沿河营衞听节制各抚按道亲歴行间监督陜西留站银三万两山西留二万两两省抚按再各借二万五千两以资军费】　秦洊饥至是大祲山西以防河闭粜斗米六钱米脂清涧延长绥徳之民流亡捐瘠米脂人从贼者十之七邑几空【吴甡疏请通晋中籴贩有防在西安接济其实西安逺而晋近而清涧绥徳葭州与晋之临县兴县永宁州皆有渡口商民从来互相贸易今晋中设兵防盗因以遏籴榆林天下雄镇多宿将劲兵若扼其吭絶之大可忧也】　肤施人张献忠据十八寨十月贼王嘉引再陷清水木等堡已又陷府谷杀
